# AI 开发环境搭建 | GPU 选型与云平台对比

搭建 AI 开发环境是动手实践的第一步。本文帮你理清 GPU 选型、本地配置与云平台选择。

## 硬件选型

### GPU 关键参数
| 参数 | 说明 | 影响 |
|------|------|------|
| 显存（VRAM） | GPU 内存容量 | 决定能跑多大的模型 |
| Tensor Core | 张量计算单元 | 直接影响训练/推理速度 |
| 带宽 | 显存读写速度 | 大 batch 训练的关键瓶颈 |
| FP16/BF16 算力 | 半精度浮点性能 | 混合精度训练的参考指标 |

### 常见 GPU 选择

| GPU | 显存 | 适合场景 | 参考价格 |
|-----|------|---------|---------|
| RTX 4060 | 8GB | 小模型推理、学习 | ¥2,000+ |
| RTX 4070 Super | 12GB | QLoRA 微调 7B 模型 | ¥4,500+ |
| RTX 4090 | 24GB | LoRA 微调 7B/13B | ¥13,000+ |
| RTX 5090 | 32GB | 全量微调 7B、LoRA 70B | ¥18,000+ |
| A100 80GB | 80GB | 全量微调 70B、大规模训练 | 云租用 |
| H100 80GB | 80GB | 最高性能训练 | 云租用 |
| Apple M2 Ultra | 192GB 统一内存 | 大模型推理、Mac 用户首选 | ¥30,000+ |

### 消费级 vs 专业级
- **消费级（RTX 系列）**：性价比高，适合个人开发与学习
- **专业级（A100/H100）**：显存大、支持 NVLink 多卡互联，适合企业级训练

## 云 GPU 平台对比

| 平台 | 特点 | 适合人群 |
|------|------|---------|
| **AutoDL** | 国内最主流，按小时计费，环境镜像丰富 | 个人开发者首选 |
| **恒源云** | 价格实惠，学生优惠 | 学生、个人学习 |
| **阿里云 PAI** | 生态完整，配套服务多 | 企业用户 |

## 本地开发环境

### NVIDIA 驱动与 CUDA
```bash
# 检查驱动
nvidia-smi

# 安装 CUDA Toolkit（推荐 12.1+）
# 搭配 PyTorch 2.x 使用
```

### Python 深度学习环境
```bash
# 创建虚拟环境
conda create -n ai python=3.11
conda activate ai

# 安装 PyTorch（根据 CUDA 版本选择）
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# 核心库
pip install transformers datasets accelerate peft
pip install bitsandbytes  # 量化训练
pip install xformers      # 加速注意力计算
```

### Docker 环境（推荐生产使用）
```dockerfile
FROM nvidia/cuda:12.1.0-devel-ubuntu22.04
RUN apt update && apt install -y python3.11 python3-pip
RUN pip3 install torch transformers vllm
```

## 环境测试
```python
import torch
print(f"CUDA 可用: {torch.cuda.is_available()}")

# 简单测试计算
x = torch.randn(1000, 1000).cuda()
y = torch.randn(1000, 1000).cuda()
z = torch.mm(x, y)  # 矩阵乘法
print("GPU 计算正常 ✓")
```
