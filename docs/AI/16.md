# RAG 检索增强生成 | 从零搭建知识库问答

RAG（Retrieval-Augmented Generation）让大模型能基于外部知识回答问题，有效缓解幻觉问题。

## 一、RAG 是什么

### 核心思想
```
传统 LLM 问答：
用户问题 → 大模型 → 回答（只能靠训练记忆）

RAG 问答：
用户问题 → 向量检索 → 找到相关文档 → 文档+问题一起给大模型 → 回答（基于外部知识）
```

### 为什么需要 RAG
| 痛点 | RAG 解决方式 |
|------|-------------|
| 知识过时 | 更新文档即可，不用重训模型 |
| 幻觉编造 | 约束模型基于文档回答 |
| 私有知识 | 不训练也可使用企业内部文档 |
| 可溯源 | 回答附引用来源，可验证 |

## 二、RAG 架构

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  文档处理     │ →   │  向量存储     │ →   │  检索增强     │
│  (Embedding)  │     │  (Vector DB)  │     │  (Retrieval)  │
└──────────────┘     └──────────────┘     └──────┬───────┘
                                                  │
                                                  ▼
                                           ┌──────────────┐
                                           │   大模型生成   │
                                           │  (Generation) │
                                           └──────────────┘
```

## 三、从零搭建 RAG

### 1. 文档加载
```python
from langchain.document_loaders import TextLoader, PyPDFLoader, Docx2txtLoader

# 加载各类文档
loaders = {
    ".txt": TextLoader,
    ".pdf": PyPDFLoader,
    ".docx": Docx2txtLoader,
}

documents = []
for ext, Loader in loaders.items():
    for file_path in glob.glob(f"docs/*{ext}"):
        loader = Loader(file_path)
        documents.extend(loader.load())
```

### 2. 文档分块
```python
from langchain.text_splitter import RecursiveCharacterTextSplitter

text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,        # 每块大小
    chunk_overlap=50,      # 块间重叠
    separators=["\n\n", "\n", "。", "！", "？", "；", "，", " ", ""]
)

chunks = text_splitter.split_documents(documents)
print(f"分割为 {len(chunks)} 个文本块")
```

### 3. 向量化存储
```python
from langchain.embeddings import HuggingFaceEmbeddings
from langchain.vectorstores import Chroma

# 嵌入模型（中文推荐）
embeddings = HuggingFaceEmbeddings(
    model_name="BAAI/bge-large-zh-v1.5",  # 中文 embedding 首选
    model_kwargs={"device": "cuda"},
    encode_kwargs={"normalize_embeddings": True}
)

# 构建向量数据库
vectorstore = Chroma.from_documents(
    documents=chunks,
    embedding=embeddings,
    persist_directory="./chroma_db",
    collection_name="my_knowledge_base"
)
```

### 4. 检索 + 生成
```python
from langchain.llms import VLLM

# 初始化 LLM
llm = VLLM(
    model="Qwen/Qwen2.5-7B-Instruct",
    max_new_tokens=1024,
    temperature=0.7,
)

# 检索器
retriever = vectorstore.as_retriever(
    search_type="mmr",    # MMR 兼顾相关性和多样性
    search_kwargs={"k": 4, "fetch_k": 20}
)

# RAG Prompt 模板
RAG_PROMPT = """你是一个知识助手。请仅基于以下参考信息回答问题。
如果信息不足，请诚实地告诉用户你不知道。

参考信息：
{context}

用户问题：{question}

回答要求：
1. 使用参考信息中的内容
2. 引用具体的信息来源
3. 不要编造信息中不存在的内容
"""
```

### 5. 完整 RAG 链
```python
from langchain.chains import RetrievalQA

qa_chain = RetrievalQA.from_chain_type(
    llm=llm,
    chain_type="stuff",
    retriever=retriever,
    return_source_documents=True,
    verbose=True,
)

# 使用
result = qa_chain({"query": "公司年假政策是什么？"})
print(result["result"])           # 答案
print(result["source_documents"]) # 来源
```

```python
# 1. 混合检索（Hybrid Search）
# 结合向量检索和 BM25 关键词检索
from langchain.retrievers import BM25Retriever, EnsembleRetriever

# BM25 检索器（擅长精确匹配）
bm25_retriever = BM25Retriever.from_documents(chunks)
bm25_retriever.k = 4

# 向量检索器（擅长语义匹配）
vector_retriever = vectorstore.as_retriever(search_kwargs={"k": 4})

# 混合检索器
ensemble_retriever = EnsembleRetriever(
    retrievers=[bm25_retriever, vector_retriever],
    weights=[0.3, 0.7]  # 混合权重
)

# 2. 重排序（Re-ranking）
from langchain.retrievers import ContextualCompressionRetriever
from sentence_transformers import CrossEncoder

class Reranker:
    def __init__(self):
        self.model = CrossEncoder("BAAI/bge-reranker-large")
    
    def rerank(self, query: str, docs: list, top_k=3):
        pairs = [[query, doc.page_content] for doc in docs]
        scores = self.model.predict(pairs)
        ranked = sorted(zip(docs, scores), key=lambda x: x[1], reverse=True)
        return [doc for doc, score in ranked[:top_k]]
```

## 七、生产部署

```python
# FastAPI RAG 服务
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class Query(BaseModel):
    question: str

@app.post("/rag")
async def rag_query(query: Query):
    # 检索
    docs = retriever.get_relevant_documents(query.question)
    context = "\n\n".join([d.page_content for d in docs])
    
    # 生成
    prompt = RAG_PROMPT.format(context=context, question=query.question)
    answer = llm.generate(prompt)
    
    return {
        "answer": answer,
        "sources": [{"content": d.page_content[:200], "source": d.metadata.get("source")} for d in docs]
    }
```
