# Function Calling 与 Tool Use 开发指南

Function Calling 让大模型从"纯聊天"升级为"能执行操作的智能体"。本文讲解从原理到实战。

## 一、什么是 Function Calling

### 核心概念
```
传统 LLM：用户 → 模型 → 文本回复

Function Calling：
用户 → 模型 → 判断需要调用哪个工具 → 结构化工具调用 → 获取结果 → 模型 → 最终回复
```

### 应用场景

## 三、Function Calling 实战

### 完整流程
```python
from openai import OpenAI
import json

client = OpenAI()

def function_calling_loop(user_message: str, tools: list, available_functions: dict):
    """Function Calling 完整循环"""
    
    messages = [{"role": "user", "content": user_message}]
    
    # 第一步：模型判断是否需要调用工具
    response = client.chat.completions.create(
        model="gpt-4",
        messages=messages,
        tools=tools,
        tool_choice="auto",  # 自动判断是否调用
    )
    
    response_message = response.choices[0].message
    
    # 第二步：处理工具调用
    if response_message.tool_calls:
        messages.append(response_message)
        
        for tool_call in response_message.tool_calls:
            function_name = tool_call.function.name
            function_args = json.loads(tool_call.function.arguments)
            
            print(f"🔧 调用工具: {function_name}({function_args})")
            
            # 执行函数
            function_response = available_functions[function_name](**function_args)
            
            # 将工具结果加入对话
            messages.append({
                "role": "tool",
                "tool_call_id": tool_call.id,
                "content": str(function_response)
            })
        
        # 第三步：模型基于工具结果生成最终回复
        final_response = client.chat.completions.create(
            model="gpt-4",
            messages=messages,
        )
        
        return final_response.choices[0].message.content
    
    # 无工具调用，直接返回
    return response_message.content
```

## 五、Agent 模式

### ReAct Agent
```python
class ReActAgent:
    """ReAct 模式的 Agent"""
    
    def __init__(self, llm, tools: dict):
        self.llm = llm
        self.tools = tools
    
    def run(self, task: str, max_steps: int = 5) -> str:
        prompt = self._build_prompt(task)
        
        for step in range(max_steps):
            # Thought → Action
            response = self.llm.generate(prompt)
            
            if "Final Answer:" in response:
                return self._extract_final(response)
            
            # 解析 Action
            action_name, action_input = self._parse_action(response)
            observation = self.tools[action_name](**action_input)
            
            # 追加 Observation
            prompt += f"\nObservation: {observation}\nThought:"
        
        return "超过最大步数"
    
    def _build_prompt(self, task: str) -> str:
        tools_desc = "\n".join([
            f"- {name}: {desc}" for name, desc in self.tools.items()
        ])
        
        return f"""You have access to the following tools:
{tools_desc}

Use the format:
Thought: what to do
Action: tool_name[input]
Observation: tool result
... (repeat)
Final Answer: final answer

Task: {task}

Thought:"""
```

## 七、安全注意事项

```python
# ⚠️ 工具调用安全守则

"""
1. 输入验证
   - 永远不要直接执行模型的代码输出
   - 对所有参数进行类型和范围验证

2. 权限控制
   - 敏感操作（删除、支付）需要用户二次确认
   - 限制工具可访问的资源范围

3. 输出检查
   - 检查工具返回内容是否安全
   - 防止工具返回内容注入恶意指令

4. 速率限制
   - 限制单次对话的工具调用次数
   - 防止无限循环调用
"""
```
