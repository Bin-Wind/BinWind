# macOS 效率神器：Alfred 完全使用指南

Alfred 是什么？简单来说，它是 Spotlight 的超级增强版。自 2010 年问世以来，这款 Mac 效率启动器已经从一个快速打开 App 的小工具，进化成了 macOS 上最强大的生产力平台之一。如果你每天在 Mac 上工作超过 4 小时，Alfred 很可能是回报率最高的效率投资。

## 基础功能：远不止搜索

按下 `Option + Space`（默认快捷键），一个简洁的命令框弹出。输入 App 名称前两个字母，回车即启动 —— 比触控板划来划去快得多。输入文件名直接打开，支持模糊匹配。更实用的是系统命令：`lock` 锁屏、`sleep` 休眠、`restart` 重启、`emptytrash` 清空废纸篓、`quit` 强制退出应用。这些操作用 Alfred 完成，手指不用离开键盘。

自定义网页搜索是另一个高频功能。在 Features → Web Search 中预设了 Google、Wikipedia、YouTube 等常用搜索，每个对应一个快捷键。比如 `wiki 人工智能` 直接打开维基百科结果页。你还可以添加任何网站的搜索接口：只要找到搜索 URL，把关键词替换为 `{query}`，一个自定义搜索就创建好了。我常用的包括豆瓣书籍搜索、NPM 包搜索、GitHub 仓库搜索。

## 剪贴板历史：你复制过的一切都在这里

打开 Alfred Preferences → Features → Clipboard History，启用后 Alfred 会记录所有复制过的文本、图片和文件。默认快捷键 `⌘ + Option + C` 弹出剪贴板面板，支持实时搜索。需要粘贴 5 分钟前复制的那段代码？打出几个关键字，回车即可。这个功能彻底解决了「复制了新内容才发现旧内容还需要用」的尴尬时刻。

更进一步是 Snippets 功能（Powerpack 独占）。你可以预设常用文本片段并绑定关键词。我日常使用的片段包括：`!email` 展开为完整邮件模板、`!date` 展开为今天日期、`!addr` 展开为常用地址。代码审查时常用的 LGTM 评论、重复性的客服回复、Markdown 表格模板……一切反复敲击的文字都值得做成片段。配合 `{clipboard}`、`{date}` 等动态占位符，片段可以非常灵活。

## Workflows：真正区分 Alfred 和其他工具的功能

如果说基础功能和剪贴板让 Alfred 好用，那么 Workflows 让它不可替代。Workflows 的本质是一个可视化自动化引擎：触发器（Hotkey、Keyword、File Action）→ 中间动作（Run Script、Open URL、Filter）→ 输出（Copy to Clipboard、Post Notification、Large Type）。

不需要编程基础，拖拽模块就能搭建简单 Workflow。当然，写几行 Bash 或 Python 脚本能让威力翻倍。社区共享了上千个 Workflow，这些是我每天依赖的几个：

- **Browser Tabs**：输入 `tabs` 搜索所有浏览器打开的标签页，直接跳转。开了 80 个标签页的人必备。
- **Password Manager**：Bitwarden / 1Password 集成，`bw 关键词` 搜索密码库，一键复制账号密码。
- **Unit Converter**：`conv 100 usd cny` 实时汇率转换，`conv 5km miles` 单位换算，不用打开浏览器搜索。
- **Kill Process**：输入 `kill` 然后搜进程名，回车强制结束。比活动监视器快一个数量级。
- **Encode / Decode**：Base64、URL 编码解码，开发调试常用。
- **IP Address**：输入 `ip` 显示本机公网 IP 和内网 IP。
- **Colors**：十六进制颜色代码 ↔ RGB/HSL 互转，前端开发常用。

创建 Workflow 的学习曲线其实很平缓。Alfred 官方文档提供了清晰的教程，社区论坛也有大量问答。花一个下午学会做第一个 Workflow，之后你想自动化什么就能自动化什么。

## Alfred vs Raycast

这大概是 Mac 效率圈最经典的争论。两者都是优秀的启动器，但理念略有不同。Raycast 更「开发者友好」，扩展商店集成度更高，UI 更现代，团队功能免费。Alfred 更精致、更稳定、更 Mac 原生。如果你喜欢一次性付费买断一个打磨了十多年的产品，选 Alfred。如果你追求免费、扩展驱动的生态和持续迭代，选 Raycast。两者都远胜 Spotlight，选择哪一个都不会错。详见本系列第 4 篇 Raycast 专文。

## Powerpack 定价

Alfred 免费版已经包含 App 启动、文件搜索、网页搜索等核心功能，对轻度用户完全够用。Powerpack 解锁 Workflows、剪贴板历史、Snippets 等高级功能，价格是 **£34 单设备终身许可证**，或者 £49 的 Mega Supporter 版（包含未来大版本更新）。一次性付费，不是订阅制。

£34 折合人民币约 310 元。一个你可能每天用几百次的工具，这个价格非常合理。我购买 Mega Supporter 是 2018 年，至今没有额外付过费，性价比极高。

## 上手建议

如果你刚安装 Alfred，建议的渐进路线：第一周只用来启动 App 和搜索文件，替代 Spotlight。第二周配置剪贴板历史，习惯 `Cmd + Option + C`。第三周设置 Snippets，把日常重复打字自动化。一个月后再研究 Workflows。

别试图一天配置完所有功能，那只会让你放弃。让习惯逐层叠加，一个月后你会发现再也回不去 Spotlight 了。
