# 写作工具推荐：从 Ulysses 到 iA Writer

写作工具的谱系可以从两个极端来理解：一端是功能全面的「写作工作室」（Ulysses、Scrivener），提供从草稿到发布的全流程支持；另一端是极简主义的「纯粹文本编辑器」（iA Writer、Byword），只提供书写体验本身，把一切干扰降到最低。你需要哪一端，取决于你的写作类型和个人偏好。

## Ulysses：博客作者的终极写作环境

Ulysses 出自德国团队 Ulysses GmbH，是一款 Mac + iOS 独占的写作应用。它的定位非常清晰：为一个写作项目提供从初稿到发布的所有工具。

Ulysses 的核心设计是「Sheet」（文稿）。每个 Sheet 是一篇独立文档，多个 Sheet 组成一个 Group（类似文件夹）。Group 可以嵌套，形成灵活的组织结构。所有文稿存储在 Ulysses 的统一资料库中（基于 iCloud 同步，速度极快），不需要手动管理文件。

写作目标功能是 Ulysses 的差异化武器：你可以为每篇文稿设定字数和截止日期目标。进度条实时显示完成百分比，今日仍需完成的字数，预计完成日期。这对专栏作家、定期更新的博主、截稿驱动的自由撰稿人来说是极好的激励和规划工具。

Ulysses 使用 Markdown 标记语言（自己的风味，叫 Markdown XL），写作时看到的是纯文本 + 轻量标记，导出时渲染为格式精美的 PDF / HTML / DOCX / ePub。支持一键发布到 WordPress、Medium、Ghost、Micro.blog 等平台，图片自动上传。如果你每周写一篇博客，这个发布流程节省的时间相当可观。

Ulysses 的写作界面干净但不简陋。附件侧边栏可以放图片、脚注、注释。Typewriter Mode 让当前行始终居中，减少视线移动。Dark Mode 精心调校，深夜写作时眼睛不累。内置的校对和统计功能（字符/词/句/段落计数、阅读时间估算）在写作过程中随用随看。

价格：**$49.99/年**（或 $5.99/月），包含 Mac + iOS 双端。订阅制是 Ulysses 最受争议的点——以前的买断制老用户自不必说，新用户需要接受每年付费的模式。但如果你靠写作赚钱，$50/年的工具投入属于极低成本。

**最适合**：博客作者、专栏作者、定期发布内容的写作者、需要写作目标来保持节奏的人。

## iA Writer：极简主义的巅峰

iA Writer 由日本团队 Information Architects 开发，设计哲学上追求极致克制。打开 iA Writer，你看到的几乎只有文字。没有侧边栏、没有格式工具栏、没有弹出提示。它信奉一个原则：写作只需要你和文字，其他一切都不重要。

Focus Mode 是 iA Writer 的标志功能：只有当前正在写的句子保持正常颜色，其余文本全部变灰。这个设计强迫你关注当下这句，不被上下文分心。写完这一句，下一句自动点亮。对容易陷入自我修改循环的写作者来说，Focus Mode 是续命功能。

Syntax Highlight 是 iA Writer 另一个独门绝技：句子成分语法高亮。形容词显示为蓝色，名词显示为黄色，动词显示为绿色，副词显示为紫色。一眼就能看出句子结构是否平衡，是否副词太多、形容词堆砌。这项功能面向的是对文字质量有较高要求的非虚构写作。

Content Blocks 是 iA Writer 的新功能，允许在纯文本文档中嵌入图片、表格、代码块并以视觉化方式呈现。这让 iA Writer 保持了纯文本的简洁性，同时能处理现代写作中不可避免的多媒体内容。所有内容仍然保存在纯文本 / Markdown 文件中。

iA Writer 提供了预览功能，但它更鼓励你在写作过程中不看预览——「写完再看」。所有文件存储在你的本地文件系统中（任意文件夹），你可以用 Git 管理版本、用任何编辑器打开。这点和 Ulysses 的资料库模式形成对比：iA Writer 信任你管理文件，Ulysses 替你管理。

价格：Mac 版 **$49.99 一次性买断**，iOS 版 $9.99，Windows 版 $29.99。一次性付费，不订阅。这个定价策略在订阅泛滥的时代是一股清流。

**最适合**：追求专注无干扰的写作者、使用 Git 管理文字的开发者、喜欢自己掌控文件组织方式的人。

## Scrivener：小说家的瑞士军刀

Scrivener 出自 Literature & Latte 团队，定位极为专一：长篇写作。小说、剧本、学术论文、长篇非虚构——任何需要管理庞大结构和大量研究素材的写作项目。

Scrivener 的组织模型是 Binder（绑定器） + Corkboard（软木板） + Editor（编辑器）。Binder 左侧树状目录，章节、场景、片段随意组织；Corkboard 以卡片墙的形式展示章节概要，可以拖拽卡片重新排列结构；编辑器支持分屏，参考材料在左，草稿在右。

Research 文件夹是 Scrivener 区别于其他写作工具的关键：你可以在项目中拖入 PDF、图片、网页、音频、视频，写作时随时查阅参考文献，不需要切换到其他应用。编译（Compile）功能把杂乱的手稿输出为格式精美的 ePub、PDF、Kindle 电子书，或排版就绪的 Word 文稿。你可以自定义编译模板，控制输出的每个格式细节。

Scrivener 的学习曲线是本文提到所有工具中最陡的。它不是一款「打开就能用」的工具，它需要你花时间学习概念（Binder、Corkboard、Compile）、配置模板、建立工作流程。但一旦熟悉，对长篇写作的生产力提升是巨大的。

价格：Mac 版 **$49 一次性买断**，iOS 版 $19.99，Windows 版 $49。长期来看比 Ulysses 划算得多（一年 Ulysses 订阅费 = Scrivener 终身使用）。

**最适合**：小说家、学术研究者、剧本作者、任何超过 3 万字的写作项目。

## Typora：所见即所得 Markdown

Typora 是我用过最舒服的 Markdown 编辑器。它的独特之处在于「所见即所得」——输入 `# 标题`，立刻渲染成标题样式，光标留在渲染后的文本中继续编辑，没有左右分栏的源码/预览割裂感。这种无缝体验让 Markdown 写作接近富文本编辑器的流畅度，同时保留了纯文本的轻便。

Typora 支持自定义主题（社区贡献了大量精美主题），文件树侧边栏方便浏览文件夹中的文档，导出格式覆盖 PDF/HTML/Word/RTF/ePub/LaTeX 等。对于技术文档写作者（README、开发日志、项目 Wiki），Typora 是目前体验最好的 Markdown 编辑器。

价格：正式版 **$14.99 一次性买断**（Beta 期间长期免费，现已结束）。

**最适合**：技术文档写作者、Markdown 重度用户、需要所见即所得编辑体验的人。

## Obsidian 的写作能力

本系列第 2 篇详细评测了 Obsidian。这里补充说明：Obsidian 本身是极好的写作工具，尤其是搭配 Longform 插件后可以管理长文写作项目。如果你同时需要知识管理和写作，Obsidian 一石二鸟。

## 选择指南

| 写作类型 | 推荐工具 | 理由 |
|---------|---------|------|
| 博客、专栏、定期发布 | Ulysses | 写作目标 + 一键发布 |
| 追求专注、Markdown 文件管理 | iA Writer | Focus Mode + 本地文件 |
| 小说、学术论文、长文 | Scrivener | Binder + Corkboard + Compile |
| 技术文档、Markdown | Typora | 所见即所得 Markdown |
| 知识管理 + 写作 | Obsidian | 双向链接 + Longform |

如果你是刚起步的写作者，从 iA Writer 或 Typora 开始最合适——工具本身不会成为干扰，你只需要关注文字。随着写作量和复杂度增长，再根据需求考虑 Ulysses（需要发布和写作目标）或 Scrivener（需要大型项目管理）。
