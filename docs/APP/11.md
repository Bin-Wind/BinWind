# 剪贴板管理工具：Maccy / Paste / Copy 'Em

Ctrl+C 和 Ctrl+V 大概是你每天使用最频繁的快捷键，但系统剪贴板只有一个槽位。复制了一段代码，5 分钟前复制的那个 API 地址就没了；截了一张图，刚才复制的表格数据就没了。剪贴板管理器就是为这个痛点而生——它记录你复制过的所有内容（文本、图片、文件），需要时随时找回。

## 为什么需要剪贴板管理器

想想你一天中的这些场景：在 StackOverflow 和 IDE 之间来回拷贝代码段；填写表格时反复复制姓名、地址、手机号；写邮件时从不同来源拼凑内容。每一次 Ctrl+C 都在覆盖上一份内容，要是忘了先粘贴就复制了新的，还得回去重新找。剪贴板管理器消除这种焦虑——所有复制过的内容都在历史中，搜索即可找回。

安全方面，优秀的剪贴板管理器会默认排除密码管理器（Bitwarden、1Password、系统钥匙串），不记录密码。你也可以手动排除其他敏感应用。

## 如果你已经在用 Alfred 或 Raycast

先说一个省事的结论：如果你已经在用 Alfred（Powerpack）或 Raycast，它们内置的剪贴板管理器已经足够好用。支持搜索、数字快捷键粘贴、固定常用项目、排除指定应用。不需要额外安装工具。跳过这篇文章就好。

## Maccy：开源轻量之选

Maccy 是我个人的选择。它是一款开源（MIT 协议）、免费、极轻量的菜单栏剪贴板管理器。

Maccy 的工作方式简单：点击菜单栏图标或按自定义热键（我设的是 `Cmd+Shift+V`），弹出剪贴板历史列表。支持实时搜索——输入 `sql` 立刻过滤出含 SQL 代码的历史记录。`Cmd+1/2/3…` 数字键直接粘贴对应项，`Option+1/2/3…` 复制而不粘贴。Pin（固定）功能让常用内容始终置顶。

Maccy 的配置项精简但实用：历史记录条数（默认 200 条，可调）、忽略的应用列表（密码管理器必须加）、纯文本模式（剥离格式）、粘贴后行为（恢复先前剪贴板内容或清空）。支持 macOS 深色模式和原生菜单栏图标。

Maccy 不支持 iCloud 同步，这是它和 Paste 的最大区别。如果你只在 Mac 上工作，这不是问题。如果你需要在 Mac 和 iPhone 之间共享剪贴历史，Maccy 做不到。

**最适合**：开源爱好者、极简主义者、Mac 单机用户。

## Paste：视觉系的剪贴板

Paste 是一款设计精美的剪贴板管理器，界面采用卡片式布局。每个复制项以大卡片展示，文本显示预览、图片显示缩略图、链接显示网站标题和 favicon。视觉识别速度远超纯文字列表。

Paste 的粘贴栈（Paste Stack）是个创新功能：你可以先 Ctrl+C 复制多个项目（它们会顺序加入栈），然后在目标应用中一次性粘贴出来。这个功能在需要按顺序粘贴多个片段时（比如填表单的多个字段）非常高效。

iCloud 同步是 Paste 的核心卖点：你 Mac 上复制的内容，立刻在 iPhone 和 iPad 上可用。Apple 生态多设备用户会爱上这个体验。分享扩展也做得不错：内容可以快速分享到其他 App。

Paste 的定价是 **$14.99/年**（或 $1.99/月），订阅制。对比 Maccy 免费开源，这个价格是最大阻力。但如果你需要多设备同步和精美的视觉体验，它值这个价。

**最适合**：Apple 多设备用户、视觉导向、愿意为设计付费的人。

## Copy 'Em：一次付费的 iOS + Mac 方案

Copy 'Em 是「买断制 + iOS 同步」的折中选择。Mac 版售价 $14.99（一次性），iOS 版 $5.99，购买后永久使用。

Copy 'Em 的特色在于组织能力。你可以创建多个列表（Collection），把剪贴板历史中的项目归类整理。比如创建「代码片段」列表存放常用函数、「客户信息」列表存放常用的回复话术、「Markdown 语法」列表存放写作中复用的格式。这种分级管理是 Maccy 和 Paste 都不提供的。

同步通过 iCloud 进行，Mac 和 iOS 之间无缝衔接。UI 偏传统风格，不如 Paste 视觉出色，但功能完整：搜索、合并粘贴、纯文本粘贴、快捷键自定义。

**最适合**：一次付费不想订阅、需要 iOS 同步、喜欢列表分类管理的用户。

## 安全设置提醒

无论选哪款工具，务必在设置中排除密码管理器：

- Maccy：Preferences → Ignore → 添加 Bitwarden / 1Password / Keychain Access
- Paste：Paste Preferences → Privacy → Ignore Copied Content From → 添加密码管理器
- Copy 'Em：Exclude Apps → 添加密码管理器
- Alfred/Raycast 剪贴板历史：Features → Clipboard History → Advanced → Ignore Apps

这一步极为重要。你肯定不希望在剪贴板管理器里看到自己复制的密码明文。

## 对比总结

| 维度 | Maccy | Paste | Copy 'Em | Alfred/Raycast 内置 |
|------|-------|-------|----------|---------------------|
| 价格 | 免费 | $14.99/年 | $14.99 Mac + $5.99 iOS 买断 | 免费（Alfred Powerpack 需付费） |
| 开源 | 是 | 否 | 否 | 否 |
| iOS 同步 | 不支持 | iCloud 同步 | iCloud 同步 | 不支持 |
| UI 风格 | 文字列表 | 卡片视觉 | 列表 + 文件夹 | 文字列表 |
| 分类管理 | Pin 置顶 | 无 | 列表（Collection） | Pin 置顶 |
| 内存占用 | 极低 | 中等 | 中等 | 极低 |
| 粘贴栈 | 不支持 | 支持 | 不支持 | Alfred Snippets 部分替代 |

## 推荐

- 使用 Alfred / Raycast 的用户：直接用内置剪贴板，零额外开销。
- 想要免费、轻量、开源：**Maccy**。无懈可击的免费工具。
- 多设备 Apple 用户 + 视觉体验：**Paste**。$15/年为你提供跨设备无缝粘贴，体验极佳。
- 一次付费、需要 iOS 同步和分类：**Copy 'Em**。买断制让长期持有成本最低。

我个人配置是 Raycast 内置剪贴板（日常使用）+ Maccy 作为备份（某些场景 Maccy 的搜索和固定逻辑更顺手）。两者同时运行没有冲突，内存占用也完全可以接受。如果你对剪贴板管理有重度需求，这种双工具组合可以试试。
