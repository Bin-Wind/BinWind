# 屏幕录制与 GIF 制作工具合集

录一段屏幕操作来演示 bug、做一个产品功能导览 GIF、录制教程视频——屏幕录制已经从「偶尔用到」升级为「日常刚需」。本文按使用场景分类，推荐从一键 GIF 到专业录制的全套工具。

## 使用场景速览

- **Bug 报告**：录下复现步骤，替代一大段文字描述。开发者会感谢你。
- **产品演示**：给客户或团队展示新功能，需要高质量画面 + 可能的语音解说。
- **教程录制**：需要换场景、加标注、剪掉废话和专业片头。
- **代码讲解**：在 Code Review 中录一段 GIF 解释复杂逻辑。
- **社交分享**：做一个搞笑 GIF 发群聊或者发社交媒体。

不同场景对工具的需求完全不同，一款工具打天下是不现实的。

## GIF 制作工具：轻量、快速、即录即得

### LICEcap

LICEcap 是一个堪称「古董级」的小工具，但至今依然是我录制简单 GIF 的首选。Windows 和 macOS 都支持，完全免费。

**使用方式**：打开 LICEcap，出现一个透明窗口。拖拽这个窗口框住你想要录制的屏幕区域，点击 Record，选择保存路径，开始录制。结束时就得到一个 GIF 文件。

**核心优势**：
- 零学习成本：没有工具栏、没有编辑功能、没有导出设置——就一个框 + 一个录制按钮
- 文件体积极小：录制的 GIF 压缩率极高，适合直接贴到 GitHub Issue 或 Slack
- 支持暂停和继续，支持限制最大帧率以控制文件大小
- 启动不到 1 秒

**适用场景**：快速录一段操作给同事看，不需要编辑，不需要美化，纯功能向。

**缺点**：功能极为有限，无法编辑、无法加标注、无法导出视频格式。macOS 版的界面看起来像是 2010 年的软件。

### Kap

Kap 是 macOS 专属的开源录屏工具，设计精美，功能强于 LICEcap 但保持了简洁。

**亮点**：
- 可以录制为 MP4、GIF、WebP、APNG 四种格式。WebP 和 APNG 是 GIF 的现代替代品，画质更好、文件更小
- 支持系统音频录制（macOS 录屏音频一直很麻烦，Kap 处理得很好）
- 录完后可以简单裁剪起止位置
- 开源、免费、界面符合 macOS 设计语言

**适用场景**：比 LICEcap 质量更高、格式更多样，同时保持开箱即用的简单体验。适合需要导出多种格式或录制系统音频的用户。

**注意**：Kap 没有标注功能，也没有多轨道编辑。它就是一个录屏工具，不是编辑器。

### GIPHY Capture

GIPHY 出品的免费 Mac 录屏工具。简单录、自动上传到 GIPHY 平台、获得分享链接。适合做表情包或搞笑 GIF 发社交媒体。最大缺点是必须登录 GIPHY 账户，且免费版有 GIPHY 水印。

## 屏幕录制 / 视频工具：从够用到专业

### QuickTime Player：你没注意到的内置原生工具

macOS 自带的 QuickTime Player 就能录屏——文件 → 新建屏幕录制。支持全屏 / 选区录制、麦克风音频。画质不错，文件直接保存为 MOV。

**适合**：偶尔需要录一段、不想装任何额外软件、对编辑没有需求。

**局限**：无法录制系统音频（只能录麦克风）、无标注、无编辑、无 GIF 导出、录出来的 MOV 文件还很大。但它是免费的，且随时随地可用。

### OBS Studio：专业级免费方案

OBS Studio 是直播和游戏录制领域的行业标准，免费、开源、跨平台。

**核心功能**：
- **场景（Scenes）和来源（Sources）**：可以随意组合多个画面来源——屏幕、窗口、摄像头、图片、文字、浏览器窗口——并一键切换。做教程视频时，可以预设「全屏编码」和「演讲者 + 屏幕画中画」两个场景来回切。
- **虚拟摄像头**：OBS 画面可以直接输出为系统虚拟摄像头，任何视频会议软件都能当你是摄像头来用
- **混音器**：每个音频来源独立控制音量、延迟、滤波器
- **直播推流**：支持几乎所有直播平台

**适合**：需要多场景切换、画中画、专业音频控制的教学视频创作者。

**不适合**：只想快速录个 30 秒 GIF 的人。OBS 的学习曲线有一定高度，场景和来源的概念需要时间消化。

### Screen Studio（见第 3 篇）

已在系列第 3 篇详细介绍。这里是简单总结：macOS 专属，¥169 一次性付费。自动平滑跟随鼠标、自动放大点击区域、导出即带精美背景和阴影。产品演示录制的天花板。

### Loom：异步沟通专用

Loom 的核心思路不是录屏，而是「录一段视频消息发给同事」。

**工作流**：打开 Loom 浏览器扩展或桌面 App → 选择录制屏幕 + 摄像头（可选） → 开始录 → 停止 → 自动上传，几秒后获得一个分享链接 → 把链接贴到 Slack/邮件/Notion。

**特色功能**：
- 观看者可以用 emoji 快速反应、可以评论在视频时间线的具体位置
- 可以看到谁看了你的视频、看了多长时间
- 支持视频转文字（英文）
- 免费版：最多 25 个视频，每个最长 5 分钟

**最佳场景**：异步工作团队中的项目更新、设计评审反馈、Bug 复现演示、新人入职讲解。说一段话比写一大段文字快，而且信息密度更高。

## AI 辅助编辑

录完素材之后，编辑往往是最耗时的环节。以下两款 AI 工具能大幅缩短编辑时间：

### Descript

Descript 把视频当作文档来编辑——你在文字稿里删掉一句话，视频里对应的那一段就没了。AI 自动识别并标记「嗯」「啊」等填充词，一键删除。还能 AI 克隆你的声音，在文字稿里打字就能生成新的配音。适合播客、教程、产品讲解等以语音为主的视频类型。免费版功能足够尝鲜，Pro 版 \$24/月。

### CapCut（剪映国际版）

字节跳动出品，免费且功能全面。自动字幕生成（中英文都支持，准确率很高）、一键模板（适合社交媒体短视频）、AI 美颜。虽然定位偏短视频和社交内容，但自动字幕功能对任何类型的教程视频都是实实在在的生产力提升。

## 场景速查表

| 场景 | 推荐工具 | 理由 |
|---|---|---|
| 快速 Bug 演示 GIF | LICEcap | 框一下、点录制、完事 |
| 代码讲解 GIF | Kap | 导出 APNG/WebP，画质好于 GIF |
| 专业教程视频 | OBS Studio | 多场景、混音器、免费 |
| 产品演示、Demo | Screen Studio | 自动鼠标特效、导出即美观 |
| 异步团队沟通 | Loom | 录完自动分享链接、追踪观看 |
| 视频后期编辑 | Descript / CapCut | AI 剪辑省时间 |
| 表情包 GIF | GIPHY Capture | 一键上传到 GIPHY |

最后提醒一句：录屏前先把桌面收拾干净——图标藏起来、通知关掉、不相关标签页关掉。整洁的画面比任何后期工具都更能提升观感。
