# 窗口管理工具：Magnet / Rectangle / Moom 怎么选？

macOS 的窗口管理一直是个谜——分屏（Split View）要进入全屏模式、窗口贴边不自动缩放、三个以上窗口排列全靠手动拖。Apple 终于要在 macOS Sequoia 加窗口吸附功能了（距离 Windows 7 的 Aero Snap 过去了 15 年），但在此之前（以及在此之后，如果你想要更强的自定义），第三方窗口管理工具依然是必需品。

本文对比 macOS 上三款最主流的窗口管理工具：免费开源的 Rectangle、App Store 经典的 Magnet、以及功能最强大的 Moom。

## 为什么需要窗口管理工具

用触控板拖窗口、调整大小、移动到合适位置——这个过程每次至少 5-10 秒。一天来回排列二三十次，加起来就是几分钟的纯机械操作。键盘快捷键做同一件事只需要 0.3 秒。

更重要的是，窗口管理工具能实现触控板做不到的操作：

- **精确分屏**：左半 / 右半、左三分之二 / 右三分之一、四分之一角。拖动永远不可能拖出精确比例。
- **快速居中**：一个快捷键让窗口在屏幕正中央以固定大小显示，不依赖手动对齐。
- **多显示器穿梭**：一键把窗口甩到另一个屏幕并自动填满或居左。

## Rectangle：免费、开源、键盘优先

Rectangle 是我最推荐的入门窗管工具。它简单到了极致。

**使用方式**：安装后没有任何窗口——只有菜单栏的一个图标。你只需要记住几个快捷键：

- `Control+Option+左箭头`：窗口占左半屏
- `Control+Option+右箭头`：窗口占右半屏
- `Control+Option+上箭头`：最大化
- `Control+Option+D`：左三分之一
- `Control+Option+F`：右三分之二
- `Control+Option+C`：居中

覆盖了 90% 的使用场景。所有快捷键都可以自定义。支持多显示器、支持触控板手势（如果你用的是带手势的鼠标）、支持「吸附到指定区域」（把窗口拖到屏幕边缘自动排列，需要手动开启）。

**优势**：开源、免费、轻量到几乎不占内存、纯粹的键盘体验。从下载到用完第一次窗口排列，不超过 30 秒。

**不足**：没有鼠标拖拽吸附（默认关闭，开启后体验一般）；不能保存窗口布局快照；不能连锁操作。但对大多数用户来说，这些都不算缺点——它本就主打简单。

**最佳场景**：喜欢键盘操作、不想花钱、需要个稳当可靠的解决方案。

## Magnet：App Store 的经典之选

Magnet 是 App Store 里最知名的窗管工具（¥50，有时降价到 ¥25）。它的定位和 Rectangle 接近，但交互更「苹果风」。

**特色**：

- **拖拽吸附**：把窗口拖到屏幕边缘，自动缩放为该区域的大小——和 Windows 的 Aero Snap 体验一致。拖到顶部最大化，拖到左侧左半屏，拖到角落四分之一屏。鼠标用户的最爱。
- **键盘快捷键**：功能覆盖和 Rectangle 几乎一样（半屏、三分之二、四分之一、居中、全屏等）。
- **菜单栏控制**：点击菜单栏图标，弹出所有布局选项，不想记快捷键也可以用鼠标选。
- **多显示器**：快捷键可以把窗口发送到下一个显示器。

**对比 Rectangle**：Magnet 的拖拽吸附做得比 Rectangle 顺滑，用鼠标的用户会更舒服。但键盘操作的快捷键不如 Rectangle 丰富（比如没有三分之一的布局）。作为付费软件，¥50 不算贵，但隔壁免费开源的 Rectangle 能实现 95% 的功能。

**最佳场景**：习惯了 Windows 拖拽吸附、不想记快捷键、愿意花几十块钱买个精致体验的用户。

## Moom：窗管工具中的「瑞士军刀」

Moom（¥68，Setapp 可订阅）是窗口管理领域的顶级选手。如果 Rectangle 是自行车、Magnet 是电动车，Moom 就是一辆可以自定义驾驶模式的全功能 SUV。

**核心功能**：

- **保存并调用布局快照**：把当前所有窗口的位置和大小保存为一个布局，一键恢复。比如「写代码」布局（VS Code 占左三分之二、终端占右下四分之一、浏览器占右上），「写报告」布局（Obsidian 全屏、一个参考网页浮在右上角）。切任务时一键切换整套窗口。
- **链式命令**：自定义一连串操作——例如「先把窗口移到显示器 2 → 缩放到 1280x800 → 放到右下角」。一个快捷键触发多步操作。
- **自定义网格**：可以把屏幕分割成自定义的网格（例如 3x3 或 5x4），窗口吸附到网格的任意单元格。不局限于系统预设的「左半」「右半」。
- **悬浮预览**：鼠标悬停在窗口的绿色最大化按钮上时，弹出一组预设位置的小图标，点击即到位。
- **触发方式多样**：快捷键、鼠标悬停、菜单栏、自定义手势——你想怎么触发都可以。

**价格**：\$10 一次性买断。相对于它能节省的时间，这个价格非常合理。

**最佳场景**：多显示器、经常在多个任务之间切换、需要特定窗口布局（如写代码 / 写作 / 设计 / 交易）的超级用户。

## 其他选择

### Raycast 窗口管理

如果你已经在用 Raycast 启动器（本系列第 1 和第 4 篇有详细介绍），直接用 Raycast 内置的窗口管理功能就够了。`Command+Option+左/右/上/下` 快速排列，还能在 Raycast 命令框里输入「Left」「Right」「Maximize」来定位。免费且好用。如果你不用 Raycast，则不值得为了窗管单独装它。

### yabai：平铺式窗口管理

yabai 是 macOS 上 i3 风格的平铺窗口管理器。安装后所有窗口自动无重叠平铺排列，新窗口打开时已有窗口自动缩小让位。完全键盘驱动，不用鼠标碰窗口边缘。开源免费。

但平铺式窗口管理是另一种工作哲学——它适合从不手动移动窗口、想让系统自动分配空间的人。对于大多数人来说学习曲线太陡，而且完整功能需要关闭 SIP（系统完整性保护），有一定安全隐患。不建议普通用户尝试，但如果你是 Linux i3 / Sway 用户转 Mac，yabai 是你在 macOS 上的精神家园。

## 选购建议

| 场景 | 推荐 | 价格 |
|---|---|---|
| 免费优先、键盘操作 | **Rectangle** | 免费 |
| 鼠标拖拽吸附、精致体验 | **Magnet** | ¥50 |
| 多布局快照、多显示器 | **Moom** | \$10 |
| 已经在用 Raycast | **Raycast 自带** | 免费 |
| i3 式平铺 | **yabai** | 免费（需 SIP） |

对于大多数用户，**Rectangle** 是最理性的选择——免费、开源、覆盖所有日常需求。如果你愿意为更好的拖拽体验和菜单栏操作付费，Moom 功能更全面。先装上 Rectangle，用两周看看是否满足需求，再决定要不要升级。
