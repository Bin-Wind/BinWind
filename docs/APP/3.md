# 截图工具推荐：从 CleanShot X 到 Shottr

macOS 自带截图（Cmd+Shift+3/4/5）能满足基本需求：截全屏、截区域、截窗口、录屏。但如果你需要标注、滚动截图、云分享、隐藏桌面图标再截图……系统自带工具就捉襟见肘了。这篇文章评测四款主流截图工具，帮你找到适合的那一款。

## CleanShot X：截图工具的天花板

CleanShot X 是目前 Mac 上综合体验最好的截图工具。$29 一次性买断（或 Setapp 订阅），它几乎重新定义了截图体验。

核心功能一览：滚动截图（自动拼接长网页或聊天记录）、屏幕录制（支持摄像头画中画叠加）、完整的标注工具箱（箭头、方框、文字、模糊、编号步进、高亮）、即时云端分享（自动上传 getcleanshot.com 生成短链接）、隐藏桌面图标再截图（一键清场，截图后自动恢复）、钉在桌面（截图浮于所有窗口之上，方便对比查看）。

CleanShot X 的细节打磨是竞品无法比拟的。比如录屏时鼠标点击处会有涟漪动画，截图后自动移除非 Retina 显示器的模糊边缘，标注工具支持 snap 对齐。这些细节你可能不会注意到，但它们让每次截图都顺畅舒适。

我使用 CleanShot X 两年，最喜欢的三个功能：第一，滚动截图捕捉整个网页，不用拼接多张截图；第二，云分享链接，发截图给别人只需要 Cmd+V 粘贴一个链接，不用打开微信文件传输助手；第三，隐藏桌面图标，产品截图需要干净的桌面背景，一键搞定。如果你是需要频繁截图的产品经理、设计师、开发者或内容创作者，CleanShot X 值回票价。

## Shottr：免费的惊喜

Shottr 是一款免费（目前仍在免费期，作者表示未来可能小额度收费）的轻量级截图工具。它的定位不是 CleanShot X 的全面替代品，但核心功能做得极其出色。

Shottr 最抢眼的特色是像素级缩放：截图后鼠标移动到任意区域，会自动放大到像素级别，方便你精确裁剪边缘。对设计师和前端开发者来说，这个功能可以精准测量间距和对齐。内置的取色器可以直接获取截图上任意像素的颜色代码（HEX/RGB/HSL）。OCR 文字识别支持中英文，可以直接从截图中复制文字。滚动截图同样支持，速度和准确度都不错。

Shottr 没有录屏功能，标注工具也比 CleanShot X 少（没有编号步进、没有模糊工具），云分享需要自己搭配图床。但这些缺失对基础截图需求来说并不致命。考虑到它的价格（免费），Shottr 的性价比几乎是无限的。如果你预算有限或者需求简单，Shottr 是闭眼入的选择。

## Xnip：社交分享友好

Xnip 是一款国内开发者开发的截图工具，在 Mac App Store 免费下载。它的特色功能包括：窗口截图自动添加阴影（macOS 风格，美观大方）、滚动截图（手动引导对齐，准确度不错）、标注工具栏类似微信截图、支持贴图功能。

Xnip 最大的优势是针对国内用户场景做了优化。微信截图、钉钉截图、文章配图……这些场景 Xnip 覆盖得很好。标注后的截图风格也更贴近中文社交平台审美。免费版功能已经够用，专业版（¥18）解锁云存储和更多标注功能。

缺点是偶尔有性能问题，录屏功能不支持（专业版也不支持录屏），云分享不如 CleanShot X 流畅。另外 Mac App Store 沙盒限制让 Xnip 无法使用全局快捷键调用（需要额外授权设置）。

## Screen Studio：这是截图吗？不，这是电影

严格来说 Screen Studio 不是截图工具，而是屏幕录像工具。但它太优秀了，值得在同类话题中提到。$89 一次性购买，Screen Studio 能把一段普通屏幕操作录像变成流畅的电影级演示视频：自动追踪鼠标路径实现平滑移动，智能缩放聚焦到操作区域，自动添加阴影和背景，支持摄像头画中画叠加。

如果你是产品经理需要录制功能演示，独立开发者需要 App Store 预览视频，或者是 YouTuber 制作教程视频，Screen Studio 是必备工具。用它录出来的视频不加任何后期剪辑就可以直接发布。当然，如果只是截图，它完全没用。这是两个不同的需求层级。

## 我的推荐

如果你只想装一个工具：CleanShot X。$29 一次性付费，每天使用几十次，三年平均每天不到 3 美分。这个投入换来的效率提升物超所值。

如果你预算有限：Shottr。免费但品质极高，95% 的截图需求都能满足。唯一的遗憾是没有录屏和云分享，但这两个需求可以用系统自带录屏 + 第三方图床弥补。

如果你是录屏为主：Screen Studio。截图用系统自带就好，但录屏效果的提升是质的飞跃。
