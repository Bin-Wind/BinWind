# Raycast：替代 Spotlight 的超级启动器

Raycast 诞生于 2020 年，由 Y Combinator 孵化，在短短几年内积累了超过 200 万用户。它是一款面向开发者和效率控的超级启动器，个人使用完全免费（团队功能收费，这是其商业模式）。如果你觉得 Spotlight 太简陋、Alfred 太传统，Raycast 可能就是你的理想工具。

## Raycast vs Alfred：一场持续争论

Alfred 和 Raycast 是目前 Mac 效率启动器的双雄。核心差异如下：

Alfred 始于 2010 年，十多年打磨让它极其稳定、精致。Workflows 靠社区共享，安装工作流需要从 GitHub 或论坛手动下载 `.alfredworkflow` 文件。UI 风格偏传统 macOS，符合原生审美。收费模式是一次性购买 Powerpack。

Raycast 诞生更晚，架构更现代。它把「扩展商店」做进了应用本身——和安装 VS Code 插件一样，在 Raycast 内浏览、搜索、点击安装。UI 更活泼现代，支持自定义主题。个人免费，团队付费。

到底选哪个？一个简单的判断标准：如果你喜欢「一次性买断一个精雕细琢的工具」，选 Alfred。如果你喜欢「免费使用 + 插件市场和持续高频更新」，选 Raycast。两者都不差，我个人的建议是：开发者优先尝试 Raycast，设计师和传统用户优先考虑 Alfred。

## 核心功能

Raycast 的热键是 `Option + Space`（可自定义），弹出命令面板。基础功能涵盖：App 启动、文件搜索、快速数学运算（输入算式直接出结果）、货币换算（`100 usd to cny`）、日历查看（查看今日/本周日程安排）、窗口管理（内置分屏布局，不需要 Magnet 这类单独工具）。

Raycast 的搜索是智能模糊匹配。你不需要打完整名称，几个字母就能定位到想打开的 App 或文件。搜索历史会智能排序，常用项自动提前。

## 扩展生态：Raycast 的核心竞争力

Raycast 的扩展商店是所有同类工具中最成熟的。目前 Store 中有超过 1000 个扩展，每一个都能在 Raycast 界面内浏览、安装、更新。以下是我每天使用的几个：

- **GitHub**：查看 Notifications、Issues、Pull Requests，创建 Repo，搜索代码。不用打开 GitHub 网页就能处理大部分日常操作。
- **VS Code**：搜索并打开最近项目，直接启动 VS Code。结合 Project Manager 插件，项目管理全在 Raycast 内完成。
- **Linear / Jira**：查看任务、创建 Issue、切换 Sprint。开发者项目管理不用离开键盘。
- **Notion**：搜索 Notion 页面并直接打开，无需打开 Notion App 导航。
- **Spotify / Apple Music**：播放控制、搜索歌曲、切换播放列表。
- **Docker**：查看容器状态、启动/停止容器。
- **Brew**：搜索、安装、更新 Homebrew 包。
- **翻译**：选中文本一键翻译，支持多语种。
- **颜色拾取**：取色器直接获取屏幕任意位置的颜色代码。

安装扩展后，每个扩展可以在 Raycast 设置中配置专属快捷键、别名、热键。比如我把 GitHub 通知设成 `ghn`、VS Code 项目搜索设成 `code`，肌肉记忆形成后操作极快。

## 窗口管理

Raycast 内置了窗口管理功能，不需要再安装 Magnet 或 Rectangle。预设布局包括：左半屏、右半屏、上半屏、下半屏、左上角、右上角、全屏、最大化、居中。每个布局可以绑定快捷键，我常用的配置是 `Cmd+Option+左箭头` 左半屏、`Cmd+Option+右箭头` 右半屏、`Cmd+Option+F` 全屏。

更重要的是，Raycast 窗口管理支持多显示器。你可以把窗口快速移到其他屏幕、对齐到网格、恢复到上一个窗口尺寸。对于使用超宽屏或双屏设置的用户，这是效率质变。

## Snippets 和剪贴板

Raycast 内置剪贴板历史（免费），记录你复制过的所有内容，支持搜索和固定。快捷键 `Cmd+Option+V` 呼出（可自定义），选择一个历史项直接粘贴。同样支持排除密码管理器等敏感应用。

Snippets 功能类似 Alfred——预设文本片段，输入关键词自动展开。用法和 Alfred Snippets 几乎一致，支持动态占位符（日期、时间、剪贴板内容等）。如果你从 Alfred 迁移，Snippets 的体验是无缝的。

## Quicklinks

Quicklinks 相当于 Alfred 的自定义搜索。你把一个搜索 URL 的查询部分替换为 `{Query}`，然后绑定一个关键字。例如 `https://www.npmjs.com/search?q={Query}` 绑定到 `npm`，在 Raycast 中输入 `npm react` 就直接打开 NPM 的 react 搜索结果。

Quicklinks 也支持打开本地文件路径、执行 Shell 命令。灵活性和 Alfred Custom Search 相当。

## Raycast AI

Raycast 还有一个独立的 AI 订阅（$8/月），整合了 GPT-4 等模型。在 Raycast 界面中可以直接提问、翻译、总结文本、生成代码、解释代码。AI 能力还会渗透到各个扩展中——比如在 GitHub 扩展里直接让 AI 总结 Issue 内容，在翻译扩展里使用 AI 翻译提高质量。

是否需要 AI 订阅因人而异。如果你已经订阅了 ChatGPT Plus 或者其他 AI 服务，Raycast AI 的价值就有限。但如果你希望所有操作（启动、搜索、AI 问答）都在同一个界面完成，这 $8 每月可能物有所值。

## 我的看法

对于开发者用户，Raycast 是目前综合体验最好的 Mac 启动器：免费、扩展多、UI 现代、更新勤快。它处理了我每天大约 80% 的「想做什么但不想开浏览器」的需求。

唯一需要提醒的是：Raycast 是 VC 投资的商业公司，未来免费策略可能调整。但目前来看，它们对个人用户免费的承诺是明确的，团队功能才是收入来源。如果你担心这个，Alfred 一次性买断更稳。

如果你还没试过，下载 Raycast（免费），安装五个你感兴趣的扩展，用三天时间感受一下。大概率你会忘记 Spotlight 的快捷键是什么。
