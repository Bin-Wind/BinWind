# 密码管理器推荐：1Password vs Bitwarden

密码复用是个人网络安全的头号风险。大多数人至少有三五十个在线账号，要求每个密码都不同且复杂（大小写字母 + 数字 + 符号 + 12 位以上）是人类大脑不可能完成的任务。密码管理器的任务就是替你记忆和生成这些密码，你只需要记住一个主密码就够了。

## 为什么必须用密码管理器

在没有密码管理器的情况下，绝大多数人会走两条路：要么所有网站用同一个或几个密码（一个网站被拖库，全网账号沦陷）；要么把密码写在记事本/备忘录/Excel 里（明文存储，不安全且不方便）。2023 年 HIBP（Have I Been Pwned）的数据显示，超过 120 亿条账号密码在公开泄露数据库中。你的某个常用密码大概率已经泄露了。

密码管理器的安全模型很简单：所有密码在离开你设备之前就已在本地加密，服务商存储的只是一堆加密后的乱码，没有你的主密码谁也解不开。这叫「零知识」架构——即使是服务商也无法访问你的数据。

## 1Password：设计驱动的安全

1Password 来自加拿大的 AgileBits 公司，从 2006 年至今一直是密码管理器领域的设计标杆。如果你需要的是「全家人都能轻松上手」的密码管理器，1Password 是最优选择。

1Password 的核心优势：
- **极致 UX**：浏览器扩展自动填充准确率极高，表单识别智能，很少需要手动复制粘贴。iOS 和 Mac 端的原生体验流畅，Face ID / Touch ID 解锁。这些细节让「使用密码管理器」这件事变得无感。
- **Watchtower（瞭望塔）**：内置安全监控功能，自动检测你的密码是否出现在已知泄露数据库中、是否使用了弱密码、是否重复使用、是否启用了双因素认证。它像一个安全顾问，持续告诉你哪些地方需要修正。
- **Travel Mode（旅行模式）**：可以在过境前从所有设备上删除指定密码库，过境后再恢复。对记者、律师、跨境商务人士来说，这是刚需功能。
- **家庭共享**：家庭版（$4.99/月，最多 5 人）可以让家人各自拥有私人密码库，同时共享 Netflix/HBO 等家庭账号。非技术背景的家庭成员也能轻松使用。
- **跨平台**：Mac、iOS、Windows、Android、Linux、浏览器扩展全平台覆盖。

1Password 的定价：个人版 **$2.99/月**（按年付费），家庭版 $4.99/月。没有免费版，但有 14 天免费试用。相比一杯咖啡的价格，为全账号安全买单并不贵。但目前 1Password 已转为订阅制，没有一次性买断选项。

## Bitwarden：开源的力量

Bitwarden 是 1Password 最强劲的竞争者。它采用开源模式，代码完全公开，经过了第三方安全审计。核心功能和 1Password 高度相似：密码生成、自动填充、跨平台同步、安全笔记、信用卡存储。

Bitwarden 的核心优势：
- **免费版极其慷慨**：无限设备同步、无限密码存储、密码生成器、自动填充。这些功能在 1Password 需要 $2.99/月，Bitwarden 完全免费。
- **Premium 仅 $10/年**：增加了高级双因素认证（YubiKey / FIDO2）、密码健康报告、文件附件、紧急访问（指定信任联系人可以在你无法登录时获取密码库访问权限）。
- **开源 + 审计**：代码托管在 GitHub，全世界安全研究者都可以审查。每年有独立第三方安全审计，报告公开发布。这种透明度是闭源产品无法提供的。
- **自部署选项**：你可以用 Docker 在自己的服务器上搭建 Bitwarden 服务端，完全掌控数据。开源社区还维护了用 Rust 重写的轻量级实现 Vaultwarden，树莓派就能跑。

Bitwarden 的不足：UI 相对 1Password 更「实用主义」，不够精致；移动端的自动填充体验不如 1Password 流畅；密码分享功能不如 1Password 方便；团队/企业功能成熟度不如 1Password 的 Business 方案。

## 安全模型对比

两者安全模型本质相同：AES-256 加密 + PBKDF2/Argon2 密钥派生 + 零知识架构。你的数据在发送到服务器之前已经在本地完成加密，服务商不持有你的加密密钥。两者的主密码都不会被记录或传输。

实际攻击面差异极小。1Password 额外使用 Secret Key（128 位随机密钥，在设备初始化时本地生成）来增强主密码的安全性，使得即使主密码偏弱，攻击者也难以暴力破解。Bitwarden 通过高迭代次数的 PBKDF2/Argon2 达到类似效果。从安全性的角度看，两者都足够可靠。

## 迁移与上手清单

两者都支持从 Chrome/Safari/LastPass/Dashlane 等源导入密码，支持 CSV 文件批量导入。从 1Password 迁移到 Bitwarden 或反之都可以在几分钟内完成。

新手配置清单：
1. 选择一个足够长且足够独特的主密码（推荐 Diceware 方法生成的词组密码，既安全又好记）。把主密码写在纸上，放在安全的地方。
2. 在密码库上启用双因素认证（TOTP 或安全密钥）。
3. 设置自动锁定超时时间（建议 5-15 分钟，根据工作环境调整）。
4. 在所有设备上安装浏览器扩展和移动 App。
5. 从浏览器导入已有密码，然后逐一修改为随机生成的强密码。
6. 以后新注册任何网站，一律用密码管理器生成随机的 16 位以上密码。

## 结论

如果你预算允许，追求最好的用户体验和家庭共享，选择 **1Password**。$2.99/月换来的是设计精美、全家人都会用的密码管理器。尤其推荐家庭版给有非技术背景家人的用户。

如果你追求免费、透明、开源，选择 **Bitwarden**。免费版功能已经足够覆盖 95% 的需求。$10/年的 Premium 是同类产品中性价比极高的付费选项。

无论选择哪一款，最重要的是开始使用密码管理器。不用的风险远大于选错的遗憾。
