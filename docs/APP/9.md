# 任务管理应用深度评测：Things vs Todoist vs TickTick

先说一句实话：没有任何任务管理 App 能让你变得高效。高效来自清晰的目标、合理的优先级和坚持执行的习惯。但反过来，一个不适合你的 App 确实会拖后腿：信息录入繁琐让你懒得记录、提醒不及时导致错过截止日期、架构不符合你思维习惯让你每次规划都觉得别扭。选对工具的意义在于消除摩擦，让记录和执行变得无感。

## Things：设计即哲学

Things 是 Cultured Code 公司的作品，Mac + iOS + iPad + Apple Watch 独占。它是我见过设计最美的任务管理 App，没有之一。界面干净到近乎禅意，每种交互都经过了深思熟虑。

Things 的组织结构：Areas（领域，如「工作」「个人」「学习」）→ Projects（项目，如「网站重构」「装修」「旅行计划」）→ Headings（标题，项目内的分组）→ To-Dos（具体任务）。任务支持添加 Checklist（子步骤）、Deadline（死线，与计划日期分开的概念）、Tag（标签，用于筛选）、When（计划执行日期，可以设「某天」「今晚」）。

这个结构的精妙之处在于「When」和「Deadline」的分离。Deadline 是「不做就会出事的日期」，When 是「我计划在这天开始做」。你有 5 天后到期的报告，但计划今天开始写第一步；你周五有个提案截止，但周三得完成初稿给领导过目。这种双日期设计是对 GTD 理念的优雅实现。

Things 的不足同样明显：没有协作功能（纯个人工具）；不支持附件（只能放链接）；没有自然语言输入（Todoist 的强项）；平台锁定（Mac + iOS，无 Windows/Android/Web 版）；价格较高（Mac $49.99 + iOS $9.99 + iPad $19.99 = 全套近 $80）。对于苹果全家桶用户来说是一次性投资，但对于跨平台需求者根本不可用。

**最适合**：苹果生态用户、追求美观和体验的个体、GTD 轻度实践者。

## Todoist：自然语言与跨平台之王

Todoist 是跨平台任务管理的事实标准。Windows、macOS、Linux、iOS、Android、Web——全平台覆盖，同步速度极快。免费版已能满足基础需求（300 活跃任务、5 个活跃项目、5 个协作者）。

Todoist 的最大特色是自然语言输入。在任务框中输入「每周二下午三点给妈妈打电话 #家庭 @电话 p1」，Todoist 自动解析为：循环任务（每周二 15:00）、所属项目（家庭）、标签（电话）、优先级（p1 / 最高）。不需要点日期选择器，不需要手动设置重复规则，打字就行。这对效率党来说极其实用。

Todoist 的筛选器（Filters）是另一个高级功能。用自定义查询语法创建自定义视图：`today & p1`（今天的高优先级任务）、`@phone & overdue`（需要打电话且已过期的任务）、`#工作 & !@waiting`（工作项目中不包含等待标签的任务）。学会筛选器之后，Todoist 变成了一套可编程的任务查询系统。

Karma 积分系统是 Todoist 的激励设计：完成任务获得积分，连续完成日目标获得升级。对喜欢量化自我的人有效，对无感的人可关闭。团队协作功能完善：任务分配、评论、附件、共享项目。

Premium 订阅 **$4/月**（按年付），解锁提醒、标签、筛选器、日历视图等高级功能。免费版和 Premium 的差异在中等偏轻用户身上体现不明显，重度用户建议升级。

**最适合**：跨平台用户、需要团队协作、喜欢自然语言输入、习惯量化自我追踪的人。

## TickTick：一切尽在一处

TickTick 定位在一站式生产力工具：任务管理 + 番茄钟 + 习惯追踪 + 日历视图 + Eisenhower 矩阵（紧急/重要四象限），全部内置。如果你不想在任务管理、番茄钟、习惯打卡之间切换三个 App，TickTick 是唯一一个整合了所有功能且做得不错的选择。

TickTick 的任务管理本身中规中矩：列表、标签、优先级、子任务、附件、评论。日历视图支持拖拽调整日期。Eisenhower 矩阵帮你把任务分成四象限，这个功能虽然在电子工具上的必要性存疑（四象限模型更适用于纸上规划），但至少它在那里。

番茄钟和习惯追踪是 TickTick 的差异化亮点。内置番茄钟可以和任务关联（选定一个任务开始计时），完成后自动记录专注时长。习惯追踪支持每日打卡、周频率目标、趋势统计。你可以在一个 App 里完成「计划今日任务 → 番茄钟专注执行 → 回顾习惯打卡」的完整闭环。

TickTick 的定价偏实惠：免费版已包含任务管理、番茄钟、习惯追踪的基础功能。Premium 版 **$2.79/月**解锁日历视图、更多筛选器、自定义番茄钟时长、历史统计。这个价格是三款中最便宜的付费方案。

TickTick 的不足：UI 设计偏功能性，美观度不如 Things；App 整体体验偏「功能堆砌」，有时候会觉得设置菜单太多；国际化上不如 Todoist（社区、集成、API 成熟度更低）。

**最适合**：需要任务 + 番茄钟 + 习惯一站式解决的人、预算敏感的用户。

| 维度 | Things | Todoist | TickTick |

## 我的推荐

**苹果全家桶 + 个人使用**：Things。贵但好。用久了你会理解为什么 Things 用户在谈到任务管理时有一种「溺爱」的语气。它不是功能最全的，但日常使用最舒适。

**跨平台或需要协作**：Todoist。自然语言解析是真的效率提升，筛选器让你的任务系统随复杂度增长而生长。免费版先试试，够用了就一直用免费版；不够用了升级 Premium，$4/月不多。

**想要一个 App 搞定一切**：TickTick。番茄钟和习惯追踪内置意味着少装两个 App、少一份认知负担。$2.79/月价格温柔，功能全面。只是在设计和细节打磨上不如前两者精致。

最后再次强调：选一个开始用，坚持一个月。不要在选择工具上花了 3 天、实际用了 3 天就放弃。任何工具都需要时间形成肌肉记忆和思维习惯。
