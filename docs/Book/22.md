# 网络文学精品化：值得一读的网文推荐

提到"网络文学"，很多人的第一反应是"爽文"、"快餐"、"套路"。这个刻板印象不是没有原因的——网文的付费模式（按章节计费）本质上鼓励篇幅膨胀、节奏灌水。但同样不可否认的是，过去二十年间，一批在商业平台上成长起来的作者，已经写出了足以与传统出版文学平等对话的作品。他们用几百万字的体量、千章左右的连载长度，搭建起了只有网文媒介才能提供的沉浸式世界观和深度人物弧线。

以下推荐的作者和作品，每一部都是网文在"精品化"道路上交出的最高分答卷。

## 猫腻：当哲学家写网文

猫腻可能是中国网文圈里最具文学自觉的作者。他不是"不小心"写得好——他是刻意地用文学标准要求自己，只是恰好选择了网文这个媒介。

《将夜》（2011-2014）是猫腻美学最集中的一次展现。大唐气象的架空世界里，一个从小被杀手养大的少年宁缺，背着一把大黑伞和一个小侍女桑桑，一步步从边城走到都城、从书院走到天上。宁缺的成长不靠奇遇金手指——靠的是一次次被命运打趴下之后的"我不服"。全书最大的命题在于宁缺和天道的对抗：天要毁灭人类，宁缺说那我们把天杀了。这是一种强烈的、在传统文学中不好直接说的人文主义宣言。

《庆余年》（2007-2009）可能更广为人知——它被改编成了一部现象级的电视剧。范闲从现代穿越到一个虚构的封建王朝，他的现代记忆和古代权力之间的张力构成全书的核心戏剧。但和电视剧不同，原著范闲更世故、更计算、更不像一个英雄。这是一种更具网文诚实的写法：一个来自现代的灵魂，在一个丛林法则的社会里究竟会变成什么？

《间客》（2009-2011）是猫腻至今最具野心的作品。表面上是一部"星际战争加机甲格斗加武侠"的混搭，但骨架是一系列关于权力和制度的命题：个人对抗体制需要多少成本？暴力和和平的选择什么时候开始本质一样野蛮？开头很慢——前几十章对你来说可能像一堆不知所云的世界设定——但一旦进入，你的阅读会像逃离不了引力场一样被拉进去。

## priest：多面手的极致

priest是个神秘的存在——没人知道她长什么样，但她的书每本都是不同题材的高水平演出，而且她做了一样绝大多数网文作者做不到的事：她写得紧。在按章付费唾手可得地鼓励玩家"把书灌到一千章"的世界里，priest的长篇几乎从不水。

《默读》（2016-2017）可能是中国当代最好的犯罪小说之一，不用加"网络"前缀。骆闻舟和费渡这对刑侦搭档推的每一个案子都以一部世界文学经典命名——第一个案子叫"于连"——《红与黑》的于连——一系列看来独立的小案最终解开费渡自身的悲剧。对话是priest的顶尖技能。你读priest像是在看一本文学作品和一段单口相声轮流上演。

《有匪》（2015-2016）是一封致武侠小说的情书。女性主人公周翡，一个真正靠自己磨剑磨出血泡的武学天才，从头至尾你没有一次怀疑这个角色的力量感不是发自内在的。被改编为电视剧后火得不成样子，但书是好的——书里那种"我就是要在所有人都觉得我不行的地方站起来"的倔劲，电视无法完全还原。

priest最让人惊叹的一本书可能是《残次品》（2017-2018）——一部史诗级别篇幅的太空歌剧。一个被星际流放的将军和一个病入膏肓的人工智能老师，用一起重建一所学校开始反抗整个银河系统的星系级暴政。priest在这个故事里讨论了一个经典的科幻命题（自由与安全的交换），但她是从一群"残次品"——被系统淘汰、被父母遗弃、被社会视为失败者的小人物——的视角切入的。读完之后你可能很难看着"网文"二字继续保持轻蔑。

## 烽火戏诸侯：雪中的诗

《雪中悍刀行》（2013-2016）是一部武侠史诗，不过烽火的武不是"打"的，是"写意"的。他不用拳拳到肉的格斗，而是用叙述来构造一种"这剑还没出鞘但你的后脖颈已经感觉到它的寒意"的氛围。主角徐凤年表面上是一个懒如死狗的北椋世子，大家以为他不学无术，但他北凉铁骑横行天下，不为他自己——是为守住他父亲用命打下的边关。

《雪中悍刀行》的配角群是网文中少有的"每一个配角都能独立撑起一本书"的阵容。剑神李淳罡、春秋三甲中的王仙芝、大秦皇帝曹长卿——烽火给每个角色写了自己的史诗，然后用主线串起来。这本书号称"网文第一群像剧"，这个称号太过分但又好像没有更合适的。词藻是骚客式的，有些读者会觉得太华而不实，但如果你接受了它的语言设定，这部书会催眠。

## 蝴蝶蓝：游戏外壳下的体育精神

蝴蝶蓝的《全职高手》（2011-2014）讲的是一名电竞职业选手叶修被俱乐部扫地出门后从零开始、一砖一瓦重建自己的故事。但别因为"电竞"二字就归类它为宅男快餐——它是这个国家写得最好的体育小说，它的体育恰好是电子竞技而已。

叶修是一个反转了中国文学常见的"天纵奇才"的写法：他真有过强者才有的从容，从不狗血，从不轻浮，从不自暴自弃。整个故事没有任何爱情线，所有的情感张力都在于一个接一个的竞赛战场。你会在凌晨两点不肯放，因为下一场团队赛要开了，而你没办法不知道谁赢。

读蝴蝶蓝你能忘掉网文的臃肿——他的叙述干净利索，配角上百个但你个个分得清。幽默感是天生的，而且他从不用愚蠢的情节来制造冲突。

## 关心则乱：宅斗文中的红楼梦味

《知否知否应是绿肥红瘦》是一部把宋代家庭生活和政治博弈织成锦缎的巨作。女主角盛明兰是庶女，母亲被人害死，父亲不在乎她，嫡母严苛，祖母是她唯一的靠山。在这样一个家庭里长大，明兰学会的唯一战斗方式就是活着、聪明地活着、比别人更沉得住气地活着。

全书从明兰幼年写到中年，节奏等同于生活——小到连一座院的伙食怎么分配、大到皇位变动和家族兴亡——全部像过日子一样自然展开着。它不是"励志"，不告诉你什么道理，但你跟明兰待了这么多章，某一天在现实生活中碰壁却居然不慌了——你发现你已经不自知地吸收了某种克制的、耐压的智慧。

改编的电视剧很成功，但在心里，《知否》是一部还没有被影像完全译尽的好书。

## 从哪里开始

如果你关心文学的质地，从猫腻开始。《庆余年》或《间客》都可以。如果你担心"网文太长"，从priest的《默读》入手——这其实是一本完全可以在传统出版社出书的不算太长的长篇。如果你只是想要一份纯粹的快乐，蝴蝶蓝的《全职高手》正等着你。如果你说我对家庭情感戏也有兴趣——关心则乱的《知否》，你在那里面待上许多天也不会想出来。

网文在精品化的路还远没走完。但上面提到的这些书代表了一件事：在一个被轻视的媒介里，依然可以长出好的东西。
