# 建立个人图书馆：选书、购书与藏书指南

一个人的书架，是这个人的思想自传。不是写出来的自传，是砌出来的。你站到一个爱书人的书房中间，看一眼——哪些书卷边翻烂了，哪些书脊崭新锃亮从来没有被翻开过，哲学挤在推理旁边，还是按国别从东往西排——你不需要任何对话，就已经读到了这个人一生的智力路径。

这篇文章写给想认认真真开始"收藏书"而不仅仅是"买书"的人：怎么选书、怎么买对版本、怎么把书架管理好。

## 为什么要有自己的藏书

电子书可以在另一个硬盘上被擦除，图书馆借的最终要还，但一本属于你的书——你某年某月某日在哪家书店第一次翻开、被哪句话震到、在书的第几页留下了哪一种划线——这本书是你时间的化石。你的书架上摆的不是纸，是你过去所有的自己。

另外一个更现实的好处：实体书的"可见性"是数字目录无法替代的。你在Kindle里有五百本书，和你在墙上看到一排书脊，这是两种不同的信息刺激。你在路过自己的书架时，眼角扫到的那本书的名字，会莫名其妙激活你脑子里的一道记忆或者一个创作念头。这种"随机触发"是实体书对人类大脑最被低估的功能。

当然，不是说"唯有实体"。数字和实体可以互相补充。实体的价值是不插电的恒久、视觉的存在、可以借给朋友；数字的价值是检索、到处带着不重、关上灯也能读。不需要选阵营。需要的是选"哪些书值得买一份实体的"。

## 买的不是每本你读的书

这是新手最常犯的错：读完一本好书就立刻下单、第二年发现家里被书塞满了、搬一次家跪一次。

简单原则：图书馆或借阅适用于第一遍读；买的书是你确定会重读或查阅的。小说尤其严格——除非这本书是你的"年度之书"，你预感十年后还会再翻开，或者它的某个段落你希望有天能在纸面上重新遇到，否则让它留在图书馆的书架上。你买它来它就只是躺在你家，不会给你第二种生活。

## 版本知识：基础入门

同是一本书，不同的版本是两本书。

精装：耐久、漂亮、贵、搬家时沉死。对长期存放或者你想传给孩子的书值得；对一次读完就过的悬疑小说极不值。

平装：便携、便宜、脊会裂。文学经典的平装版可以选择"企鹅经典豪华版"（Penguin Classics Deluxe Editions）——漂亮的毛边封面、通常有更好的注释、价不贵。

学术评注版：如果你读文学经典（比如莎士比亚、简·奥斯汀、陀思妥耶夫斯基），诺顿评注版（Norton Critical Editions）是业内最靠谱的——有导读论文、有历史背景材料、每一版可能有好几个前后修订的文本注释。对"研究型"读者来说是恩物。

电子书不用我说。但有一个冷知识：亚马逊不是电子书唯一的选择。独立书店聚合平台Libro.fm支持独立书店（有声书），Kobo电子书店多语言支持和格式开放性都优于Kindle。如果你想"拥有"自己的电子书而不是租用，记得找DRM-free的版本。

## 中文出版版图：认准这些品牌

你站在书店的新书区，面对成千上万个封面。怎么办？认出版品牌——这些出版品牌有自己的审美和知识立场。以下是中国读书人公认的高品质出版品牌：

- 三联书店——学术、社会、历史。社科读物如果是三联出的，你就可以多信几分。
- 中华书局——古典中国。如果你想读诸子百家、二十四史断代、唐宋笔记，不要挑别的。
- 上海译文出版社——翻译文学的头牌。译文社有百年历史，很多作家在他们家出全集——昆德拉、杜拉斯、村上春树的中文版权他们包了。翻译质量整体稳定。
- 新星出版社——午夜文库。国内悬疑推理读者都知道"午夜文库"四个字代表的含金量。夜间版本的小开本精装设计是全中国最美的口袋书系列之一。
- 理想国——台湾文化人主创，知识分子的文选。既有社科大部头也有精致散文，是M系列的精装典藏版让很多人开始认真对待"买一个好看版本"这件事的。理想国译丛的选题常常是冲击性的——不一定会让你舒服，但会让你想。

买书不只是买内容，也是选你信任谁的审美的。

## 书架整理：一种个人哲学

选一种适合你的分类法。按类型排、按国别排、按作者字母排——都是OK的。按颜色排？这是全世界爱书人内战的总爆点（好看但完全不实用）。按"已读 vs 未读"排列更诚实——一半以上的爱书人都有未读的书，把它们藏起来只是自我欺骗。日本甚至有"tsundoku"一词——积読，买来不读——这是个世界性问题。

我自己的方法：虚构小说按作者的姓的拼音首字母排，A到Z（或A到Z分中英文），非虚构按主题聚块（历史类、科学类、哲学类、批评类），同一主题内再按字母排列。关键只是"你能找到你想找的书"。如果你做不到这一点，再美的书架也只是墙面装修。

## 除草：定期清理书架

没什么比"占有"一堆你其实不喜欢的书更压制你的阅读欲了。每隔一两年，你应该坐下来，把书架上每一本书拿起来问自己三个问题：

我会把这书推荐给一个朋友吗？——如果答案"不会而且也不想"，那它占着这个架子干什么？
我会真的重读这本书吗？——如果是"大概吧"就等于"不会重读"。把"不重读"的和图书馆没区别的书清出来，送给别人。书是用来读的，不是用来当家具的。
它代表了过去的某一个你，而你已经不再是那个人了吗？——这个不需要解释，你明白我在说什么。

近藤麻理惠说的"spark joy"也可以用在这里：拿起一本书，如果它让你胸口一紧——留下；如果它让你想起的可能是一堂没学成的法语课挫败感——送走。书不是墓志铭，不是每本读过的都必须为你的余生守灵。

## 编目统计

你终有一天会忘记自己到底有没有买过某本书。如果不信，现在就去你的书架上检查，有没有一本书你完全不记得买过但确实在那里。解决方案是编目：Goodreads或豆瓣的"拥有"书架功能，LibraryThing（可能是世界上最严谨的私人藏书数据库），或者最简单但极其有效的方法——拿手机拍一整排。保险起见，拍得清楚一点（书脊字要读得出来）。万一哪年哪月发生什么，保险公司的记录里你需要它们的证据。

## 结语：从书架看你

不要因为"别人家的书架是这样布局的"就去复制。你的书架你说了算。如果你所有的书都是推理小说，就把书架建成推理小说朝圣地。如果你读书是为了向外借——你的书架可能还有一本借书登记本。无论怎么设计，原则一条：让它出现你喜欢回到那个角落的状态。除此之外，一切所谓标准都是第二位的。
