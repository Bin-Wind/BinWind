# 二战文学精选：在废墟中寻找人性

二战是一场改变了文学基因的战争。在此之前，战争在文学中还可以披着荣耀的外衣——荷马史诗里的英雄、骑士文学中的荣誉、甚至一战初期的那种悲壮感。奥斯维辛之后，一切都变了。在纳粹集中营和原子弹之后，你怎么还能写一首乐观的诗？你怎么还能相信文明在进步？二战文学面对的是人类文明最大的破产，而它要回答的问题只有一个：在见证了人可以对人做出什么之后，"人性"这个词还能不能继续用。

这篇导读选择了八本书。它们从不同的角度切入那场战争——空中、地面、加害者的国、受害者的营、欧洲的废墟和亚洲的土地。每本书都是一个入口，通往同一个深渊的不同层面。

## 《五号屠场》（库尔特·冯内古特，1969）

冯内古特在德累斯顿大轰炸中活了下来。1945年2月，盟军对这个没有军事价值的德国文化古城进行了地毯式轰炸，大约两万五千名平民被烧死。冯内古特当时是一名美军战俘，被关在德累斯顿的地下屠宰场里。他从屠宰场爬出来，看到了烧成焦炭的城市。

二十三年来他想写这件事，写不出来。当他终于写出来时，他创造了一个被外星人绑架、从时间的顺序中解脱出来的主角比利·皮尔格林。比利的时间是碎片化的——他这一刻在德累斯顿的瓦砾中搬尸体，下一刻在纽约郊区的验光店里配眼镜，再下一刻在一个外星动物园的玻璃罩里。每次面对死亡，全书就出现四个字："事情就是这样。"

冯内古特发明了这个短语，也用它消解了所有关于战争光荣的叙事。德累斯顿死了两万五千人。事情就是这样。不要试着从这个句子中寻找意义——它拒绝给你意义，而拒绝本身就是最有力量的反战宣言。

## 《第二十二条军规》（约瑟夫·海勒，1961）

如果你只想读一本关于战争荒谬性的书，就是这一本。

二战期间，美军轰炸机飞行员约塞连想从战争中脱身。他找到了一个法律漏洞：第二十二条军规。这条军规规定，如果一个飞行员疯了，他可以申请停飞。但申请停飞这个动作本身证明你在意自己的生命安全，而一个在意自己生命安全的人没有疯。所以你将永远飞下去。

第二十二条军规不是法律，它是一个逻辑陷阱，一种用语言建造的监狱。这本书读起来像一个无休止的白日噩梦：每一章的喜剧效果都叠加在前一章的恐怖之上，直到笑不出来了。一个叫米洛的军需官用军队的资源建立了一个跨国企业，最后替德国空军轰炸了自己的空军基地——从公司的资产负债表上看，这笔交易是盈利的。

海勒在后记里说过一句让我久久说不出话的话：这本书里最荒谬的情节，反而都是真的发生过的事。那些看似疯狂到不可能的了情节——他在军队里亲眼见过。

## 《偷书贼》（马克斯·苏萨克，2005）

如果一本书的叙述者是死神本人，它可能会是什么语气？苏萨克给出的答案是：疲惫、温柔，和一种对人类的困惑。

莉赛尔是一个德国女孩。1939年，她的母亲把她和弟弟送往慕尼黑郊外的寄养家庭。弟弟在路上死了。在埋葬弟弟的雪地里，莉赛尔偷了第一本书——《掘墓人手册》。她还不识字，但她知道书里有她失去的东西。

莉赛尔的寄养父母汉斯和罗莎·胡贝尔曼是这本书最伟大的创造。汉斯是一个油漆匠，他在莉赛尔半夜做噩梦时陪她坐到天亮，用一架手风琴和一盏灯，一个字一个字地教她读第一本书。罗莎是一个粗声粗气、动不动骂人"母猪"的女人，但当犹太人马克斯躲在他们家的地下室里时，她是第一个把一碗汤推到他面前的人。

《偷书贼》的力量不是写了集中营，而是写了战争中的德国平民——他们同样被希特勒的机器碾碎，同样在恐惧和饥饿中保存着人性。死神在结尾说了一句他被人类弄得很疲惫。读到最后你也会很疲惫，但你也会记住汉斯·胡贝尔曼油腻的杏色油漆桶和莉赛尔偷来的每一本书上的灰尘。

## 《所有我们看不见的光》（安东尼·多尔，2014）

玛丽-洛尔是一个盲童，住在巴黎自然历史博物馆附近。第二次世界大战爆发时她和父亲逃到了圣马洛——法国海岸的一座被城墙环绕的古城。维尔纳是一个德国孤儿，他因为修理收音机的天赋被选拔进一所纳粹精英学校，然后被送往前线追踪抵抗组织的无线电信号。

这本书的语言是抒情而精确的——每一章短得像一颗子弹，交替切换着两个孩子的视角。多尔用玛丽-洛尔的触觉和听觉写她的世界：海螺壳的螺旋、父亲手刻的木制街区模型、圣马洛城墙上的鸟鸣。他用维尔纳的无线电写战争的声音：是音乐把这两个灵魂连在一起的——圣马洛阁楼上播放的德彪西。

全书两条叙事线在1944年的圣马洛围城战中汇合。那一刻——我不做更多剧透——那一刻，整个普利策奖是值得的。

## 《夜》（埃利·维瑟尔，1956）

这是一本不到一百页的书。读完它可能不到两个小时。但它携带的重量，足以让你接下来一个星期的睡眠都不那么安稳。

埃利·维瑟尔在十五岁那年被送进了奥斯维辛。他和父亲在里面待了一年。他从那里活着出来，然后在法国沉默写作了十年。最终他的手稿被压缩到只有一百二十七页的短书，拿掉了所有修饰，只留下必须被记下的事实。

书里有一个场景：一个孩子被吊死在集中营的广场上。所有囚犯被迫观看。有人在维瑟尔身后问："上帝在哪里？"维瑟尔听见自己心里有一个声音回答："他在这儿——他正吊在这个绞刑架上。"

这是二战文学中最沉重的一句话。维瑟尔不是一个无神论者，他是一个被上帝背叛的信徒。《夜》不是为了让人"理解"浩劫——它不是为了理解而写的。它是为了见证。为了对得起那些没能活着讲述的人。

## 《深入北方的小路》（理查德·弗拉纳根，2013）

弗拉纳根的父亲是澳大利亚战俘，被日军强迫修建泰缅铁路——那条以每一条枕木下面埋着一个战俘而闻名的"死亡铁路"。弗拉纳根为这本书花了十二年，五易其稿，最终获得了曼布克奖。

书的中心人物多里戈·埃文斯是一个澳大利亚军医、一个被爱情钉在十字架上的人。泰国的丛林里，他被囚禁在世上最接近地狱的地方——饥饿、霍乱、疟疾、殴打和死亡像雨季的蚊子一样无处不在。他幸存下来的理由不是求生的意志，而是一段他和军队同僚的妻子之间发生的爱情——在战争开始之前就已经铸成的、注定无疾而终的爱情。

弗拉纳根的选择令人胆寒又令人敬佩：他不仅写战俘营的残酷，还写战后——那些幸存下来的人如何度过剩下的五十年。活下去的人不一定比死去的人幸福。这本书给了每一个幸存者一条漫长的余生，看着他们被记忆慢慢掏空。

## 中国视角

哈金的《南京安魂曲》是南京大屠杀过去七十年后，一个用英语写作的中国人对这个伤口的触碰。全书以金陵女子学院的明妮·魏特琳的视角展开——美国传教士，在南京建立难民营拯救了近万名中国妇女儿童。哈金的笔极尽可能之克制——他选择不渲染暴力细节，而是写魏特琳在保护了那么多人之后，依然在余生中被"做得不够"的念头吞噬。战争结束多年之后，这个救了无数中国女人生命的外国女人自杀了。

余华的《活着》不是严格意义上的"二战小说"——它的时间跨度从民国到新中国，战争只是福贵苦难的一部分。但那些兵荒马乱中的"活着"就足够沉重了。从战场上把战友的尸体背回来的那种活着。

## 如何负责任地阅读二战文学

我必须说一件可能扫兴的事：二战文学对我们来说是书架上的选择，对亲历者是压在骨头里的铁。我们在咖啡馆里、在床头灯下翻开一本《夜》或《五号屠场》，几个小时后又放回书架。这是奢侈。意识到这种奢侈，是阅读二战文学最基本的准备。

带着敬畏打开这些书。不是为了获得"感动"，而是为了在八十年后，让那些没能说话的人，借你的眼睛再看一眼他们再也看不到的世界。
