# Java 开发环境搭建

万事开头难，搭建好 Java 开发环境是学习的第一步。本文将带你从零开始安装 JDK、配置环境变量、安装 IDE，并运行第一个 Java 程序。

## JDK（Java Development Kit）安装

Java 程序运行在 JVM（Java Virtual Machine）之上，编写代码需要 JDK，运行只需要 JRE（Java Runtime Environment）。推荐安装长期支持（LTS）版本：**JDK 17** 或 **JDK 21**。

下载方式：
- **Oracle JDK**：[oracle.com](https://www.oracle.com/java/technologies/downloads/) — 官方版本，商用需注意授权
- **OpenJDK**：[adoptium.net](https://adoptium.net/) — 开源免费，企业首选
- **包管理器**：macOS 用 `brew install openjdk@21`，Ubuntu 用 `sudo apt install openjdk-21-jdk`

验证安装：

```bash
java -version
# openjdk version "21.0.1" 2023-10-17 LTS
javac -version
# javac 21.0.1
```

## JAVA_HOME 环境变量

`JAVA_HOME` 指向 JDK 安装目录，Maven、Gradle、Tomcat 等工具都依赖它。

**macOS / Linux** — 编辑 `~/.zshrc` 或 `~/.bashrc`：

```bash
export JAVA_HOME=$(/usr/libexec/java_home)
export PATH=$JAVA_HOME/bin:$PATH
```

**Windows** — 系统环境变量中新增 `JAVA_HOME`，值为 `C:\Program Files\Java\jdk-21`，并将 `%JAVA_HOME%\bin` 加入 `Path`。

## 构建工具：Maven 和 Gradle

现代 Java 项目离不开依赖管理和构建自动化。

- **Maven** — XML 配置（`pom.xml`），约定优于配置，生态成熟
- **Gradle** — Groovy/Kotlin DSL，增量构建，性能更优

Maven 最小 `pom.xml` 示例：

```xml
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0
                             http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.example</groupId>
    <artifactId>hello-world</artifactId>
    <version>1.0-SNAPSHOT</version>
</project>
```

## IntelliJ IDEA

推荐使用 **IntelliJ IDEA Community Edition**（免费）作为主力 IDE。

- 下载：[jetbrains.com/idea/download](https://www.jetbrains.com/idea/download/)
- 安装后配置 JDK：`File → Project Structure → SDK`，选择已安装的 JDK 路径
- 推荐安装插件：SonarLint（代码质量）、Key Promoter X（快捷键提示）

## Hello World

创建 `HelloWorld.java`：

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

编译并运行：

```bash
javac HelloWorld.java   # 编译生成 HelloWorld.class
java HelloWorld          # JVM 执行字节码
# 输出: Hello, Java!
```

要点解析：
- `public class HelloWorld` — 类名必须与文件名一致
- `public static void main(String[] args)` — 程序入口，签名固定
- `System.out.println()` — 标准输出，带换行
