# Lambda 表达式与 Stream API

JDK 8 引入的 Lambda 表达式和 Stream API 彻底改变了 Java 的编程范式，让集合操作从命令式转向声明式，代码更简洁、更易并行。

## 函数式接口（Functional Interface）

只有一个抽象方法的接口称为函数式接口，可用 `@FunctionalInterface` 注解标注。Lambda 表达式正是函数式接口的实现：

```java
@FunctionalInterface
interface Calculator {
    int calc(int a, int b);          // 唯一的抽象方法
}

// 传统匿名内部类
Calculator add = new Calculator() {
    @Override
    public int calc(int a, int b) { return a + b; }
};

// Lambda 等效写法
Calculator add2 = (a, b) -> a + b;
Calculator sub = (a, b) -> a - b;
System.out.println(add2.calc(1, 2));  // 3
```

JDK 内置的核心函数式接口（`java.util.function`）：

| 接口              | 方法             | 说明              |
|-------------------|------------------|-------------------|
| `Function<T,R>`   | `R apply(T t)`   | 转换函数          |
| `Consumer<T>`     | `void accept(T)` | 消费者（无返回）  |
| `Supplier<T>`     | `T get()`        | 供应者（无输入）  |
| `Predicate<T>`    | `boolean test(T)`| 断言              |
| `BiFunction<T,U,R>` | `R apply(T,U)` | 双参数函数        |

## Lambda 语法

```java
// 完整形式：(参数列表) -> { 方法体 }
(int a, int b) -> { return a + b; }

// 省略类型（类型推断）
(a, b) -> { return a + b; }

// 单行省略 return 和花括号
(a, b) -> a + b;

// 单参数省略括号
s -> s.toUpperCase();

// 无参数
() -> System.out.println("Hello");

// 方法体多行时保留花括号
(x, y) -> {
    int result = x + y;
    System.out.println("result = " + result);
    return result;
};
```

## 方法引用（Method Reference）

Lambda 的简写形式，四种类型：

```java
import java.util.function.*;

// 1. 静态方法引用：Class::staticMethod
Function<String, Integer> parser = Integer::parseInt;
System.out.println(parser.apply("123"));  // 123

// 2. 实例方法引用（特定对象）：object::instanceMethod
String prefix = "Hello, ";
Consumer<String> printer = System.out::println;  // out 是具体对象
printer.accept(prefix + "World");                 // Hello, World

// 3. 实例方法引用（任意对象）：Class::instanceMethod
Function<String, Integer> length = String::length;
System.out.println(length.apply("abc"));  // 3

// 4. 构造方法引用：Class::new
Supplier<ArrayList<String>> factory = ArrayList::new;
List<String> list = factory.get();
```

## Stream API 核心操作

Stream 不是数据结构，而是对数据源的流水线式操作。分为三类：

### 创建 Stream

```java
// 从集合
Stream<String> s1 = List.of("a", "b", "c").stream();
Stream<String> s2 = Set.of("x", "y").stream();

// 从数组
Stream<Integer> s3 = Arrays.stream(new int[]{1, 2, 3}).boxed();

// 直接创建
Stream<String> s4 = Stream.of("one", "two");
Stream<Integer> s5 = Stream.iterate(0, n -> n + 2).limit(10);  // 0,2,4,...,18
Stream<Double> s6 = Stream.generate(Math::random).limit(5);
```

### 中间操作（Intermediate Operations）— 惰性求值

```java
List<String> words = List.of("Java", "Python", "Go", "Rust", "Kotlin");

// filter 过滤
words.stream()
    .filter(w -> w.length() > 3)
    .forEach(System.out::println);  // Java Python Kotlin

// map 映射
words.stream()
    .map(String::toUpperCase)
    .forEach(System.out::println);

// flatMap 扁平化
List<List<String>> nested = List.of(
    List.of("a", "b"), List.of("c", "d")
);
nested.stream()
    .flatMap(Collection::stream)
    .forEach(System.out::print);  // abcd

// sorted, distinct, skip, limit
words.stream()
    .sorted()
    .distinct()
    .skip(1)
    .limit(3)
    .forEach(System.out::println);
```

### 终端操作（Terminal Operations）— 触发执行

```java
List<Integer> nums = List.of(3, 1, 4, 1, 5, 9, 2, 6);

// collect 收集
List<Integer> sorted = nums.stream()
    .sorted()
    .collect(Collectors.toList());
System.out.println(sorted);  // [1, 1, 2, 3, 4, 5, 6, 9]

// 分组
Map<Integer, List<Integer>> byMod = nums.stream()
    .collect(Collectors.groupingBy(n -> n % 3));
System.out.println(byMod);  // {0=[3,9,6], 1=[1,4,1], 2=[5,2]}

// reduce 归约
int sum = nums.stream()
    .reduce(0, Integer::sum);  // 等价于 .reduce(0, (a, b) -> a + b)
System.out.println(sum);  // 31

// 统计
long count = nums.stream().filter(n -> n > 5).count();
int max = nums.stream().max(Integer::compareTo).orElse(-1);
boolean allPositive = nums.stream().allMatch(n -> n > 0);

// 查找
Optional<Integer> first = nums.stream()
    .filter(n -> n > 5)
    .findFirst();      // Optional[9]
Optional<Integer> any = nums.parallelStream()
    .filter(n -> n > 5)
    .findAny();        // 并行流中任意匹配元素
```

## Collectors 常用操作

```java
import java.util.stream.Collectors;

List<Person> people = List.of(
    new Person("Alice", 30),
    new Person("Bob", 25),
    new Person("Charlie", 35)
);

// toList, toSet, toMap
Map<String, Integer> nameAge = people.stream()
    .collect(Collectors.toMap(Person::getName, Person::getAge));

// joining 字符串拼接
String names = people.stream()
    .map(Person::getName)
    .collect(Collectors.joining(", ", "[", "]"));

// groupingBy + counting
Map<Integer, Long> ageCount = people.stream()
    .collect(Collectors.groupingBy(Person::getAge, Collectors.counting()));

// partitioningBy
Map<Boolean, List<Person>> adultPartition = people.stream()
    .collect(Collectors.partitioningBy(p -> p.getAge() >= 30));
```

## 并行流（Parallel Stream）

```java
// 大数据量场景可提升性能（注意线程安全）
List<Integer> largeList = IntStream.rangeClosed(1, 10_000_000)
    .boxed()
    .collect(Collectors.toList());

long count = largeList.parallelStream()
    .filter(n -> n % 2 == 0)
    .count();
```

Lambda 和 Stream 是 Java 现代化的标志。掌握它们之后，你将告别冗长的循环和 if 嵌套，写出更优雅、更易读的代码。
