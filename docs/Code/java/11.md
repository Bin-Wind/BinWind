# 文件 IO 操作

文件读写是几乎所有应用程序都需要的核心能力。Java 提供了 `java.io`（经典 IO）和 `java.nio.file`（NIO.2）两套 API，后者更加现代和高效。

## File 类（经典 IO）

`java.io.File` 是传统的文件路径抽象，功能有限但简单直观：

```java
import java.io.File;
import java.io.IOException;

File file = new File("example.txt");

// 基本信息
System.out.println(file.exists());         // 是否存在
System.out.println(file.getName());        // 文件名
System.out.println(file.getAbsolutePath());// 绝对路径
System.out.println(file.length());         // 字节大小
System.out.println(file.isDirectory());    // 是否为目录

// 创建和删除
file.createNewFile();          // 创建文件
file.mkdir();                  // 创建单级目录
file.mkdirs();                 // 创建多级目录
file.delete();                 // 删除文件或空目录

// 列出目录内容
File dir = new File(".");
File[] files = dir.listFiles((d, name) -> name.endsWith(".java"));
```

## Path 和 Files（NIO.2 — JDK 7+）

`java.nio.file.Path` 是不可变、更强大的路径抽象，配合 `Files` 工具类使用：

```java
import java.nio.file.*;
import java.io.IOException;

Path path = Paths.get("data", "config.properties");
// 等价于 Path path = Path.of("data/config.properties");  // JDK 11+

// 路径操作
System.out.println(path.getFileName());       // config.properties
System.out.println(path.getParent());         // data
Path absolute = path.toAbsolutePath();         // 转为绝对路径
Path normalized = absolute.normalize();        // 规范化（去除 . 和 ..）
Path resolved = path.resolve("sub/file.txt");  // 拼接路径

// 便捷的文件操作
Files.createFile(path);                    // 创建文件（已存在则抛异常）
Files.createDirectory(path.getParent());    // 创建目录
Files.createDirectories(path.getParent());  // 递归创建目录

Files.delete(path);                        // 删除（不存在抛异常）
Files.deleteIfExists(path);                // 安全删除

Files.copy(Path.from, Path.to, StandardCopyOption.REPLACE_EXISTING);
Files.move(Path.from, Path.to, StandardCopyOption.ATOMIC_MOVE);
```

## 读取与写入文件

```java
import java.nio.charset.StandardCharsets;
import java.nio.file.*;
import java.util.List;

Path file = Path.of("sample.txt");

// === 一次性读取 ===
String content = Files.readString(file);                     // JDK 11+
List<String> lines = Files.readAllLines(file, StandardCharsets.UTF_8);

// === 一次性写入 ===
Files.writeString(file, "Hello, World!");                   // JDK 11+
Files.write(file, List.of("line1", "line2"), StandardCharsets.UTF_8);

// === 流式读取（大文件） ===
try (var reader = Files.newBufferedReader(file)) {
    String line;
    while ((line = reader.readLine()) != null) {
        process(line);
    }
}

// === 流式写入（大文件） ===
try (var writer = Files.newBufferedWriter(file,
        StandardOpenOption.APPEND)) {   // 追加模式
    writer.write("appended line");
    writer.newLine();
}
```

## InputStream / OutputStream（字节流）

处理二进制文件（图片、音频、视频）时使用：

```java
import java.io.*;

Path src = Path.of("input.jpg");
Path dst = Path.of("output.jpg");

// 经典写法（带缓冲区）
try (InputStream in = new BufferedInputStream(Files.newInputStream(src));
     OutputStream out = new BufferedOutputStream(Files.newOutputStream(dst))) {
    byte[] buffer = new byte[8192];
    int bytesRead;
    while ((bytesRead = in.read(buffer)) != -1) {
        out.write(buffer, 0, bytesRead);
    }
}
// 更简洁的方式（JDK 9+）
Files.copy(src, dst, StandardCopyOption.REPLACE_EXISTING);
```

## Reader / Writer（字符流）

处理文本文件时使用，自动处理字符编码：

```java
import java.io.*;

Path textFile = Path.of("notes.txt");

// BufferedReader 逐行读取
try (BufferedReader reader = new BufferedReader(
        new FileReader(textFile.toFile()))) {
    String line;
    while ((line = reader.readLine()) != null) {
        System.out.println(line);
    }
}

// BufferedWriter 写入
try (BufferedWriter writer = new BufferedWriter(
        new FileWriter(textFile.toFile(), true))) {  // true = 追加
    writer.write("新的一行");
    writer.newLine();
}
```

推荐：大多数字符流场景直接用 `Files.readString()` / `Files.newBufferedReader()`，代码更简洁。

## Files.walk 遍历目录树

```java
import java.nio.file.*;
import java.util.stream.Stream;

Path root = Path.of("src/main/java");

// 遍历目录（默认深度优先）
try (Stream<Path> walk = Files.walk(root)) {
    walk.filter(Files::isRegularFile)                     // 只取普通文件
        .filter(p -> p.toString().endsWith(".java"))      // 只取 java 文件
        .forEach(System.out::println);
}

// 控制遍历深度
try (Stream<Path> walk = Files.walk(root, 2)) {  // 深度最多 2 层
    walk.forEach(System.out::println);
}

// 找最大文件
try (var stream = Files.walk(root)) {
    stream.filter(Files::isRegularFile)
          .max((a, b) -> {
              try { return Long.compare(Files.size(a), Files.size(b)); }
              catch (IOException e) { throw new UncheckedIOException(e); }
          })
          .ifPresent(p -> System.out.println("Max: " + p));
}
```

## IO 选型速查

| 场景                 | 推荐 API                    |
|----------------------|-----------------------------|
| 文本文件一次性读写   | `Files.readString / writeString` |
| 文本文件逐行处理     | `Files.newBufferedReader / Writer` |
| 二进制文件           | `InputStream / OutputStream` + Buffer |
| 文件路径操作         | `Path` + `Files`            |
| 目录遍历             | `Files.walk / list`         |
| 文件监控（变化通知） | `WatchService`              |

> 新项目优先用 `java.nio.file`（NIO.2），API 更现代，异常更友好，性能更优。
