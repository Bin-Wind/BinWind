# 运算符与表达式

运算符（Operator）是程序中执行计算和判定的基本单元。Java 提供了丰富的运算符集合，理解优先级和短路行为可以避免众多隐蔽的 Bug。

## 算术运算符（Arithmetic Operators）

```java
int a = 10, b = 3;
System.out.println(a + b);   // 13  加
System.out.println(a - b);   // 7   减
System.out.println(a * b);   // 30  乘
System.out.println(a / b);   // 3   整数除法（截断）
System.out.println(a % b);   // 1   取模（取余）

double c = 10.0 / 3;        // 3.3333333333333335（浮点除法）
// 注意除零：整数除零抛 ArithmeticException，浮点除零得 Infinity
```

自增/自减（`++` / `--`）分前缀和后缀：

```java
int x = 5;
int y = x++;   // y=5, x=6（先赋值后自增）
int z = ++x;   // z=7, x=7（先自增后赋值）
```

## 比较运算符（Comparison Operators）

返回 `boolean` 类型，常用于条件判断：

```java
int m = 10, n = 20;
System.out.println(m == n);  // false  等于
System.out.println(m != n);  // true   不等于
System.out.println(m > n);   // false  大于
System.out.println(m < n);   // true   小于
System.out.println(m >= n);  // false  大于等于
System.out.println(m <= n);  // true   小于等于

// 对象比较必须用 equals()
String s1 = new String("abc");
String s2 = new String("abc");
System.out.println(s1 == s2);      // false（比较引用）
System.out.println(s1.equals(s2)); // true （比较内容）
```

## 逻辑运算符（Logical Operators）

具有短路行为（short-circuit evaluation），从左到右，确定结果后不再计算：

```java
boolean t = true, f = false;
System.out.println(t && f);  // false  AND（短路）
System.out.println(t || f);  // true   OR（短路）
System.out.println(!t);      // false  NOT

// 短路示例：防止 NPE
String str = null;
if (str != null && str.length() > 0) {  // str.length() 不会执行
    System.out.println("safe");
}
```

非短路版本 `&` 和 `|` 总是计算两侧（也用于位运算）：

```java
if (a > 10 & b++ > 5) { }   // b++ 一定执行
if (a > 10 && b++ > 5) { }  // 若 a<=10，b++ 不执行
```

## 位运算符（Bitwise Operators）

操作二进制位，主要用于性能优化和底层编程：

```java
int p = 0b1100;  // 12
int q = 0b1010;  // 10

System.out.println(p & q);   // 8   (1000) 按位与
System.out.println(p | q);   // 14  (1110) 按位或
System.out.println(p ^ q);   // 6   (0110) 按位异或
System.out.println(~p);      // -13 按位取反

System.out.println(p << 1);  // 24  左移 1 位（乘以 2）
System.out.println(p >> 1);  // 6   有符号右移 1 位（除以 2）
System.out.println(p >>> 1); // 6   无符号右移（高位补 0）
```

## 三元运算符（Ternary Operator）

`condition ? valueIfTrue : valueIfFalse`

```java
int score = 85;
String grade = score >= 60 ? "Pass" : "Fail";

// 嵌套使用（注意可读性）
String rating = score >= 90 ? "A" : score >= 80 ? "B" : "C";
```

## 运算符优先级（Precedence）

从高到低的关键层级：

| 优先级 | 运算符                              |
|--------|-------------------------------------|
| 1      | `()` `[]` `.`                       |
| 2      | `!` `~` `++` `--` `+`(正) `-`(负)  |
| 3      | `*` `/` `%`                         |
| 4      | `+` `-`                             |
| 5      | `<<` `>>` `>>>`                     |
| 6      | `<` `<=` `>` `>=` `instanceof`      |
| 7      | `==` `!=`                           |
| 8      | `&`                                 |
| 9      | `^`                                 |
| 10     | `\|`                                |
| 11     | `&&`                                |
| 12     | `\|\|`                              |
| 13     | `?:` (三元)                         |
| 14     | `=` `+=` `-=` 等赋值               |

**口诀**：不确定优先级时，加括号最稳妥。括号总是能清晰表达意图。

```java
// 混乱 vs 清晰
int result1 = a + b * c >> 1;        // 难读
int result2 = a + ((b * c) >> 1);    // 清晰表达意图
```
