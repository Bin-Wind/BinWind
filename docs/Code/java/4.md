# 控制流程

控制流程决定了代码的执行路径。Java 提供 `if/else`、`switch`、循环和跳转语句，构成了程序逻辑的骨架。

## if / else 条件判断

```java
int score = 85;

if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}
// 输出: 良好
```

单行语句可省略花括号，但**强烈建议始终保留花括号**，避免悬空 `else` 等问题：

```java
// 不推荐
if (score > 60)
    System.out.println("pass");

// 推荐
if (score > 60) {
    System.out.println("pass");
}
```

## switch 语句

### 传统 switch（JDK 14 之前）

```java
String day = "MONDAY";

switch (day) {
    case "MONDAY":
        System.out.println("周一开工");
        break;    // 没有 break 会穿透到下一个 case
    case "FRIDAY":
        System.out.println("周五摸鱼");
        break;
    case "SATURDAY":
    case "SUNDAY":
        System.out.println("周末愉快");
        break;
    default:
        System.out.println("普通工作日");
}
```

**注意**：忘记 `break` 会造成 case 穿透（fall-through），这是经典 Bug 来源。

### 新式 switch 表达式（JDK 14+）

更简洁、安全、支持返回值：

```java
String day = "MONDAY";
String mood = switch (day) {
    case "MONDAY" -> "又是周一";
    case "FRIDAY" -> "马上周末";
    case "SATURDAY", "SUNDAY" -> "放假！";
    default -> "搬砖中";
};
System.out.println(mood);

// yield 在代码块中返回值
int numLetters = switch (day) {
    case "MONDAY", "FRIDAY", "SUNDAY" -> 6;
    case "TUESDAY" -> 7;
    default -> {
        System.out.println("未知天数");
        yield -1;
    }
};
System.out.println(numLetters); // 6
```

## for 循环

```java
// 经典 for
for (int i = 0; i < 5; i++) {
    System.out.print(i + " ");
}
// 输出: 0 1 2 3 4

// 增强 for（for-each）— 遍历数组和集合
int[] numbers = {10, 20, 30, 40};
for (int num : numbers) {
    System.out.print(num + " ");
}
// 输出: 10 20 30 40

// 多变量初始化和更新
for (int i = 0, j = 10; i < j; i++, j--) {
    System.out.println(i + " - " + j);
}
```

## while 和 do-while

```java
// while：先判断再执行
int count = 0;
while (count < 3) {
    System.out.println("while: " + count);
    count++;
}

// do-while：先执行一次再判断
int n = 5;
do {
    System.out.println("do-while: " + n);  // 至少执行一次
    n++;
} while (n < 3);
// 输出: do-while: 5
```

## break 与 continue

```java
// break：终止整个循环
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    System.out.print(i + " ");
}
// 输出: 0 1 2 3 4

// continue：跳过本次迭代
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    System.out.print(i + " ");
}
// 输出: 1 3 5 7 9
```

## 带标签的 break / continue（Labeled Break）

跳出多层嵌套循环：

```java
outer:
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (i * j > 4) {
            System.out.println("break outer at i=" + i + ", j=" + j);
            break outer;   // 跳出外层循环
        }
        System.out.println(i + " * " + j + " = " + (i * j));
    }
}
// 输出到 i=2, j=3 时跳出

// 标签也适用于 continue
```

带标签的跳转在复杂嵌套逻辑中很实用，远比用 `flag` 变量更清晰。
