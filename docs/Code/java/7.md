# 数组与集合框架

Java 集合框架（Collections Framework）是日常开发最高频使用的 API。从原始数组到功能丰富的集合类，选对数据结构能显著提升代码质量和性能。

## 数组（Array）

数组是固定长度的同类型元素序列，长度不可变：

```java
// 声明与初始化
int[] nums = new int[5];               // 默认值 0
String[] names = {"Alice", "Bob", "Charlie"};
int[][] matrix = {{1, 2}, {3, 4}};    // 二维数组

// 遍历
for (int i = 0; i < names.length; i++) {
    System.out.println(names[i]);
}
for (String name : names) {            // for-each
    System.out.println(name);
}

// 工具方法
int[] copy = Arrays.copyOf(nums, 10);         // 扩容
Arrays.sort(copy);                             // 排序
int idx = Arrays.binarySearch(copy, 3);       // 二分查找（需排序）
System.out.println(Arrays.toString(copy));     // 打印
```

## List 接口

有序、可重复的集合。最常用实现是 **ArrayList**（基于动态数组）和 **LinkedList**（基于双向链表）：

```java
import java.util.*;

// ArrayList — 随机访问快 O(1)，中间插入慢 O(n)
List<String> list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.add(0, "Go");         // 在索引 0 处插入
list.set(1, "Kotlin");     // 替换
list.remove("Python");     // 按元素删除
list.remove(0);            // 按索引删除
System.out.println(list.get(0));  // 按索引访问

// LinkedList — 头尾操作快 O(1)，适合做队列/栈
LinkedList<String> linked = new LinkedList<>();
linked.addFirst("first");
linked.addLast("last");
linked.pollFirst();       // 移除并返回头部

// 遍历方式
for (int i = 0; i < list.size(); i++) { }     // 传统
for (String s : list) { }                      // for-each
list.forEach(System.out::println);             // lambda
Iterator<String> it = list.iterator();         // 迭代器（遍历时安全删除）
while (it.hasNext()) {
    if (it.next().startsWith("J")) it.remove();
}
```

## Set 接口

不允许重复元素的集合：

```java
// HashSet — 基于 HashMap，无序，O(1) 增删查
Set<String> set = new HashSet<>();
set.add("A");
set.add("B");
set.add("A");              // 重复，不会加入
System.out.println(set.contains("A"));  // true

// LinkedHashSet — 维护插入顺序
Set<String> orderedSet = new LinkedHashSet<>();
orderedSet.add("C");
orderedSet.add("A");
orderedSet.add("B");
System.out.println(orderedSet);  // [C, A, B]

// TreeSet — 红黑树，元素有序（自然顺序或 Comparator）
Set<Integer> treeSet = new TreeSet<>();
treeSet.add(3);
treeSet.add(1);
treeSet.add(2);
System.out.println(treeSet);  // [1, 2, 3]
```

## Map 接口

键值对（key-value）结构，键不可重复：

```java
// HashMap — 最常用，无序，O(1) 查找
Map<String, Integer> map = new HashMap<>();
map.put("apple", 10);
map.put("banana", 5);
map.put("orange", 8);
System.out.println(map.get("apple"));              // 10
System.out.println(map.getOrDefault("pear", 0));    // 0（默认值）

map.putIfAbsent("grape", 3);   // 仅当不存在时写入
map.compute("apple", (k, v) -> v == null ? 0 : v + 1);  // 原子更新

// TreeMap — 按键排序（红黑树）
Map<String, Integer> sortedMap = new TreeMap<>();
sortedMap.putAll(map);
System.out.println(sortedMap);  // {apple=11, banana=5, grape=3, orange=8}

// 遍历 Map
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " -> " + entry.getValue());
}
map.forEach((k, v) -> System.out.println(k + ": " + v));
```

## Collections 工具类

提供排序、查找、同步包装等常用操作：

```java
List<Integer> numbers = new ArrayList<>(List.of(3, 1, 4, 1, 5, 9));

Collections.sort(numbers);                    // 升序
Collections.sort(numbers, Collections.reverseOrder());  // 降序
System.out.println(Collections.max(numbers)); // 9
System.out.println(Collections.min(numbers)); // 1
Collections.shuffle(numbers);                 // 乱序
Collections.fill(numbers, 0);                 // 填充

// 返回不可修改的集合
List<String> immutable = Collections.unmodifiableList(list);

// 线程安全包装
List<String> syncList = Collections.synchronizedList(new ArrayList<>());
```

## 选型建议

| 场景                       | 推荐数据结构            |
|----------------------------|------------------------|
| 快速随机访问               | `ArrayList`            |
| 频繁头尾插入/删除          | `LinkedList` / `ArrayDeque` |
| 去重、成员判断             | `HashSet`              |
| 需要排序                   | `TreeSet` / `TreeMap`  |
| 键值查找                   | `HashMap`              |
| 线程安全                   | `ConcurrentHashMap`    |

> 面对不确定的选型，优先用 `ArrayList` + `HashMap`，它们覆盖了 80% 的日常场景。
