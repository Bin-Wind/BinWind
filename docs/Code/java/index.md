# Java 编程 | 企业级开发的基石

**前言**

Java 是世界上最流行的编程语言之一，以「一次编写，到处运行」的跨平台能力和强大的企业级生态著称。欢迎来到「Java 编程」栏目。

### 栏目定位
- ☕ **Java 核心**：从语法基础到 JVM 原理
- 🏗️ **企业框架**：Spring Boot、MyBatis 全家桶
- 🔧 **工程实践**：Maven/Gradle、测试、CI/CD
- 📐 **设计模式**：GoF 模式在 Java 中的实践

### 内容方向
1. **基础入门** — 类型系统、面向对象、集合、IO
2. **进阶核心** — 并发、JVM、泛型、反射
3. **框架实战** — Spring Boot、MyBatis、安全
4. **工程能力** — 构建工具、测试、部署、调优
