# DOM 操作与事件

DOM（Document Object Model）是浏览器提供的编程接口，让 JavaScript 能够动态操作 HTML 内容和样式。

## 选择元素

```javascript
// 按 CSS 选择器（推荐）
const main = document.querySelector("#main");          // 单个元素
const items = document.querySelectorAll(".item");      // NodeList（静态）
const inputs = document.querySelectorAll("input[type='text']");

// 传统方法（特定场景更快）
const byId = document.getElementById("main");
const byClass = document.getElementsByClassName("item");   // HTMLCollection（动态）
const byTag = document.getElementsByTagName("div");
const byName = document.getElementsByName("email");
```

**注意**：`querySelectorAll` 返回**静态** NodeList，之后 DOM 变化不会更新。`getElementsByClassName` 返回**动态** HTMLCollection，会随 DOM 自动更新。

## 元素操作

```javascript
// 创建元素
const div = document.createElement("div");
div.className = "card";
div.id = "card-1";
div.setAttribute("data-id", "123");
div.dataset.status = "active"; // 对应 data-status="active"

// 内容设置
div.textContent = "纯文本内容";         // 安全，自动转义 HTML
div.innerHTML = "<strong>HTML 内容</strong>"; // 支持 HTML，注意 XSS 风险

// 插入 DOM
const container = document.querySelector("#container");
container.appendChild(div);               // 末尾追加
container.prepend(div);                   // 开头插入
container.insertBefore(div, container.firstChild); // 指定位置前

// 现代插入方法
div.before(newDiv);      // div 之前插入
div.after(newDiv);       // div 之后插入
div.replaceWith(newDiv); // 替换 div

// 移除元素
div.remove();                // 现代方法
// container.removeChild(div); // 旧方法

// 克隆
const clone = div.cloneNode(true);  // true = 深克隆（含子节点）
const shallow = div.cloneNode(false); // 仅当前节点
```

## 样式与类名操作

```javascript
const box = document.querySelector(".box");

// classList — 推荐
box.classList.add("active");         // 添加类
box.classList.remove("hidden");      // 移除类
box.classList.toggle("dark");        // 切換：有则移除，无则添加
box.classList.toggle("dark", true);  // 强制添加（第二个参数）
box.classList.contains("active");    // 是否包含
box.classList.replace("old", "new"); // 替换类名

// 行内样式
box.style.backgroundColor = "red";
box.style.transform = "translateX(100px)";
box.style.cssText = "color: white; padding: 10px;"; // 批量设置

// 读取计算后样式（含 CSS 文件中的）
const styles = getComputedStyle(box);
console.log(styles.width);      // "200px"
console.log(styles.paddingTop); // "10px"

// 尺寸与位置
box.offsetWidth;     // 内容+padding+border
box.clientWidth;     // 内容+padding（不含滚动条）
box.scrollWidth;     // 内容总宽（含溢出部分）
box.getBoundingClientRect(); // { top, left, bottom, right, width, height } — 相对视口
```

## 事件监听

```javascript
// addEventListener — 现代方式（推荐）
const button = document.querySelector("#btn");

function handleClick(event) {
  console.log("点击位置:", event.clientX, event.clientY);
  console.log("目标元素:", event.target);
  console.log("当前元素:", event.currentTarget);
}

button.addEventListener("click", handleClick);

// 选项参数
button.addEventListener("click", handler, {
  once: true,    // 只触发一次后自动移除
  capture: true, // 在捕获阶段触发（默认 false 为冒泡阶段）
  passive: true, // 不会调用 preventDefault()（提升滚动性能）
});

// 移除监听（必须传入同一个函数引用）
button.removeEventListener("click", handleClick);

// 常用事件类型
// 鼠标: click, dblclick, mousedown, mouseup, mousemove, mouseenter, mouseleave
// 键盘: keydown, keyup, keypress（已废弃）
// 表单: submit, change, input, focus, blur
// 文档: DOMContentLoaded, load, scroll, resize
// 触摸: touchstart, touchmove, touchend

// 键盘事件
document.addEventListener("keydown", (e) => {
  if (e.ctrlKey && e.key === "s") {
    e.preventDefault();
    console.log("Ctrl+S 被拦截");
  }
  console.log("按键:", e.key, "键码:", e.code);
});
```

## 事件委托（Event Delegation）

利用事件冒泡机制，在父元素上监听，通过 `event.target` 判断实际触发元素。避免了为每个子元素单独绑定事件。

```javascript
// ❌ 为每个 li 绑定事件 — 元素多时性能差，新增元素需重新绑定
document.querySelectorAll("li").forEach(li => {
  li.addEventListener("click", handleItemClick);
});

// ✅ 事件委托 — 只在父元素绑定一次
const list = document.querySelector("#todo-list");
list.addEventListener("click", (event) => {
  const target = event.target;

  // 通过 closest 找到最近的匹配元素
  const li = target.closest("li");
  if (!li) return; // 点击的不是 li，忽略

  const id = li.dataset.id;

  if (target.matches(".delete-btn")) {
    handleDelete(id);
  } else if (target.matches(".edit-btn")) {
    handleEdit(id);
  } else {
    handleSelect(id);
  }
});
```

## 事件传播：捕获与冒泡

DOM 事件传播分为三个阶段：**捕获阶段**（从 window 到目标元素）、**目标阶段**、**冒泡阶段**（从目标元素到 window）。

```javascript
// HTML 结构: <div id="outer"><div id="inner">点击</div></div>

outer.addEventListener("click", () => {
  console.log("outer — 冒泡阶段");
});

outer.addEventListener("click", () => {
  console.log("outer — 捕获阶段");
}, true); // 第三个参数 true = 捕获阶段

inner.addEventListener("click", (e) => {
  console.log("inner — 冒泡阶段");
  e.stopPropagation(); // 阻止事件继续传播（冒泡）
  // e.stopImmediatePropagation(); // 阻止传播 + 阻止同元素其他监听器
});

// 点击 inner 输出：
// "outer — 捕获阶段"
// "inner — 冒泡阶段"
// （outer 冒泡被 stopPropagation 阻止）

// preventDefault — 阻止默认行为（不阻止传播）
document.querySelector("a").addEventListener("click", (e) => {
  e.preventDefault(); // 阻止跳转
  // 事件仍会继续冒泡
});
```

## 表单操作

```javascript
const form = document.querySelector("#user-form");

form.addEventListener("submit", (e) => {
  e.preventDefault(); // 阻止默认提交

  // FormData — 快速收集表单数据
  const formData = new FormData(form);

  // 遍历
  for (const [key, value] of formData.entries()) {
    console.log(key, value);
  }

  // 转为普通对象
  const data = Object.fromEntries(formData.entries());
  console.log(data); // { username: "alice", email: "alice@mail.com" }

  // 直接发送
  fetch("/api/users", {
    method: "POST",
    body: formData, // 自动设置 Content-Type: multipart/form-data
  });
});
```

## 总结

- `querySelector` / `querySelectorAll` 是元素选择首选
- `classList` 操作类名比直接修改 `className` 更安全
- `textContent` 比 `innerHTML` 安全，避免 XSS
- 事件委托减少监听器数量，兼容动态添加的元素
- 理解捕获/冒泡机制，合理使用 `stopPropagation` 和 `preventDefault`
- `FormData` 是表单数据收集的最佳实践
