# 函数进阶：闭包、柯里化、高阶函数

函数是 JavaScript 的一等公民（First-Class Citizen）。这意味着函数可以赋值给变量、作为参数传递、作为返回值。这种特性催生了闭包、柯里化（Currying）、高阶函数（HOF）等强大模式。本文从实用场景切入，带你掌握这些进阶技巧。

## 闭包（Closure）— 函数记住它的词法环境

闭包不是一个需要刻意"创造"的东西——每当函数访问了外部作用域的变量，闭包就自然产生了。

```javascript
// 经典闭包：创建私有变量
function createCounter(initial = 0) {
  let count = initial; // 外部无法直接访问
  return {
    increment: () => ++count,
    decrement: () => --count,
    get value() { return count; }
  };
}

const counter = createCounter(10);
counter.increment(); // 11
counter.increment(); // 12
console.log(counter.value); // 12
```

### 闭包实战场景

```javascript
// 1. 防抖（debounce）— 用闭包保存 timer
function debounce(fn, delay) {
  let timer = null;
  return function (...args) {
    clearTimeout(timer);
    timer = setTimeout(() => fn.apply(this, args), delay);
  };
}

// 2. 单例模式
const getInstance = (() => {
  let instance = null;
  return (Constructor, ...args) => {
    if (!instance) instance = new Constructor(...args);
    return instance;
  };
})();
```

### 经典的 for+var 闭包陷阱

```javascript
// ❌ 错误：打印 5 个 5（所有回调共享同一个 i）
for (var i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 0);
}

// ✅ 解决一：用 let 创建块级作用域
for (let i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 0); // 0 1 2 3 4
}

// ✅ 解决二：IIFE 创建闭包副本
for (var i = 0; i < 5; i++) {
  ((j) => setTimeout(() => console.log(j), 0))(i);
}
```

## 柯里化（Currying）— 把多参函数变成单参函数链

```javascript
const curry = (fn) => {
  return function curried(...args) {
    if (args.length >= fn.length) return fn.apply(this, args);
    return (...next) => curried(...args, ...next);
  };
};

// 使用案例：日志工具
const log = (level, module, message) =>
  console.log(`[${level}][${module}] ${message}`);

const curriedLog = curry(log);
const errorLog = curriedLog('ERROR');
const dbError = errorLog('Database');

dbError('连接超时'); // [ERROR][Database] 连接超时

// URL 构建
const buildUrl = curry((protocol, domain, path) =>
  `${protocol}://${domain}/${path}`);

const apiUrl = buildUrl('https')('api.example.com');
apiUrl('users');    // https://api.example.com/users
apiUrl('products'); // https://api.example.com/products
```

## 组合（Compose / Pipe）— 把函数像管道一样串联

```javascript
// 从右向左执行 — compose
const compose = (...fns) => (x) =>
  fns.reduceRight((acc, fn) => fn(acc), x);

// 从左向右执行 — pipe
const pipe = (...fns) => (x) =>
  fns.reduce((acc, fn) => fn(acc), x);

// 文本处理管道
const trim = (s) => s.trim();
const lowercase = (s) => s.toLowerCase();
const removeSpaces = (s) => s.replace(/\s+/g, '-');

const slugify = pipe(trim, lowercase, removeSpaces);
slugify('  Hello World  JS  '); // "hello-world-js"
```

## 高阶函数（HOF）— 接收或返回函数的函数

```javascript
// before/after — 为函数添加前后处理
function withLogging(fn, name = fn.name) {
  return (...args) => {
    console.log(`[${name}] 调用, 参数:`, args);
    const start = performance.now();
    const result = fn(...args);
    console.log(`[${name}] 完成, 耗时: ${(performance.now() - start).toFixed(2)}ms`);
    return result;
  };
}
const patchedFetch = withLogging(fetchUser, 'fetchUser');
```

## 记忆化（Memoization）— 缓存函数计算结果

```javascript
function memoize(fn) {
  const cache = new Map();
  return (...args) => {
    const key = JSON.stringify(args);
    if (cache.has(key)) return cache.get(key);
    const result = fn(...args);
    cache.set(key, result);
    return result;
  };
}

// 斐波那契：不 memo 复杂度 O(2^n)，memo 后 O(n)
const fib = memoize((n) => {
  if (n < 2) return n;
  return fib(n - 1) + fib(n - 2);
});
fib(40); // 第一次计算
fib(40); // 瞬间返回，命中缓存

// LRU 缓存 — 限制缓存大小
function memoizeLRU(fn, limit = 100) {
  const cache = new Map();
  return (...args) => {
    const key = JSON.stringify(args);
    if (cache.has(key)) {
      const val = cache.get(key);
      cache.delete(key);
      cache.set(key, val);
      return val;
    }
    const result = fn(...args);
    cache.set(key, result);
    if (cache.size > limit) cache.delete(cache.keys().next().value);
    return result;
  };
}
```

闭包适合封装私有状态，柯里化让参数复用优雅，compose/pipe 是函数式编程的基石。用对场景，代码自然简洁有力。
