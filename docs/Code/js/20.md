# Proxy 与 Reflect

`Proxy` 是 ES2015 引入的元编程特性，它能在目标对象外层创建一个拦截层，定制对象基本操作的行为。`Reflect` 则提供了与 Proxy traps 一一对应的默认方法集合。二者配合，可以优雅地实现数据验证、响应式系统、虚拟属性等高级模式。

## Proxy 基础 — 创建代理

```javascript
const target = { name: 'Alice', age: 25 };
const handler = {
  get(obj, prop) {
    console.log(`读取 ${String(prop)}`);
    return obj[prop];
  },
  set(obj, prop, value) {
    console.log(`设置 ${String(prop)} = ${value}`);
    obj[prop] = value;
    return true; // 严格模式下必须返回 true
  }
};

const proxy = new Proxy(target, handler);
proxy.name;       // 输出 "读取 name"，返回 "Alice"
proxy.age = 26;   // 输出 "设置 age = 26"
```

## 核心 Traps 详解

Proxy 有 13 个 traps，以下是最核心的 6 个：

```javascript
const handler = {
  // 1. get — 读取属性
  get(target, prop, receiver) {
    if (prop in target) return Reflect.get(target, prop, receiver);
    return `属性 ${String(prop)} 不存在`;
  },

  // 2. set — 设置属性
  set(target, prop, value, receiver) {
    if (typeof value === 'string' && value.trim() === '') {
      throw new Error(`${String(prop)} 不能为空字符串`);
    }
    return Reflect.set(target, prop, value, receiver);
  },

  // 3. has — in 操作符
  has(target, prop) {
    console.log(`检查 ${String(prop)} 是否存在`);
    return Reflect.has(target, prop);
  },

  // 4. deleteProperty — delete 操作
  deleteProperty(target, prop) {
    if (prop.startsWith('_')) {
      throw new Error(`不能删除私有属性 ${String(prop)}`);
    }
    return Reflect.deleteProperty(target, prop);
  },

  // 5. ownKeys — Object.keys / for...in
  ownKeys(target) {
    return Reflect.ownKeys(target).filter(k => !String(k).startsWith('_'));
  },

  // 6. apply — 拦截函数调用（target 是函数时）
  apply(target, thisArg, args) {
    console.log(`调用函数，参数:`, args);
    return Reflect.apply(target, thisArg, args);
  }
};
```

## 实战：数据验证代理

```javascript
function createValidator(schema) {
  return new Proxy({}, {
    set(target, prop, value) {
      const rule = schema[prop];
      if (!rule) throw new Error(`${String(prop)} 不在 schema 中`);

      if (typeof value !== rule.type) {
        throw new TypeError(`${String(prop)} 应为 ${rule.type}`);
      }
      if (rule.min !== undefined && value < rule.min) {
        throw new RangeError(`${String(prop)} 不能小于 ${rule.min}`);
      }
      if (rule.max !== undefined && value > rule.max) {
        throw new RangeError(`${String(prop)} 不能大于 ${rule.max}`);
      }
      return Reflect.set(target, prop, value);
    }
  });
}

const user = createValidator({
  name: { type: 'string' },
  age: { type: 'number', min: 0, max: 150 }
});
user.name = 'Bob';   // OK
user.age = 200;      // RangeError: age 不能大于 150
```

## 实战：Observable（观察者模式）

Vue 3 的 `reactive()` 底层就用 Proxy 实现：

```javascript
function observable(target, onChange) {
  return new Proxy(target, {
    set(obj, prop, value, receiver) {
      const old = obj[prop];
      const result = Reflect.set(obj, prop, value, receiver);
      if (old !== value) onChange(prop, value, old);
      return result;
    },
    deleteProperty(obj, prop) {
      const had = prop in obj;
      const result = Reflect.deleteProperty(obj, prop);
      if (had) onChange(prop, undefined, obj[prop]);
      return result;
    }
  });
}

// 使用
const state = observable({ count: 0, text: '' }, (key, newVal, oldVal) => {
  console.log(`${String(key)}: ${oldVal} → ${newVal}`);
  updateUI();
});
state.count++; // count: 0 → 1
```

## 负索引数组（Python 风格）

```javascript
function createArray(...items) {
  return new Proxy(items, {
    get(target, prop) {
      const index = Number(prop);
      if (!isNaN(index) && index < 0) {
        return Reflect.get(target, target.length + index);
      }
      return Reflect.get(target, prop);
    }
  });
}
const arr = createArray('a', 'b', 'c', 'd');
console.log(arr[-1]); // 'd'
console.log(arr[-2]); // 'c'
```

## Reflect API — Proxy 的天然搭档

`Reflect` 提供和 Proxy traps 一一对应的静态方法：

```javascript
// 不用 Reflect（旧方式）
get(target, prop) { return target[prop]; }
// 用 Reflect（处理 getter 的 receiver 更正确）
get(target, prop, receiver) { return Reflect.get(target, prop, receiver); }

// Reflect 常用方法
Reflect.ownKeys(obj)       // → Object.getOwnPropertyNames + getOwnPropertySymbols
Reflect.has(obj, 'name')   // → 'name' in obj
Reflect.deleteProperty(obj, 'key')  // → delete obj.key
Reflect.construct(Target, args)     // → new Target(...args)
Reflect.apply(fn, thisArg, args)    // → fn.apply(thisArg, args)
```

## 可撤销代理

```javascript
// 标准 Proxy — 无法解除
const proxy = new Proxy(target, handler);

// 可撤销 Proxy
const { proxy, revoke } = Proxy.revocable(target, handler);
proxy.name; // OK
revoke();
proxy.name; // TypeError: 代理已被撤销
// 适用场景：临时授权访问，用完后 revoke
```

Proxy 是"少用但一用就漂亮"的特性。用对地方——数据校验、响应式、API 优雅化——它能让代码从"能跑"升级到"设计优雅"。
