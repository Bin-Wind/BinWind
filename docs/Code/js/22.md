# Web API 实战

浏览器提供了大量 Web API，覆盖从 DOM 观察、用户交互到设备功能的全方位能力。合理组合这些 API，能打造出原生 App 级别的 Web 体验。本文聚焦 6 个高频使用的 API 及其实战场景。

## Intersection Observer — 交叉观察器

取代 `scroll` 事件监听，性能更好。用于懒加载、无限滚动、曝光埋点。

```javascript
// 图片懒加载
const imageObserver = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const img = entry.target;
      img.src = img.dataset.src;
      img.classList.remove('lazy');
      imageObserver.unobserve(img);
    }
  });
}, { rootMargin: '200px' }); // 提前 200px 加载

document.querySelectorAll('img[data-src]').forEach(img => {
  imageObserver.observe(img);
});

// 无限滚动
const sentinel = document.getElementById('scroll-sentinel');
const scrollObserver = new IntersectionObserver(async (entries) => {
  if (entries[0].isIntersecting) await loadMoreItems();
});
scrollObserver.observe(sentinel);
```

## Mutation Observer — DOM 变化观察

```javascript
const domObserver = new MutationObserver((mutations) => {
  for (const mutation of mutations) {
    for (const node of mutation.addedNodes) {
      if (node.nodeType !== 1) continue;
      if (node.matches?.('.dynamic-form')) {
        initFormValidation(node);
      }
    }
  }
});

domObserver.observe(document.body, { childList: true, subtree: true });

// 水印防篡改
const watermarkObserver = new MutationObserver((mutations) => {
  for (const m of mutations) {
    if (m.removedNodes.length > 0 && Array.from(m.removedNodes).includes(watermarkEl)) {
      document.body.appendChild(watermarkEl);
    }
  }
});
watermarkObserver.observe(watermarkEl.parentElement, { childList: true });
```

## Resize Observer — 元素尺寸变化

比 `window.resize` 精确得多，能感知任意元素尺寸变化。

```javascript
const resizeObserver = new ResizeObserver((entries) => {
  for (const entry of entries) {
    const { width, height } = entry.contentRect;
    // 响应式容器：按宽度自适应布局
    if (width < 600) {
      entry.target.classList.add('compact');
    } else {
      entry.target.classList.remove('compact');
    }
  }
});

document.querySelectorAll('.responsive-container').forEach(el => {
  resizeObserver.observe(el);
});
```

## Geolocation API — 地理位置

```javascript
function getCurrentPosition() {
  return new Promise((resolve, reject) => {
    if (!navigator.geolocation) {
      reject(new Error('浏览器不支持 Geolocation'));
      return;
    }
    navigator.geolocation.getCurrentPosition(
      (pos) => resolve({
        lat: pos.coords.latitude,
        lng: pos.coords.longitude,
        accuracy: pos.coords.accuracy
      }),
      (err) => reject(err),
      { enableHighAccuracy: true, timeout: 10000, maximumAge: 60000 }
    );
  });
}

// 实时追踪位置变动（如外卖骑手追踪）
const watchId = navigator.geolocation.watchPosition(
  (pos) => updateMap(pos.coords),
  (err) => console.error(err),
  { enableHighAccuracy: true }
);
// navigator.geolocation.clearWatch(watchId);
```

## Clipboard API — 剪贴板操作

```javascript
// 写入剪贴板
async function copyText(text) {
  try {
    await navigator.clipboard.writeText(text);
    showToast('已复制');
  } catch (err) {
    console.error('复制失败:', err);
  }
}

// 读取剪贴板（需要用户授权或 HTTPS）
async function pasteText() {
  const text = await navigator.clipboard.readText();
  console.log('剪贴板内容:', text);
}
```

## Notification API — 桌面通知

```javascript
async function notify(title, options = {}) {
  if (!('Notification' in window)) return;
  const permission = await Notification.requestPermission();
  if (permission !== 'granted') return;

  const notification = new Notification(title, {
    body: options.body || '',
    icon: '/icon-192.png',
    tag: options.tag, // 相同 tag 会替换已有通知
    data: options.data
  });

  notification.onclick = () => {
    window.focus();
    notification.close();
    options.onClick?.(notification.data);
  };
}

notify('新消息', {
  body: '张三给你发了一条消息',
  tag: 'msg-123',
  data: { conversationId: 123 },
  onClick: (data) => navigateTo(`/chat/${data.conversationId}`)
});
```

## 综合案例：协作编辑器实时同步

结合上述 API 实现协作感知：

```javascript
class CollaborationView {
  constructor(container) {
    this.container = container;
    // 发现容器尺寸变化 → 重新布局
    this.ro = new ResizeObserver(() => this.reLayout());
    this.ro.observe(container);
    // 监测远端用户光标插入 → 渲染协作标记
    this.mo = new MutationObserver((ms) => this.renderRemoteCursors(ms));
    this.mo.observe(container, { childList: true, characterData: true, subtree: true });
    // 追踪用户是否在可视区 — 在时才推送协作
    this.io = new IntersectionObserver(([entry]) => {
      this.isVisible = entry.isIntersecting;
    });
    this.io.observe(container);
  }

  reLayout() { /* ... */ }
  renderRemoteCursors(mutations) { /* ... */ }

  destroy() {
    this.ro.disconnect();
    this.mo.disconnect();
    this.io.disconnect();
  }
}
```

浏览器 API 是一套沉睡的宝藏。每当你想用第三方库时，先看看浏览器是否已经原生支持——性能更好、体积为零、无需维护依赖。
