# 现代 JavaScript 趋势 2025

JavaScript 生态演进从未减慢。TC39 提案流程持续产出新特性，WebAssembly 走向成熟，TypeScript 几乎成为标配，边缘计算和 AI 集成带来新范式。本文梳理 2025 年最重要的趋势和已落地的特性。

## Temporal — 现代化的日期时间 API

Temporal（Stage 3）将取代 `Date`，解决其时区混乱、可变、API 反直觉等历史问题。

```javascript
// Temporal 已是 Stage 3，2025 年主流浏览器已实现
// 当前使用需 Polyfill: @js-temporal/polyfill

// 即时时间点
const now = Temporal.Now.instant();

// 带时区的日期时间
const meeting = Temporal.ZonedDateTime.from('2025-07-01T14:00[Asia/Shanghai]');
const tokyoTime = meeting.withTimeZone('Asia/Tokyo');
console.log(tokyoTime.toString()); // 自动转换时区

// 纯日期，不受时区干扰
const birthday = Temporal.PlainDate.from('1995-03-15');
const age = birthday.until(Temporal.Now.plainDateISO()).years;

// Duration 不丢失精度
const duration = Temporal.Duration.from({ hours: 2, minutes: 30 });
const total = duration.add({ minutes: 45 });
console.log(total.toString()); // "PT3H15M"

// 日期比较
const isWeekend = (date) => date.dayOfWeek >= 6;
```

## Decorators — Stage 3 原生装饰器

ES 原生装饰器不同于 TypeScript 的实验性装饰器：

```javascript
// 类成员装饰器（Stage 3 规范）
function logged(value, context) {
  if (context.kind === 'method') {
    return function (...args) {
      console.log(`调用 ${String(context.name)}，参数:`, args);
      const result = value.call(this, ...args);
      console.log(`返回:`, result);
      return result;
    };
  }
}

class UserService {
  @logged
  async fetchUser(id) {
    const res = await fetch(`/api/users/${id}`);
    return res.json();
  }
}

// @init: 初始化器装饰器
function uppercase(value, context) {
  if (context.kind === 'accessor') {
    return { init(v) { return v.toUpperCase(); } };
  }
}

class Config {
  @uppercase
  accessor env = 'development'; // 初始化为 'DEVELOPMENT'
}
```

## Records 与 Tuples（Stage 2）

不可变数据结构的原生支持：

```javascript
// Record — 不可变对象（#{...} 语法）
const user = #{ name: 'Alice', age: 30 };
// user.name = 'Bob'; // ❌ TypeError: 不可修改

// 结构相等比较（值比较，非引用比较）
console.log(#{ a: 1, b: 2 } === #{ a: 1, b: 2 }); // true

// Tuple — 不可变数组（#[...] 语法）
const point = #[10, 20];
const moved = #[point[0] + 5, point[1]]; // #[15, 20]

// 在 Map/Set 中作为 key
const cache = new Map();
cache.set(#[1, 2, 3], 'value');
```

## WebAssembly 互操作

Wasm GC 和 Component Model 让 Wasm 与 JS 对象直接交互。

```javascript
// WebAssembly.compileStreaming — 流式编译
const { instance } = await WebAssembly.instantiateStreaming(
  fetch('/module.wasm'),
  { imports: { /* ... */ } }
);
// 用 Rust/C++ 编写性能敏感模块，编译为 Wasm 后在前端调用
```

## TypeScript 主流化

```typescript
// 模板字面量类型
type EventName = `user:${'login' | 'logout' | 'update'}`;
// "user:login" | "user:logout" | "user:update"

// satisfies 操作符 — 检查类型但不扩宽
const config = {
  api: 'https://api.example.com',
  timeout: 5000
} satisfies Record<string, string | number>;
// config.api 类型仍是 string

// const 类型参数（TS 5.0+）
function createRouter<const T extends Record<string, string>>(routes: T) {
  return routes;
}
const routes = createRouter({ home: '/', about: '/about' });
// routes.home 类型是 '/' 而非 string
```

## 其他重要趋势

**边缘计算（Edge Computing）**：Cloudflare Workers、Deno Deploy、Vercel Edge Functions 让 JS 在 CDN 边缘节点运行，延迟进入毫秒级。

**信号式响应系统**：Solid.js、Preact Signals、Vue Vapor Mode 推广 signals 这种精确更新的响应式模型：

```javascript
// Solid.js 信号示例
import { createSignal, createEffect } from 'solid-js';
const [count, setCount] = createSignal(0);
createEffect(() => console.log(`Count: ${count()}`));
```

**Bun / Deno 2 崛起**：Node.js 不再是唯一选择。Bun 2.0 兼容 Node 生态同时运行速度更快，Deno 2 原生 TypeScript 和安全沙箱渐成优势。

**AI SDK 集成**：LLM 调用已成前端常态——Vercel AI SDK、LangChain.js 让 JS 开发者可直接部署 AI Agent。
