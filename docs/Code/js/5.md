# 流程控制

流程控制决定了代码的执行路径。JavaScript 提供了丰富的控制结构，从经典的 `if/else` 到现代的 `for...of` 循环。

## if / else 语句

```javascript
const score = 85;

if (score >= 90) {
  console.log("优秀");
} else if (score >= 80) {
  console.log("良好");
} else if (score >= 60) {
  console.log("及格");
} else {
  console.log("不及格");
}
```

**三元运算符（Ternary Operator）** 是 `if/else` 的表达式版本：

```javascript
const status = score >= 60 ? "pass" : "fail";
const level = score >= 90 ? "A" : score >= 80 ? "B" : score >= 60 ? "C" : "D";

// 嵌套三元时可读性下降，建议拆分为 if/else 或函数
```

**Guard Clauses（卫语句）** — 提前 return 减少嵌套：

```javascript
// ❌ 深层嵌套
function processUser(user) {
  if (user) {
    if (user.isActive) {
      if (user.hasPermission) {
        // 实际逻辑
      }
    }
  }
}

// ✅ 卫语句 — 提前退出
function processUser(user) {
  if (!user) return;
  if (!user.isActive) return;
  if (!user.hasPermission) return;
  // 实际逻辑 — 零缩进
}
```

## switch 语句

```javascript
const day = "Monday";

switch (day) {
  case "Monday":
    console.log("周一");
    break;                        // 必须有 break，否则会穿透
  case "Tuesday":
  case "Wednesday":               // 多个 case 共享逻辑
    console.log("工作日");
    break;
  case "Saturday":
  case "Sunday":
    console.log("周末");
    break;
  default:
    console.log("未知");
}
```

可以用对象映射替代复杂 `switch`：

```javascript
const handlers = {
  "GET": (req) => fetchData(req),
  "POST": (req) => createData(req),
  "DELETE": (req) => removeData(req),
};

const handler = handlers[method] || defaultHandler;
handler(request);
```

## for 循环

```javascript
// 经典 for — 最灵活
for (let i = 0; i < 5; i++) {
  console.log(i); // 0, 1, 2, 3, 4
}

// for...of — 遍历可迭代对象（数组、字符串、Map、Set）
const fruits = ["apple", "banana", "orange"];
for (const fruit of fruits) {
  console.log(fruit);
}

// for...of 配合 entries() 获取索引
for (const [index, value] of fruits.entries()) {
  console.log(`${index}: ${value}`);
}

// for...of 遍历字符串
for (const char of "hello") {
  console.log(char); // h, e, l, l, o
}
```

**注意**：`for...of` 不能遍历普通对象（Object 不是可迭代的），需要 `Object.keys()` / `Object.entries()` 转换：

```javascript
const user = { name: "Alice", age: 25 };

for (const key of Object.keys(user)) {
  console.log(key, user[key]);
}
for (const [key, value] of Object.entries(user)) {
  console.log(`${key}: ${value}`);
}
```

## for...in — 遍历可枚举属性

`for...in` 遍历对象自身及原型链上的**可枚举**属性，不推荐用于数组。

```javascript
const obj = { a: 1, b: 2, c: 3 };

for (const key in obj) {
  if (obj.hasOwnProperty(key)) { // 过滤原型链属性
    console.log(`${key}: ${obj[key]}`);
  }
}

// ❌ 不推荐 — 遍历数组可能包含原型扩展
Array.prototype.custom = "oops";
const arr = [10, 20, 30];
for (const i in arr) {
  console.log(i); // "0", "1", "2", "custom" — 意外！
}
```

## while / do...while

```javascript
// while — 先判断后执行
let count = 0;
while (count < 3) {
  console.log(count);
  count++;
}

// do...while — 先执行后判断（至少执行一次）
let attempts = 0;
do {
  attempts++;
  // 尝试操作...
} while (!success && attempts < maxRetries);
```

## break 与 continue

```javascript
// break — 完全退出循环
for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i); // 0, 1, 2, 3, 4
}

// continue — 跳过当前迭代
for (let i = 0; i < 10; i++) {
  if (i % 2 === 0) continue;
  console.log(i); // 1, 3, 5, 7, 9
}

// 标签（label）用于多层循环
outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) break outer; // 退出外层循环
    console.log(i, j);
  }
}
```

## 数组方法的"声明式"循环

很多时候可以用数组方法取代显式循环：

```javascript
const numbers = [1, 2, 3, 4, 5];

// 替代 for 循环的常见场景
numbers.forEach(n => console.log(n));            // 遍历
const doubled = numbers.map(n => n * 2);          // 转换
const evens = numbers.filter(n => n % 2 === 0);   // 过滤
const sum = numbers.reduce((acc, n) => acc + n, 0); // 聚合
const hasLarge = numbers.some(n => n > 3);        // 是否存在
const allPositive = numbers.every(n => n > 0);    // 全部满足
```

## 总结

- `if/else` 配合卫语句，减少缩进层级
- `switch` 用对象映射替换，更简洁灵活
- `for...of` 是遍历数组首选；`for...in` 用于对象（注意过滤原型属性）
- `forEach` / `map` / `filter` / `reduce` 是声明式循环，可读性更好
- 使用 `break` / `continue` 控制循环流程，label 处理多层循环
