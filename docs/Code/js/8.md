# 数组操作大全

数组是 JavaScript 中最常用的数据结构。掌握数组的高阶方法，能让代码从几十行缩减为一行。

## 遍历与转换

```javascript
const numbers = [1, 2, 3, 4, 5];

// forEach — 遍历，无返回值（无法 break）
numbers.forEach((item, index, arr) => {
  console.log(`${index}: ${item}`);
});

// map — 映射，返回新数组
const doubled = numbers.map(n => n * 2);       // [2, 4, 6, 8, 10]
const withIndex = numbers.map((n, i) => `${i}:${n}`); // ["0:1", "1:2", ...]

// filter — 过滤，返回满足条件的新数组
const evens = numbers.filter(n => n % 2 === 0); // [2, 4]
const truthy = [0, "", "hello", null, 5].filter(Boolean); // ["hello", 5]

// reduce — 归约，将数组聚合为单个值
const sum = numbers.reduce((acc, cur) => acc + cur, 0);       // 15
const max = numbers.reduce((acc, cur) => Math.max(acc, cur)); // 5

// reduce 实现分组
const people = [
  { name: "Alice", role: "admin" },
  { name: "Bob", role: "user" },
  { name: "Carol", role: "admin" },
];
const grouped = people.reduce((acc, person) => {
  (acc[person.role] = acc[person.role] || []).push(person);
  return acc;
}, {});
// { admin: [{name:"Alice",...}, {name:"Carol",...}], user: [{name:"Bob",...}] }
```

## 查找与检测

```javascript
const users = [
  { id: 1, name: "Alice" },
  { id: 2, name: "Bob" },
  { id: 3, name: "Carol" },
];

// find — 返回第一个匹配的元素
const bob = users.find(u => u.name === "Bob"); // { id: 2, name: "Bob" }

// findIndex — 返回第一个匹配元素的索引
const bobIndex = users.findIndex(u => u.name === "Bob"); // 1

// findLast / findLastIndex (ES2023)
const lastAdmin = users.findLast(u => u.role === "admin");

// includes — 是否包含某值
[1, 2, 3].includes(2);       // true
[1, 2, 3].includes(4);       // false

// indexOf / lastIndexOf
[1, 2, 3, 2].indexOf(2);     // 1 — 第一个出现位置
[1, 2, 3, 2].lastIndexOf(2); // 3 — 最后一个出现位置

// some — 是否存在满足条件的元素
numbers.some(n => n > 4);    // true

// every — 所有元素是否都满足条件
numbers.every(n => n > 0);   // true
numbers.every(n => n > 3);   // false
```

## 排序

```javascript
const fruits = ["banana", "apple", "orange", "grape"];

// sort — 原地排序（修改原数组），默认按字符串 Unicode 排序
fruits.sort(); // ["apple", "banana", "grape", "orange"]

// 数字排序需要提供比较函数
const nums = [10, 2, 30, 4];
nums.sort((a, b) => a - b);  // 升序 [2, 4, 10, 30]
nums.sort((a, b) => b - a);  // 降序 [30, 10, 4, 2]

// 对象按属性排序
users.sort((a, b) => a.name.localeCompare(b.name)); // 按名字字母排序
users.sort((a, b) => a.age - b.age);                // 按年龄排序

// toSorted (ES2023) — 非破坏性排序，返回新数组
const sorted = nums.toSorted((a, b) => a - b); // 新数组，nums 不变
```

## 截取与切片

```javascript
const arr = [1, 2, 3, 4, 5];

// slice(begin, end) — 浅拷贝部分数组，不修改原数组
arr.slice(1, 3);    // [2, 3] — 索引 1 到 3（不含 3）
arr.slice(2);       // [3, 4, 5] — 从索引 2 到末尾
arr.slice(-2);      // [4, 5] — 最后两个
arr.slice();        // [1, 2, 3, 4, 5] — 完整浅拷贝

// splice(start, deleteCount, ...items) — 增删改，修改原数组
const removed = arr.splice(2, 2, "x", "y"); // 从索引 2 删除 2 项，插入 "x", "y"
console.log(arr);     // [1, 2, "x", "y", 5]
console.log(removed); // [3, 4]

// 插入（不删除）
arr.splice(2, 0, "inserted"); // [1, 2, "inserted", "x", "y", 5]
```

## 添加与删除

```javascript
const stack = [];

// 末尾操作 — push / pop
stack.push("a", "b"); // 2 — 返回新长度，["a", "b"]
stack.pop();          // "b" — 返回被删除的元素，["a"]

// 头部操作 — unshift / shift
stack.unshift("c");   // 2 — ["c", "a"]
stack.shift();        // "c" — ["a"]

// 注意：unshift/shift 性能远低于 push/pop，因为需要移动所有元素
```

## 合并与扁平化

```javascript
// concat — 合并数组，返回新数组
const a = [1, 2];
const b = [3, 4];
a.concat(b);       // [1, 2, 3, 4] — a 不变
a.concat(b, [5, 6]); // [1, 2, 3, 4, 5, 6]

// flat — 扁平化嵌套数组
const nested = [1, [2, [3, [4]]]];
nested.flat();     // [1, 2, [3, [4]]] — 默认展开一层
nested.flat(2);    // [1, 2, 3, [4]] — 展开两层
nested.flat(Infinity); // [1, 2, 3, 4] — 完全展开

// flatMap — map + flat(1) 的组合
const sentences = ["Hello world", "How are you"];
const words = sentences.flatMap(s => s.split(" ")); // ["Hello", "world", "How", "are", "you"]
```

## 数组去重

```javascript
const dupes = [1, 2, 2, 3, 3, 3, 4];

// 方法 1：Set（最简单）
const unique = [...new Set(dupes)]; // [1, 2, 3, 4]

// 方法 2：filter + indexOf
const unique2 = dupes.filter((v, i, arr) => arr.indexOf(v) === i);

// 方法 3：reduce
const unique3 = dupes.reduce((acc, cur) =>
  acc.includes(cur) ? acc : [...acc, cur], []
);

// 对象数组按属性去重
const objects = [
  { id: 1, name: "A" },
  { id: 2, name: "B" },
  { id: 1, name: "A (dup)" },
];
const seen = new Map();
const uniqueObjects = objects.filter(item =>
  !seen.has(item.id) && seen.set(item.id, true)
);
```

## 数组与类数组转换

```javascript
// Array.from — 将类数组/可迭代对象转为数组
const nodeList = document.querySelectorAll("div");
const divs = Array.from(nodeList); // NodeList → Array

// 带映射的 Array.from
const squared = Array.from([1, 2, 3], x => x * x); // [1, 4, 9]

// 生成 range
const range = Array.from({ length: 5 }, (_, i) => i); // [0, 1, 2, 3, 4]

// Array.of — 创建数组（解决 new Array 的歧义）
Array.of(3);    // [3] — 而 new Array(3) 创建的是 [empty × 3]
Array.of(1, 2); // [1, 2]
```

## 总结

| 方法 | 用途 | 是否修改原数组 |
|------|------|:---:|
| `forEach` | 遍历 | 否 |
| `map` | 转换 | 否 |
| `filter` | 过滤 | 否 |
| `reduce` | 聚合 | 否 |
| `find` / `findIndex` | 查找 | 否 |
| `some` / `every` | 检测 | 否 |
| `sort` | 排序 | **是** |
| `splice` | 增删改 | **是** |
| `slice` | 截取 | 否 |
| `push` / `pop` / `shift` / `unshift` | 增删 | **是** |

优先使用非破坏性方法（`map`、`filter`、`slice`），避免意外修改原始数据。
