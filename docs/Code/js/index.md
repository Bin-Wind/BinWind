# JavaScript 开发 | Web 世界的通用语言

**前言**

JavaScript 是 Web 开发的核心语言，从浏览器到服务端无处不在。欢迎来到「JavaScript 开发」栏目，专注原生 JS 的深度探索。

### 栏目定位
- 📜 **JS 核心**：从 ES6+ 到最新 ECMAScript 标准
- 🌐 **浏览器 API**：DOM、BOM、事件、存储
- ⚡ **异步编程**：Promise、Async/Await、Event Loop
- 🧪 **现代工程**：模块化、Web APIs、测试

### 内容方向
1. **语言基础** — 类型、作用域、闭包、原型链
2. **异步核心** — Event Loop、Promise、异步模式
3. **浏览器实战** — DOM 操作、事件、网络请求
4. **现代特性** — ES6+、模块化、Web APIs
