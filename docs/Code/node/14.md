# WebSocket 实时通信 (Socket.io)

WebSocket 是一种在单个 TCP 连接上实现全双工通信的协议。相比 HTTP 轮询，它能大幅降低延迟和带宽消耗。Socket.io 是 Node.js 生态中最流行的 WebSocket 库，提供了自动重连、房间管理、广播等开箱即用的功能。

## WebSocket vs HTTP 轮询

| 方式 | 原理 | 延迟 | 带宽 | 适用场景 |
|------|------|------|------|----------|
| HTTP 短轮询 | 客户端定时发请求 | 高 | 浪费 | 低频更新 |
| HTTP 长轮询 | 客户端请求，服务端挂起直到有数据 | 中 | 一般 | 兼容性要求高 |
| WebSocket | 全双工持久连接 | 低 | 节省 | 实时聊天、游戏、协作 |

**何时选择 WebSocket：** 聊天应用、实时通知、在线协作编辑、股票行情、多人在线游戏。如果只是偶尔推送通知，Server-Sent Events (SSE) 可能是更轻量的选择。

## 服务端搭建

```javascript
// server.js
const http = require('http');
const { Server } = require('socket.io');

const httpServer = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/plain' });
  res.end('Socket.io Server Running');
});

const io = new Server(httpServer, {
  cors: {
    origin: 'http://localhost:5173', // 前端地址
    methods: ['GET', 'POST'],
  },
});

io.on('connection', (socket) => {
  console.log(`用户连接: ${socket.id}`);

  // 监听客户端消息
  socket.on('chat message', (msg) => {
    console.log(`收到消息: ${msg}`);
    // 广播给所有客户端（包括发送者）
    io.emit('chat message', msg);
  });

  // 断开连接
  socket.on('disconnect', () => {
    console.log(`用户断开: ${socket.id}`);
  });
});

httpServer.listen(3000, () => {
  console.log('服务运行在 http://localhost:3000');
});
```

## 客户端接入

```javascript
// client.js (浏览器端)
import { io } from 'socket.io-client';

const socket = io('http://localhost:3000', {
  autoConnect: true,
  reconnection: true,
  reconnectionAttempts: 5,
  reconnectionDelay: 1000,
});

socket.on('connect', () => {
  console.log('已连接到服务器');
});

// 发送消息
document.getElementById('form').addEventListener('submit', (e) => {
  e.preventDefault();
  const input = document.getElementById('input');
  socket.emit('chat message', input.value);
  input.value = '';
});

// 接收消息
socket.on('chat message', (msg) => {
  const li = document.createElement('li');
  li.textContent = msg;
  document.getElementById('messages').appendChild(li);
});
```

## 核心 API

### emit() 和 on()

```javascript
// 发送事件（可附带任意数量的参数）
socket.emit('eventName', data1, data2);

// 监听事件
socket.on('eventName', (data1, data2) => {
  console.log(data1, data2);
});

// 发送并等待确认（ACK）
socket.emit('question', '你叫什么？', (answer) => {
  console.log('服务端回答:', answer);
});

// 服务端响应 ACK
socket.on('question', (msg, callback) => {
  callback('我是服务器');
});
```

### 广播

```javascript
// 向除发送者外的所有客户端广播
socket.broadcast.emit('user joined', username);

// 向所有客户端广播（包括发送者）
io.emit('global notification', '服务器即将维护');

// 排除某个房间的广播
socket.broadcast.except('room1').emit('event');
```

### 房间 (Rooms)

房间是 Socket.io 的核心概念，允许多路复用一个连接。

```javascript
// 加入房间
socket.on('join room', (roomName) => {
  socket.join(roomName);
  console.log(`${socket.id} 加入了房间 ${roomName}`);
});

// 离开房间
socket.on('leave room', (roomName) => {
  socket.leave(roomName);
});

// 向房间内广播
io.to('room1').emit('room message', '房间1的消息');

// 向多个房间广播
io.to(['room1', 'room2']).emit('multi-room', '跨房间消息');

// 排除发送者
socket.to('room1').emit('msg', '其他人都能看到');
```

## 完整聊天应用示例

```javascript
// chat-server.js
const http = require('http');
const { Server } = require('socket.io');

const server = http.createServer();
const io = new Server(server, { cors: { origin: '*' } });

const users = new Map(); // socket.id → username

io.on('connection', (socket) => {
  socket.on('login', (username) => {
    users.set(socket.id, username);
    socket.broadcast.emit('notification', `${username} 加入了聊天室`);
    io.emit('online users', Array.from(users.values()));
  });

  socket.on('chat message', (msg) => {
    const username = users.get(socket.id) || '匿名';
    io.emit('chat message', { username, msg, time: new Date().toISOString() });
  });

  socket.on('disconnect', () => {
    const username = users.get(socket.id);
    users.delete(socket.id);
    if (username) {
      socket.broadcast.emit('notification', `${username} 离开了聊天室`);
      io.emit('online users', Array.from(users.values()));
    }
  });
});

server.listen(3000, () => console.log('聊天服务器启动在 :3000'));
```

## 错误处理与重连

```javascript
// 服务端错误处理
io.on('connection', (socket) => {
  socket.on('error', (err) => {
    console.error(`Socket ${socket.id} 错误:`, err.message);
  });
});

io.engine.on('connection_error', (err) => {
  console.error('连接错误:', err.message);
});

// 客户端重连配置
const socket = io('http://localhost:3000', {
  reconnection: true,
  reconnectionAttempts: Infinity,
  reconnectionDelay: 1000,
  reconnectionDelayMax: 5000,
  randomizationFactor: 0.5,
});

socket.on('connect_error', (err) => {
  console.error('连接失败:', err.message);
});

socket.on('reconnect_attempt', (attempt) => {
  console.log(`第 ${attempt} 次重连尝试`);
});

socket.on('reconnect', () => {
  console.log('重连成功');
});
```

## 总结

Socket.io 封装了 WebSocket 协议的复杂性，提供了房间、广播、自动重连等企业级特性。对于实时通信场景，优先考虑 WebSocket；对于简单的服务器推送场景，SSE 可能更合适。在生产环境中，建议配合 Redis Adapter 实现多进程下的消息广播。
