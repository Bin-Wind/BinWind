# 变量、数据类型与运算符

Python 是动态类型语言，变量无需声明类型即可直接赋值。掌握基本数据类型和运算符是编写任何程序的前提。

## 变量与赋值

```python
# 变量直接赋值，无需声明类型
name = "Alice"
age = 28
height = 1.68
is_student = False

# 多重赋值
x, y, z = 1, 2, 3
a = b = c = 0          # 链式赋值，三者均为 0

# 变量交换 - Python 特有写法
a, b = b, a

# 类型检查
print(type(name))      # <class 'str'>
print(type(age))       # <class 'int'>
```

## 数字类型 int / float / complex

```python
# int - 整数（无限制精度）
n = 42
big = 10 ** 100        # Python 支持任意大整数
binary = 0b1010        # 10
hex_num = 0xFF         # 255

# float - 浮点数（双精度）
pi = 3.14159
sci = 1.5e3            # 1500.0

# 数值运算
div = 7 / 2            # 3.5  （真除法）
floor_div = 7 // 2     # 3    （地板除）
mod = 7 % 2            # 1    （取模）
power = 2 ** 10        # 1024 （乘方）

# 运算优先级：** > 正负号 > * / // % > + -
```

## 字符串 str

```python
# 单引号、双引号均可
s1 = 'hello'
s2 = "world"

# 多行字符串
multi = """第一行
第二行
第三行"""

# 常用操作
greeting = "Hello, " + "World"        # 拼接
repeat = "Ha" * 3                      # "HaHaHa"
length = len("Python")                 # 6
char = "Python"[0]                     # 'P'
sub = "Python"[1:4]                    # 'yth' （切片）
print("Python" in "I love Python")    # True （成员检查）

# 转义字符
print("第一行\n第二行")
print(r"raw string \n 不会转义")      # 原始字符串
```

## 列表 list - 有序可变集合

```python
fruits = ["apple", "banana", "cherry"]

# 增删改查
fruits.append("orange")                # 末尾追加
fruits.insert(1, "mango")              # 指定位置插入
fruits.remove("banana")                # 按值删除
popped = fruits.pop()                  # 弹出末尾元素
popped_index = fruits.pop(1)           # 弹出指定位置
fruits[0] = "grape"                    # 修改元素

# 切片
first_two = fruits[:2]                 # 前两个
reversed_list = fruits[::-1]           # 反转

# 常用函数
nums = [3, 1, 4, 1, 5, 9]
sorted_nums = sorted(nums)             # 返回新列表
nums.sort()                            # 原地排序
print(len(nums), max(nums), min(nums), sum(nums))
```

## 元组 tuple - 有序不可变集合

```python
point = (3, 4)
single = (1,)           # 单元素元组必须有逗号
rgb = (255, 128, 0)

x, y = point            # 元组解包
print(point[0])         # 3

# 元组不可修改，以下会报错：
# point[0] = 10  # TypeError
```

## 字典 dict - 键值对映射

```python
user = {
    "name": "Bob",
    "age": 30,
    "email": "bob@example.com"
}

print(user["name"])                    # 访问
user["age"] = 31                       # 修改
user["city"] = "Shanghai"             # 新增
email = user.get("phone", "N/A")      # 安全访问，带默认值

# 遍历
for key, value in user.items():
    print(f"{key}: {value}")

# 删除
del user["email"]
removed = user.pop("city")
```

## 集合 set - 无序不重复

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)   # {1,2,3,4,5,6}   并集
print(a & b)   # {3,4}           交集
print(a - b)   # {1,2}           差集
print(a ^ b)   # {1,2,5,6}       对称差集

# 去重
nums = [1, 2, 2, 3, 3, 3]
unique = list(set(nums))           # [1, 2, 3]
```

## 类型转换

```python
int("42")            # 42
float("3.14")        # 3.14
str(100)             # "100"
list("abc")          # ['a', 'b', 'c']
tuple([1, 2, 3])     # (1, 2, 3)
dict([("a", 1)])     # {'a': 1}
bool(0)              # False
bool("hello")        # True
```

## 运算符

```python
# 比较运算符：== != > < >= <=
# 逻辑运算符：and or not
age = 25
has_license = True
if age >= 18 and has_license:
    print("可以驾驶")

# 身份运算符 is / is not（比较内存地址）
a = [1, 2]
b = [1, 2]
print(a == b)   # True  （值相等）
print(a is b)   # False （不是同一对象）
```

> 理解每种数据结构的特性——可变性、有序性、是否允许重复——是写出正确 Python 代码的基础。
