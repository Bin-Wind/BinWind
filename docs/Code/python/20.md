# Selenium 动态网页爬取

现代网站大量使用 JavaScript 动态渲染内容。传统的 `requests + BeautifulSoup` 只能获取初始 HTML，无法执行 JS。Selenium 通过控制真实浏览器来解决这个问题——它能加载 JS、点击按钮、填表、截图，就像真实用户一样操作网页。

## 环境准备

```bash
pip install selenium webdriver-manager
```

`webdriver-manager` 自动管理浏览器驱动版本，省去手动下载配置的麻烦。

## 第一个 Selenium 脚本

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.chrome.options import Options
from webdriver_manager.chrome import ChromeDriverManager

# 配置 Chrome 选项
options = Options()
options.add_argument("--headless")            # 无头模式（不显示浏览器窗口）
options.add_argument("--no-sandbox")
options.add_argument("--disable-dev-shm-usage")
options.add_argument("--window-size=1920,1080")

# 启动浏览器
service = Service(ChromeDriverManager().install())
driver = webdriver.Chrome(service=service, options=options)

try:
    driver.get("https://httpbin.org/html")
    print(driver.title)  # HTTPBIN - HTTP Requests & Responses
    print(driver.page_source[:200])  # 页面 HTML
finally:
    driver.quit()  # 确保关闭浏览器
```

## 等待策略：处理动态加载

这是 Selenium 实战中最重要的一课。页面元素可能还没加载出来，你的代码就去操作它了 —— 必然报错。

```python
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# 不好的做法：time.sleep(5)  — 浪费固定时间，网络快时等太久，慢时不够
# 正确做法：显式等待
wait = WebDriverWait(driver, timeout=10)

# 等待元素出现
element = wait.until(EC.presence_of_element_located((By.ID, "result")))

# 等待元素可见
element = wait.until(EC.visibility_of_element_located((By.CLASS_NAME, "loaded")))

# 等待元素可点击
btn = wait.until(EC.element_to_be_clickable((By.CSS_SELECTOR, "button.submit")))

# 等待元素包含特定文本
wait.until(EC.text_to_be_present_in_element((By.TAG_NAME, "h1"), "Welcome"))
```

常用 Expected Conditions 速查：

| Condition | 含义 |
|-----------|------|
| `presence_of_element_located` | 元素出现在 DOM 中（不一定可见） |
| `visibility_of_element_located` | 元素可见（有尺寸且非 hidden） |
| `element_to_be_clickable` | 元素可见且可点击 |
| `text_to_be_present_in_element` | 元素文本包含指定内容 |
| `invisibility_of_element` | 元素不可见（等待消失） |
| `staleness_of_element` | 元素从 DOM 中移除（页面刷新时有用） |

## 元素定位

```python
from selenium.webdriver.common.by import By

# 8 种定位策略
driver.find_element(By.ID, "username")
driver.find_element(By.NAME, "email")
driver.find_element(By.CLASS_NAME, "btn-primary")
driver.find_element(By.TAG_NAME, "h1")
driver.find_element(By.CSS_SELECTOR, "div.container > form input[name='q']")
driver.find_element(By.XPATH, "//button[text()='Submit']")
driver.find_element(By.LINK_TEXT, "Click here")
driver.find_element(By.PARTIAL_LINK_TEXT, "Click")

# 批量查找
elements = driver.find_elements(By.CSS_SELECTOR, "ul.items > li")
for el in elements:
    print(el.text)
```

CSS Selector 是推荐的首选策略：表达力强、易于理解和维护。

## 点击、填表与键盘操作

```python
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.support.ui import Select

# 输入文本
search_box = driver.find_element(By.NAME, "q")
search_box.clear()              # 先清空
search_box.send_keys("Python Selenium tutorial")
search_box.send_keys(Keys.RETURN)  # 按回车

# 点击按钮
btn = driver.find_element(By.CSS_SELECTOR, "button[type='submit']")
btn.click()

# 下拉选择框
select = Select(driver.find_element(By.NAME, "category"))
select.select_by_visible_text("Books")
select.select_by_value("bk")        # 按 value 属性
select.select_by_index(2)           # 按索引（从 0 开始）

# 复选框 & 单选框
checkbox = driver.find_element(By.CSS_SELECTOR, "input[type='checkbox']")
if not checkbox.is_selected():
    checkbox.click()

# JavaScript 执行（处理复杂交互）
driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")
driver.execute_script("arguments[0].click();", btn)
driver.execute_script("arguments[0].scrollIntoView(true);", element)
```

## 实战：爬取动态加载的图书列表

```python
import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from selenium.webdriver.chrome.options import Options
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager

options = Options()
options.add_argument("--headless")
options.add_argument("--window-size=1920,1080")
options.add_argument("--no-sandbox")

service = Service(ChromeDriverManager().install())
driver = webdriver.Chrome(service=service, options=options)
wait = WebDriverWait(driver, 10)

try:
    driver.get("https://books.toscrape.com/")
    # 等待书籍列表加载
    wait.until(EC.presence_of_element_located((By.CSS_SELECTOR, "article.product_pod")))

    books_data = []
    page = 1
    while True:
        print(f"Scraping page {page}...")
        articles = driver.find_elements(By.CSS_SELECTOR, "article.product_pod")

        for article in articles:
            title = article.find_element(By.CSS_SELECTOR, "h3 a").get_attribute("title")
            price = article.find_element(By.CSS_SELECTOR, "p.price_color").text
            books_data.append({"title": title, "price": price})

        # 尝试点击"下一页"
        try:
            next_btn = driver.find_element(By.CSS_SELECTOR, "li.next > a")
            next_btn.click()
            time.sleep(1)
            page += 1
        except Exception:
            break  # 没有下一页

    print(f"共爬取 {len(books_data)} 本书")
    for book in books_data[:5]:
        print(f"  {book['title']}: {book['price']}")

finally:
    driver.quit()
```

## Selenium 最佳实践

1. **优先用 API** — 目标网站提供 API 时，绝不用 Selenium。
2. **显式等待，不用 `time.sleep()`** — 更可靠、更快。
3. **使用无头模式** — 在服务器 / CI 环境运行必备。
4. **设置合理的视口大小** — 某些网站的响应式布局在小视口下元素不可见。
5. **善用 `execute_script`** — 当 Selenium API 不够用时，直接执行 JS。
6. **及时 `quit()`** — 用 `try...finally` 确保资源释放，避免浏览器进程残留。
7. **适度爬取** — 遵守 robots.txt，控制频率，不要成为 DDoS 攻击者。

Selenium 是处理 JS 动态渲染网页的王牌工具。对于简单的 JS 渲染场景，也可以考虑更轻量的 `Playwright` 或 `requests-html`，它们提供了类似的动态渲染能力并且往往性能更好。
