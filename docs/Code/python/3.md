# 条件判断与循环控制

程序的核心逻辑无非是"做判断"和"重复做"。Python 提供了直观的条件分支和循环结构，配合推导式让代码简洁有力。

## if / elif / else 条件分支

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"成绩等级：{grade}")   # B
```

条件表达式（三元运算符）：

```python
age = 20
status = "成年" if age >= 18 else "未成年"

# 逻辑运算符组合
x = 10
if 0 < x < 20:               # Python 支持链式比较
    print("x 在范围内")

# 真值测试 - 以下均视为 False：
# None, False, 0, 0.0, "", [], {}, set(), ()
items = []
if not items:
    print("列表为空")
```

## for 循环

```python
# 遍历列表
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# range() 生成序列
for i in range(5):           # 0, 1, 2, 3, 4
    print(i)

for i in range(2, 10, 2):    # 2, 4, 6, 8  (起始, 终止, 步长)
    print(i)

# 遍历字典
user = {"name": "Alice", "age": 28, "city": "Beijing"}
for key in user:
    print(key)                     # 遍历键

for value in user.values():
    print(value)                   # 遍历值

for key, value in user.items():
    print(f"{key} -> {value}")      # 同时遍历键和值
```

## enumerate - 索引 + 值同时遍历

```python
names = ["张三", "李四", "王五"]

for i, name in enumerate(names):
    print(f"第 {i} 位：{name}")

# 自定义起始索引
for i, name in enumerate(names, start=1):
    print(f"第 {i} 位：{name}")
```

## zip - 并行遍历多个序列

```python
names = ["张飞", "关羽", "刘备"]
ages = [35, 38, 42]
cities = ["涿郡", "河东", "涿郡"]

for name, age, city in zip(names, ages, cities):
    print(f"{name}，{age}岁，来自{city}")

# 不等长时以最短为准（可用 itertools.zip_longest 补全）
```

## break / continue / else

```python
# break：完全退出循环
for i in range(10):
    if i == 5:
        break
    print(i)          # 输出 0-4

# continue：跳过当前迭代
for i in range(5):
    if i == 2:
        continue
    print(i)          # 输出 0, 1, 3, 4

# for...else：循环正常完成时执行（未被 break 中断）
for n in [2, 3, 5, 7]:
    if n % 2 == 0:
        print(f"{n} 是偶数")
        break
else:
    print("没有偶数")      # 正常遍历完才会执行
```

## 推导式入门

推导式（Comprehension）是 Python 的精髓语法之一，将循环和条件浓缩为一行。

```python
# 列表推导式
squares = [x ** 2 for x in range(10) if x % 2 == 0]
# 结果：[0, 4, 16, 36, 64]

# 等价于传统写法：
squares = []
for x in range(10):
    if x % 2 == 0:
        squares.append(x ** 2)

# 字典推导式
word = "hello"
letter_count = {c: word.count(c) for c in set(word)}
# 结果：{'h': 1, 'e': 1, 'l': 2, 'o': 1}

# 集合推导式
unique_lengths = {len(word) for word in ["a", "bb", "ccc", "bb"]}
# 结果：{1, 2, 3}
```

## 实战：FizzBuzz 例子

```python
for i in range(1, 31):
    if i % 15 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
```

> Python 的循环哲学：`for` 遍历可迭代对象，`while` 处理未知次数的重复。善用 `enumerate`、`zip` 和推导式，可以少写很多索引代码。
