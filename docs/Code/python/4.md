# 函数定义与参数详解

函数是组织代码的基本单元。Python 的函数参数机制非常灵活，理解 `*args`、`**kwargs`、默认值、类型提示和 lambda 是写出优雅代码的关键。

## 基本函数定义

```python
def greet(name):
    """向指定用户打招呼。（这是 docstring）"""
    return f"你好，{name}！"

print(greet("小明"))                    # 你好，小明！

# 多返回值（本质是返回元组）
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder

q, r = divide(17, 5)                   # q=3, r=2
```

## 参数默认值

```python
def create_user(name, age=18, city="未知"):
    return {"name": name, "age": age, "city": city}

user1 = create_user("Alice", 25, "北京")
user2 = create_user("Bob")             # 使用默认值
user3 = create_user("Carol", city="上海")  # 关键字参数

# ⚠️ 默认值陷阱：可变对象作为默认值
# 错误写法：
def add_item_bad(item, items=[]):
    items.append(item)
    return items

print(add_item_bad(1))   # [1]
print(add_item_bad(2))   # [1, 2]  ← 意外！共享了同一个列表

# 正确写法：
def add_item_good(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```

## *args - 可变位置参数

```python
def sum_all(*args):
    """接受任意数量的位置参数"""
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3))              # 6
print(sum_all(1, 2, 3, 4, 5))       # 15

# 配合普通参数使用
def multiply(factor, *numbers):
    return [factor * n for n in numbers]

print(multiply(10, 1, 2, 3))         # [10, 20, 30]
```

## **kwargs - 可变关键字参数

```python
def build_profile(**kwargs):
    """接受任意数量的关键字参数"""
    return kwargs

profile = build_profile(name="Alice", age=28, role="Engineer")
print(profile)  # {'name': 'Alice', 'age': 28, 'role': 'Engineer'}

# 实际应用：通用 API 请求函数
def api_request(endpoint, **params):
    query_string = "&".join(f"{k}={v}" for k, v in params.items())
    return f"GET {endpoint}?{query_string}"

url = api_request("/users", page=2, limit=50, sort="name")
print(url)  # GET /users?page=2&limit=50&sort=name
```

## 参数顺序总结

完整参数序：`普通参数, *args, 关键字参数, **kwargs`

```python
def full_signature(name, *args, age=18, **kwargs):
    print(f"name={name}, args={args}, age={age}, kwargs={kwargs}")

full_signature("Tom", 1, 2, 3, age=25, city="NY", job="Dev")
# name=Tom, args=(1, 2, 3), age=25, kwargs={'city': 'NY', 'job': 'Dev'}
```

## 类型提示 Type Hints

```python
from typing import Optional, Union, Callable

def greet(name: str) -> str:
    return f"你好，{name}！"

def safe_divide(a: int, b: int) -> Optional[float]:
    """返回 Optional[X] 表示可能返回 None"""
    if b == 0:
        return None
    return a / b

# 联合类型
def process(data: Union[str, list[str]]) -> int:
    if isinstance(data, str):
        return len(data)
    return len(data)

# 函数作为参数
def apply(func: Callable[[int, int], int], x: int, y: int) -> int:
    return func(x, y)

result = apply(lambda a, b: a + b, 3, 5)   # 8
```

类型提示在运行时不做强制检查，但配合 mypy / pyright 等工具可以在编码阶段发现错误。

## lambda 匿名函数

```python
# 语法：lambda 参数: 表达式
square = lambda x: x ** 2
add = lambda a, b: a + b

print(square(5))            # 25
print(add(3, 7))            # 10

# 常用场景：作为排序 key
students = [
    {"name": "张三", "score": 85},
    {"name": "李四", "score": 92},
    {"name": "王五", "score": 78},
]
sorted_by_score = sorted(students, key=lambda s: s["score"])
# sorted_by_score  = sorted(students, key=lambda s: s["score"], reverse=True)
```

## 变量作用域 LEGB 规则

```python
x = "global"

def outer():
    x = "enclosing"

    def inner():
        x = "local"
        print(x)               # local

    inner()
    print(x)                   # enclosing

outer()
print(x)                       # global

# 修改外层变量需要声明
def counter():
    count = 0
    def increment():
        nonlocal count         # 修改 enclosing 域的变量
        count += 1
        return count
    return increment

c = counter()
print(c())   # 1
print(c())   # 2
```

> 函数是 Python 的一等公民——可以赋值给变量、作为参数传递、作为返回值。这是理解装饰器等高级特性的基础。
