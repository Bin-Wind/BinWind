# Vue 3 环境搭建与项目创建

要开始 Vue 3 开发，首先需要搭建本地开发环境。Vue 3 官方推荐使用 Vite 作为构建工具，通过 `create-vue` 脚手架快速初始化项目。

## Node.js 安装

Vue 3 项目依赖 Node.js 运行环境，建议使用 LTS（长期支持）版本。前往 [nodejs.org](https://nodejs.org) 下载安装，完成后在终端验证：

```bash
node -v   # v18.x 或更高
npm -v    # npm 随 Node 一起安装
```

推荐同时安装 `pnpm` 作为包管理器，速度更快、磁盘占用更少：

```bash
npm install -g pnpm
```

## 创建项目

使用官方脚手架 `create-vue` 创建新项目，它会引导你选择 TypeScript、JSX、Router、Pinia 等特性：

```bash
npm create vue@latest
```

交互式选项说明：

```text
✔ Project name: … my-vue-app         # 项目名称
✔ TypeScript? … Yes                   # 是否使用 TypeScript
✔ JSX Support? … No                   # 是否启用 JSX
✔ Vue Router? … Yes                   # 单页应用路由
✔ Pinia? … Yes                        # 状态管理
✔ Vitest? … Yes                       # 单元测试
✔ ESLint? … Yes                       # 代码规范
✔ Prettier? … Yes                     # 代码格式化
```

创建完成后进入项目并安装依赖：

```bash
cd my-vue-app
pnpm install
```

## Vite 开发服务器

Vite 提供了极快的冷启动和 HMR（模块热替换）。启动开发服务器：

```bash
pnpm dev
```

默认在 `http://localhost:5173` 启动。修改任何 `.vue` 文件，浏览器会即时反映变化，无需手动刷新。

### Vite 配置一览

`vite.config.ts` 是 Vite 的核心配置文件：

```typescript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      '@': '/src'   // 路径别名，用 @ 代替 /src
    }
  },
  server: {
    port: 3000,      // 自定义端口
    open: true       // 启动时自动打开浏览器
  }
})
```

## 项目目录结构

初始化后的标准目录如下：

```text
my-vue-app/
├── index.html              # Vite 入口 HTML
├── package.json            # 项目依赖与脚本
├── vite.config.ts          # Vite 配置
├── tsconfig.json           # TypeScript 配置
├── src/
│   ├── main.ts             # 应用入口，创建 Vue 实例
│   ├── App.vue             # 根组件
│   ├── assets/             # 静态资源（图片、样式）
│   ├── components/         # 可复用组件
│   ├── router/             # 路由配置
│   ├── stores/             # Pinia 状态管理
│   └── views/              # 页面级组件
└── public/                 # 不经过处理的静态文件
```

## 入口文件解析

`src/main.ts` 是应用的启动文件：

```typescript
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'
import router from './router'

const app = createApp(App)

app.use(createPinia())   // 注册 Pinia
app.use(router)           // 注册 Router

app.mount('#app')         // 挂载到 index.html 的 #app 元素
```

## 构建与预览

开发完成后，构建生产版本：

```bash
pnpm build          # 输出到 dist/ 目录
pnpm preview        # 在本地预览构建结果
```

构建产物是纯静态文件，可部署到任意静态服务器或 CDN。

## 推荐的 VS Code 插件

- **Vue - Official** (Volar)：语法高亮、类型检查、智能提示
- **TypeScript Vue Plugin**：TS 对 .vue 文件的支持
- **ESLint** + **Prettier**：代码规范与自动格式化

## 小结

至此，Vue 3 开发环境已准备就绪。关键命令速记：

| 命令 | 用途 |
|------|------|
| `npm create vue@latest` | 创建新项目 |
| `pnpm dev` | 启动开发服务器 |
| `pnpm build` | 生产构建 |
| `pnpm preview` | 预览构建结果 |
