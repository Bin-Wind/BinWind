# 过渡与动画

Vue 提供了 `<Transition>` 和 `<TransitionGroup>` 两个内置组件，用于在元素/组件进入和离开 DOM 时应用过渡动画。它们基于 CSS transitions/animations，同时也支持 JavaScript 钩子。

## Transition 组件

`<Transition>` 为**单个元素或组件**的进入和离开添加过渡效果：

```vue
<script setup>
import { ref } from 'vue'
const visible = ref(false)
</script>

<template>
  <button @click="visible = !visible">切换</button>
  <Transition name="fade">
    <p v-if="visible">Hello Vue</p>
  </Transition>
</template>

<style>
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.5s ease;
}
.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>
```

### 过渡的 6 个 CSS 类名

Vue 在过渡的不同阶段自动添加/移除以下类名：

```text
进入阶段                        离开阶段
------                         ------
name-enter-from (起始)          name-leave-from (起始)
name-enter-active (过渡中)      name-leave-active (过渡中)
name-enter-to (结束)            name-leave-to (结束)
```

时间轴示意：

```text
进入:  enter-from  →  [enter-active]  →  enter-to
       (插入前)        (插入后)           (完成后)

离开:  leave-from  →  [leave-active]  →  leave-to
       (移除前)        (移除中)           (移除后)
```

## 常用过渡示例

### Fade 淡入淡出

```vue
<style>
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s ease;
}
.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>
```

### Slide 滑动

```vue
<style>
.slide-enter-active,
.slide-leave-active {
  transition: all 0.3s ease-out;
}
.slide-enter-from {
  transform: translateX(20px);
  opacity: 0;
}
.slide-leave-to {
  transform: translateX(-20px);
  opacity: 0;
}
</style>
```

### Scale 缩放

```vue
<style>
.scale-enter-active,
.scale-leave-active {
  transition: all 0.3s ease;
}
.scale-enter-from,
.scale-leave-to {
  transform: scale(0.8);
  opacity: 0;
}
</style>
```

## CSS Animation 方式

使用 `@keyframes` 代替 transition 实现更复杂的动画：

```vue
<style>
.bounce-enter-active {
  animation: bounce-in 0.5s;
}
.bounce-leave-active {
  animation: bounce-in 0.5s reverse;
}

@keyframes bounce-in {
  0% {
    transform: scale(0);
    opacity: 0;
  }
  50% {
    transform: scale(1.25);
  }
  100% {
    transform: scale(1);
    opacity: 1;
  }
}
</style>
```

## JavaScript 钩子

当 CSS 过渡无法满足需求时，使用 JS 钩子直接操作 DOM（可配合 GSAP、Anime.js 等动画库）：

```vue
<template>
  <Transition
    @before-enter="onBeforeEnter"
    @enter="onEnter"
    @after-enter="onAfterEnter"
    @enter-cancelled="onEnterCancelled"
    @before-leave="onBeforeLeave"
    @leave="onLeave"
    @after-leave="onAfterLeave"
    @leave-cancelled="onLeaveCancelled"
  >
    <div v-if="visible">内容</div>
  </Transition>
</template>

<script setup>
function onBeforeEnter(el) {
  el.style.opacity = '0'
  el.style.transform = 'translateY(20px)'
}

function onEnter(el, done) {
  // 使用 Web Animations API
  el.animate(
    [
      { opacity: 0, transform: 'translateY(20px)' },
      { opacity: 1, transform: 'translateY(0)' }
    ],
    {
      duration: 300,
      easing: 'ease-out',
      fill: 'forwards'
    }
  ).finished.then(done)  // 动画完成后调用 done()
}

function onLeave(el, done) {
  el.animate(
    [
      { opacity: 1, transform: 'translateY(0)' },
      { opacity: 0, transform: 'translateY(-20px)' }
    ],
    {
      duration: 200,
      easing: 'ease-in',
      fill: 'forwards'
    }
  ).finished.then(done)
}
</script>
```

**注意**：纯 JS 动画必须在 `enter` 和 `leave` 钩子中调用 `done` 回调，否则过渡不会正常结束。如果只使用 CSS 过渡，**不要调用** `done`。

## mode — 切换模式

当一个元素离开同时另一个元素进入时，通过 `mode` 控制顺序：

```vue
<Transition name="fade" mode="out-in">
  <!-- out-in: 旧元素先离开，完成后新元素再进入（避免布局抖动） -->
  <component :is="currentTab" />
</Transition>
```

| 模式 | 效果 |
|------|------|
| 默认 | 进入和离开同时发生 |
| `out-in` | 旧元素先离开，新元素再进入 |
| `in-out` | 新元素先进入，旧元素再离开 |

## appear — 初次渲染也过渡

默认情况下 `<Transition>` 仅在 `v-if`/`v-show` 切换时触发过渡。添加 `appear` 让元素初次渲染时也执行过渡：

```vue
<Transition appear name="fade">
  <h1 v-if="show">标题</h1>
</Transition>
```

## TransitionGroup — 列表过渡

`<TransitionGroup>` 为**列表中的多个元素/组件**添加过渡，它渲染一个真实的 DOM 元素（默认 `<span>`，通过 `tag` 指定）：

```vue
<script setup>
import { ref } from 'vue'

const items = ref([
  { id: 1, text: 'Apple' },
  { id: 2, text: 'Banana' },
  { id: 3, text: 'Cherry' }
])

let nextId = 4

function add() {
  items.value.push({ id: nextId++, text: `Item ${nextId - 1}` })
}

function remove(id) {
  items.value = items.value.filter(i => i.id !== id)
}

function shuffle() {
  items.value = items.value.sort(() => Math.random() - 0.5)
}
</script>

<template>
  <button @click="add">添加</button>
  <button @click="shuffle">打乱</button>

  <TransitionGroup name="list" tag="ul">
    <li v-for="item in items" :key="item.id" @click="remove(item.id)">
      {{ item.text }}
    </li>
  </TransitionGroup>
</template>

<style>
/* 进入和离开过渡 */
.list-enter-active,
.list-leave-active {
  transition: all 0.5s ease;
}
.list-enter-from,
.list-leave-to {
  opacity: 0;
  transform: translateX(30px);
}

/* 移动过渡 — FLIP 动画的核心 */
.list-move {
  transition: transform 0.4s ease;
}

/* 确保离开中的元素占位，让其他元素平滑移动 */
.list-leave-active {
  position: absolute;
}
</style>
```

### FLIP 动画原理

`TransitionGroup` 自动应用 FLIP（First, Last, Invert, Play）技术：
1. 记录元素移动前位置 (First)
2. 记录元素移动后位置 (Last)
3. 计算差异并反转 (Invert)
4. 移除反转，触发过渡 (Play)

只需为 `.list-move` 设置 `transition: transform`，Vue 自动完成其余工作。

## 可复用过渡组件

将常用过渡封装为独立组件：

```vue
<!-- FadeTransition.vue -->
<template>
  <Transition name="fade" mode="out-in" appear>
    <slot />
  </Transition>
</template>

<style>
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s ease;
}
.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>
```

```vue
<!-- 使用 -->
<FadeTransition>
  <div v-if="visible">包装的内容</div>
</FadeTransition>
```

## 路由过渡动画

结合 Vue Router 实现页面切换动画：

```vue
<template>
  <RouterView v-slot="{ Component, route }">
    <Transition :name="route.meta.transition || 'fade'" mode="out-in">
      <component :is="Component" :key="route.path" />
    </Transition>
  </RouterView>
</template>
```

路由配置中指定过渡类型：

```javascript
const routes = [
  {
    path: '/login',
    component: Login,
    meta: { transition: 'slide' }
  },
  {
    path: '/dashboard',
    component: Dashboard,
    meta: { transition: 'fade' }
  }
]
```

## 性能提示

- 优先使用 `opacity` 和 `transform` 做动画，它们不触发重排（reflow），性能开销小
- 避免对 `width`、`height`、`top`、`left` 等触发重排的属性做动画
- `position: absolute` 让离开元素脱离文档流，避免列表动画时其他元素跳动
- 纯 CSS 动画比 JS 动画性能更好，能用 CSS 则不用 JS

## 小结

| 组件 | 适用场景 |
|------|----------|
| `<Transition>` | 单个元素/组件的进入/离开动画 |
| `<TransitionGroup>` | 列表元素的增删排序动画（FLIP） |
| CSS 类名 | `name-enter-from/to`、`name-leave-from/to`、`name-move` |
| JS 钩子 | 复杂动画、第三方动画库集成 |
| `mode="out-in"` | 路由切换避免双元素同时存在的布局问题 |
