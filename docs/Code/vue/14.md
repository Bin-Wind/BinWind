# TypeScript 与 Vue 3

Vue 3 对 TypeScript 的支持是语言级别的。得益于 Composition API，类型推导在 `<script setup>` 中几乎是自动的。本章覆盖 Props、Emits、Ref、Composable 和 defineExpose 的类型标注。

## 带类型的 defineProps 与 defineEmits

```vue
<script setup lang="ts">
interface Props {
  title: string
  count?: number
  tags: string[]
}
const props = withDefaults(defineProps<Props>(), {
  count: 0,
  tags: () => []
})

// 事件类型声明（Vue 3.3+ 简短写法）
const emit = defineEmits<{
  'update:modelValue': [value: string]
  submit: [payload: { name: string; age: number }]
}>()

emit('submit', { name: 'Tom', age: 25 })
</script>
```

`withDefaults` 为可选属性提供默认值。引用类型（数组、对象）必须使用工厂函数以避免共享引用。

## Ref 与 reactive 类型标注

```typescript
import { ref, reactive } from 'vue'

const count = ref(0)                    // Ref<number> 自动推导
const message = ref<string>('')         // 显式标注
type Status = 'idle' | 'loading' | 'error'
const status = ref<Status>('idle')      // 联合类型
const inputEl = ref<HTMLInputElement | null>(null)

// reactive 标注类型
const state: { x: number; y: number } = reactive({ x: 0, y: 0 })
```

## Composable 类型

Composable 是 Vue 3 最复用的单元。良好的类型标注让调用方获得完整的 IntelliSense：

```typescript
// composables/useCounter.ts
import { ref, computed, type Ref, type ComputedRef } from 'vue'

interface UseCounterOptions { min?: number; max?: number }

export function useCounter(initial = 0, options: UseCounterOptions = {}) {
  const { min = -Infinity, max = Infinity } = options
  const count = ref(initial)
  const double = computed(() => count.value * 2)
  function increment() { if (count.value < max) count.value++ }
  function decrement() { if (count.value > min) count.value-- }
  function reset() { count.value = initial }
  return { count, double, increment, decrement, reset }
}
// 返回类型自动推导，调用方获得完整类型提示
```

## defineExpose 获取子组件实例类型

```vue
<!-- Child.vue -->
<script setup lang="ts">
const data = ref('internal')
defineExpose({ data, focus: () => { /* ... */ } })
</script>
```

父组件：

```vue
<script setup lang="ts">
import Child from './Child.vue'
const childRef = ref<InstanceType<typeof Child> | null>(null)
childRef.value?.focus()
// Vue 3.5+ 使用 useTemplateRef<InstanceType<typeof Child>>('childRef')
</script>
```

## Options API 中的类型标注

```tsx
import { defineComponent, type PropType } from 'vue'

export default defineComponent({
  props: {
    options: {
      type: Array as PropType<{ label: string; value: string }[]>,
      required: true
    }
  }
})
```

## 总结

| 场景 | 推荐写法 |
|------|----------|
| Props | `defineProps<Props>()` + `withDefaults` |
| Emits | `defineEmits<{...}>()` 简短语法 |
| Ref | 依赖自动推导，必要时 `ref<T>()` |
| Composable | 函数返回类型自动推导即可 |
| 子组件实例 | `useTemplateRef` (3.5+) 或 `InstanceType<typeof X>` |
