# 自定义指令

Vue 3 自定义指令（Custom Directive）提供直接操作 DOM 的方式，适合处理原生事件、聚焦、懒加载等底层行为，与 Composition API 的声明式逻辑形成互补。

## 基本概念与钩子

指令通过 `v-` 前缀使用，在 `<script setup>` 中以 `v` 开头的 camelCase 变量注册：

| 钩子 | 触发时机 |
|------|----------|
| `mounted` | 元素挂载到 DOM 后 |
| `updated` | 组件更新后 |
| `unmounted` | 卸载后 |
| `beforeMount` / `beforeUpdate` / `beforeUnmount` | 对应阶段前 |

每个钩子接收 `(el, binding, vnode, prevVnode)` 四个参数。

## 焦点指令

```vue
<script setup lang="ts">
const vFocus = {
  mounted: (el: HTMLElement) => el.focus()
}
</script>
<template>
  <input v-focus />
</template>
```

## 点击外部指令（Click Outside）

```typescript
// directives/clickOutside.ts
import type { Directive } from 'vue'

export const vClickOutside: Directive = {
  mounted(el: HTMLElement, binding) {
    const handler = (e: MouseEvent) => {
      if (!el.contains(e.target as Node)) binding.value(e)
    }
    ;(el as any)._clickOutside = handler
    document.addEventListener('click', handler)
  },
  unmounted(el) {
    document.removeEventListener('click', (el as any)._clickOutside)
  }
}
```

使用：

```vue
<div v-click-outside="closeMenu" class="dropdown">...</div>
```

## 图片懒加载指令

```typescript
// directives/lazyLoad.ts
import type { Directive } from 'vue'

export const vLazyLoad: Directive<HTMLImageElement, string> = {
  mounted(el, binding) {
    el.style.backgroundColor = '#f0f0f0'
    const observer = new IntersectionObserver(
      (entries) => {
        if (entries[0].isIntersecting) {
          el.src = binding.value
          observer.unobserve(el)
        }
      },
      { rootMargin: '100px' }
    )
    observer.observe(el)
    ;(el as any)._observer = observer
  },
  unmounted(el) {
    ;(el as any)._observer?.disconnect()
  }
}
```

## Textarea 自适应高度

```typescript
export const vAutosize: Directive<HTMLTextAreaElement> = {
  mounted(el) {
    const resize = () => {
      el.style.height = 'auto'
      el.style.height = el.scrollHeight + 'px'
    }
    el.addEventListener('input', resize)
    ;(el as any)._handler = resize
    resize()
  },
  unmounted(el) {
    el.removeEventListener('input', (el as any)._handler)
  }
}
```

## 全局注册

```typescript
// main.ts
app.directive('lazy-load', {
  mounted(el, binding) { /* ... */ }
})

// 函数简写 —— 同时处理 mounted + updated
app.directive('highlight', (el, binding) => {
  el.style.color = binding.value
})
```

## 总结

自定义指令适合需要直接 DOM 访问的场景。如果逻辑可以用 `watch` 或 `ref` 实现，优先选用声明式方案。指令保留给焦点管理、事件监听、第三方库集成等场景。
