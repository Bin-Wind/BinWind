# 表单验证方案

Vue 3 生态最成熟的表单验证方案是 **VeeValidate + Zod**。本章从基本使用到提交处理，覆盖核心模式。

## 安装

```bash
npm install vee-validate zod @vee-validate/zod
```

## useForm + Zod Schema

```vue
<script setup lang="ts">
import { useForm } from 'vee-validate'
import { toTypedSchema } from '@vee-validate/zod'
import { z } from 'zod'

const schema = toTypedSchema(z.object({
  name: z.string().min(2, '姓名至少2字符'),
  email: z.string().email('格式不正确'),
  age: z.number().min(18).max(60),
  password: z.string().min(6, '至少6位'),
  confirm: z.string()
}).refine(data => data.password === data.confirm, {
  message: '两次密码不一致', path: ['confirm']
}))

const { handleSubmit, errors, values } = useForm({ validationSchema: schema })

const onSubmit = handleSubmit((data) => {
  console.log('提交:', data) // 调用 API
})
</script>

<template>
  <form @submit.prevent="onSubmit">
    <input v-model="values.name" placeholder="姓名" />
    <span class="error">{{ errors.name }}</span>

    <input v-model="values.email" placeholder="邮箱" />
    <span class="error">{{ errors.email }}</span>

    <input v-model.number="values.age" type="number" />
    <input v-model="values.password" type="password" />
    <input v-model="values.confirm" type="password" />
    <span class="error">{{ errors.confirm }}</span>

    <button type="submit">提交</button>
  </form>
</template>
```

## useField —— 单字段控制

```typescript
const { value: username, errorMessage, meta } = useField<string>(
  'username',
  (val) => {
    if (!val) return '不能为空'
    if (val.length < 3) return '至少3字符'
    return true
  }
)
// meta: dirty, touched, valid, pending
```

## 自定义规则

```typescript
import { defineRule } from 'vee-validate'

defineRule('phone', (v: string) =>
  /^1[3-9]\d{9}$/.test(v) || '手机号不正确'
)

defineRule('unique', async (v: string) => {
  const res = await fetch(`/api/check-username?name=${v}`)
  return (await res.json()).ok || '用户名已被占用'
})
```

## Yup 集成

```bash
npm install yup @vee-validate/yup
```

```typescript
import { toTypedSchema } from '@vee-validate/yup'
import * as yup from 'yup'

const schema = toTypedSchema(yup.object({
  title: yup.string().required('必填').min(5, '至少5字'),
  tags: yup.array().of(yup.string()).min(1, '至少选一个标签')
}))
```

Zod（~12KB）vs Yup（~8KB）：Zod 原生 TS 推导，Yup 需额外 `InferType`。

## 提交处理

```vue
<script setup lang="ts">
const serverError = ref('')

const onSubmit = handleSubmit(async (data) => {
  serverError.value = ''
  try {
    await fetch('/api/posts', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data)
    })
    resetForm() // 成功后重置
  } catch (e: any) {
    serverError.value = e.message || '提交失败'
  }
})
</script>

<template>
  <form @submit.prevent="onSubmit">
    <!-- 字段... -->
    <p v-if="serverError">{{ serverError }}</p>
    <button :disabled="isSubmitting">
      {{ isSubmitting ? '提交中...' : '提交' }}
    </button>
  </form>
</template>
```

## 总结

- `useForm` 管理整表单，`useField` 控制单字段
- 自定义规则用 `defineRule`，支持同步/异步
- 提交处理需管理 loading、server error、成功重置
