# 国际化（i18n）

Vue I18n v9 拥抱 Composition API，支持延迟加载语言包、消息插值、复数规则。本章构建完整多语言系统。

## 安装与配置

```bash
npm install vue-i18n@9
```

```typescript
// i18n/index.ts
import { createI18n } from 'vue-i18n'

const messages = {
  en: {
    welcome: 'Welcome to Vue',
    user: { profile: 'Profile', settings: 'Settings' },
    button: { save: 'Save', cancel: 'Cancel' }
  },
  zh: {
    welcome: '欢迎使用 Vue',
    user: { profile: '用户信息', settings: '设置' },
    button: { save: '保存', cancel: '取消' }
  }
}

const i18n = createI18n({
  legacy: false,         // Composition API 模式
  locale: 'zh',
  fallbackLocale: 'en',
  messages
})

export default i18n
```

`main.ts` 中 `app.use(i18n)` 安装。

## 用法与插值

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'
const { t, locale } = useI18n()
</script>

<template>
  <p>{{ t('welcome') }}</p>
  <p>{{ t('user.profile') }}</p>

  <!-- 命名参数插值 -->
  <p>{{ t('greeting', { name: 'Tom' }) }}</p>

  <!-- 复数规则：en: "no items | one item | {count} items" -->
  <p>{{ t('itemCount', 5) }}</p>

  <button @click="locale = locale === 'zh' ? 'en' : 'zh'">切换</button>
</template>
```

## 延迟加载语言包

避免把所有语言打包进入口：

```typescript
// i18n/index.ts
const i18n = createI18n({
  legacy: false, locale: 'zh', fallbackLocale: 'en', messages: {}
})
const loadedLanguages = new Set<string>()

export async function loadLocaleMessages(lang: string) {
  if (loadedLanguages.has(lang)) return
  const messages = await import(`@/locales/${lang}.json`)
  i18n.global.setLocaleMessage(lang, messages.default)
  loadedLanguages.add(lang)
}

export async function switchLanguage(lang: string) {
  await loadLocaleMessages(lang)
  i18n.global.locale.value = lang
  document.querySelector('html')?.setAttribute('lang', lang)
  localStorage.setItem('lang', lang)
}

export default i18n
```

Composable 封装：

```typescript
// composables/useLang.ts
import { useI18n } from 'vue-i18n'
import { switchLanguage } from '@/i18n'

export function useLang() {
  const { locale } = useI18n()
  return { locale, setLang: switchLanguage }
}
```

## 文件组织

```
locales/
├── en/
│   ├── common.json    # 通用文本
│   ├── user.json      # 用户模块
│   └── order.json     # 订单模块
└── zh/
    ├── common.json
    ├── user.json
    └── order.json
```

```typescript
// i18n.d.ts
import 'vue-i18n'
declare module 'vue-i18n' {
  interface DefineLocaleMessage {
    welcome: string
    user: { profile: string; settings: string }
    button: { save: string; cancel: string }
  }
}
```

## 最佳实践

1. **`legacy: false`** —— 启用 Composition API 模式
2. **延迟加载** —— 非默认语言按需加载
3. **按模块拆分** —— 避免单文件数千行
4. **fallbackLocale** —— 防止缺失 key 显示原始 key
5. **参数插值代替拼接** —— 尊重语序差异
6. **Pinia 同步 locale** —— 方便非组件场景使用
