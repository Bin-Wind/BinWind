# 模板语法与响应式基础

Vue 使用基于 HTML 的模板语法，将组件实例的数据绑定到 DOM。配合响应式系统，当数据变化时视图自动更新，无需手动操作 DOM。

## 文本插值

使用双大括号 `{{ }}` 将数据渲染为文本：

```vue
<template>
  <p>{{ message }}</p>
  <p>{{ count + 1 }}</p>
  <p>{{ isAdmin ? '管理员' : '普通用户' }}</p>
</template>

<script setup>
import { ref } from 'vue'

const message = ref('Hello Vue 3')
const count = ref(10)
const isAdmin = ref(true)
</script>
```

## 属性绑定 (v-bind)

使用 `v-bind` 将表达式绑定到 HTML 属性上，简写为 `:`：

```vue
<template>
  <img :src="imageUrl" :alt="title" />
  <div :class="['box', isActive && 'active']"></div>
  <button :disabled="isLoading">提交</button>
  <div :style="{ color: textColor, fontSize: size + 'px' }"></div>
</template>

<script setup>
import { ref } from 'vue'
const imageUrl = ref('/logo.png')
const title = ref('Logo')
const isActive = ref(true)
const isLoading = ref(false)
const textColor = ref('#333')
const size = ref(16)
</script>
```

## 响应式基础：ref 与 reactive

Vue 3 提供了两种声明响应式状态的方式。

### ref — 基本类型的响应式包装

`ref` 将值包装为响应式对象，在 `<script>` 中通过 `.value` 访问：

```vue
<script setup>
import { ref } from 'vue'

const name = ref('Alice')       // 字符串
const age = ref(25)             // 数字
const items = ref([])           // 数组

function updateName() {
  name.value = 'Bob'            // script 中需要 .value
}
</script>

<template>
  <p>{{ name }}</p>             <!-- template 中自动解包，无需 .value -->
</template>
```

### reactive — 对象的响应式代理

`reactive` 直接代理整个对象，访问属性不需要 `.value`：

```vue
<script setup>
import { reactive } from 'vue'

const state = reactive({
  user: { name: 'Alice', age: 25 },
  todos: [{ id: 1, text: '学习 Vue', done: false }]
})

function addTodo(text) {
  state.todos.push({ id: Date.now(), text, done: false })
}
</script>
```

### ref vs reactive 选择建议

| 场景 | 推荐 |
|------|------|
| 基本类型 (string, number, boolean) | `ref` |
| 简单对象 | `ref` 或 `reactive` 均可 |
| 表单对象、复杂嵌套结构 | `reactive` |
| 需要整体替换的数组 | `ref` |

## 双向绑定 (v-model)

`v-model` 是 `v-bind` + `v-on:input` 的语法糖，实现表单元素的双向绑定：

```vue
<template>
  <input v-model="username" placeholder="请输入用户名" />
  <p>当前输入：{{ username }}</p>
</template>

<script setup>
import { ref } from 'vue'
const username = ref('')
</script>
```

## 计算属性 (computed)

`computed` 根据已有响应式数据派生新值，具有缓存机制，依赖不变时不会重新计算：

```vue
<script setup>
import { ref, computed } from 'vue'

const items = ref([1, 2, 3, 4, 5])

const total = computed(() => {
  console.log('计算 total...')
  return items.value.reduce((sum, n) => sum + n, 0)
})

const doubled = computed(() => items.value.map(n => n * 2))
</script>
```

## 侦听器 (watch)

`watch` 用于在响应式数据变化时执行副作用操作（API 请求、DOM 操作等）：

```vue
<script setup>
import { ref, watch } from 'vue'

const keyword = ref('')
const page = ref(1)

// 侦听单个 ref
watch(keyword, (newVal, oldVal) => {
  console.log(`搜索关键词从 "${oldVal}" 变为 "${newVal}"`)
})

// 侦听多个源
watch([keyword, page], ([newKw, newPage]) => {
  fetchResults(newKw, newPage)
})

// immediate：立即执行一次
watch(keyword, fetchResults, { immediate: true })
</script>
```

## computed vs watch 对比

| 特性 | computed | watch |
|------|----------|------|
| 用途 | 派生计算新值 | 执行副作用 |
| 缓存 | 有，依赖不变不重新计算 | 无 |
| 返回值 | 返回计算后的值 | 无返回值 |
| 异步 | 不支持 | 支持 |

**口诀**：需要"算出一个新值"用 `computed`，需要"当值变化时做某事"用 `watch`。

## 小结

- `{{ }}` 文本插值，`v-bind:` (或 `:`) 绑定属性
- `ref` 处理基本类型，`reactive` 处理对象
- `v-model` 简化表单双向绑定
- `computed` 用于派生计算，带缓存
- `watch` 用于监听变化并执行副作用
