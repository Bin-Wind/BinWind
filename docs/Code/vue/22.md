# Nuxt 3 入门

Nuxt 3 是基于 Vue 3 的全栈框架，提供文件路由、自动导入、SSR/SSG 等开箱即用能力。

## 创建项目

```bash
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app && npm run dev
```

## 文件路由

`pages/` 目录自动生成路由：

```
pages/
├── index.vue           →  /
├── about.vue           →  /about
├── users/
│   └── [id].vue        →  /users/:id
└── blog/
    └── [...slug].vue   →  /blog/**
```

```vue
<!-- pages/users/[id].vue -->
<script setup lang="ts">
const route = useRoute()
const id = route.params.id
</script>
<template>
  <p>用户 ID: {{ id }}</p>
  <NuxtLink to="/">首页</NuxtLink>
</template>
```

## 自动导入

Vue API、Composables 和 `components/` 全部自动导入：

```vue
<script setup lang="ts">
// ref, computed, useFetch 等无需 import
const count = ref(0)
const double = computed(() => count.value * 2)
</script>
<template>
  <AppHeader />   <!-- components/ 自动可用 -->
</template>
```

## 数据获取

```vue
<script setup lang="ts">
// useFetch —— 最简，自动集成 SSR
const { data, pending, error, refresh } = await useFetch('/api/posts')

// useAsyncData —— 自定义异步
const { data: users } = await useAsyncData('users', () => $fetch('/api/users'))

// 参数变化自动重请求
const page = ref(1)
const { data: list } = await useFetch('/api/articles', {
  query: { page },
  watch: [page]
})
</script>
```

## Server API

`server/` 目录创建 API 端点，无需单独后端：

```typescript
// server/api/posts.ts
export default defineEventHandler(async (event) => {
  const query = getQuery(event)
  return { code: 0, data: await fetchPosts(query.page) }
})

// server/api/users/[id].ts
export default defineEventHandler((event) => {
  return { id: getRouterParam(event, 'id') }
})

// POST + 校验
export default defineEventHandler(async (event) => {
  const body = await readBody(event)
  // validate body...
  return { token: 'jwt-token' }
})
```

## 布局

```vue
<!-- layouts/default.vue -->
<template>
  <AppHeader />
  <slot />
  <AppFooter />
</template>
```

页面中选择：`definePageMeta({ layout: 'admin' })`

## 渲染模式

```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  ssr: true,
  routeRules: {
    '/admin/**': { ssr: false },         // SPA
    '/blog/**': { static: true },        // SSG
    '/api/**': { cache: { maxAge: 60 } } // ISR
  }
})
```

## 路由中间件

```typescript
// middleware/auth.ts
export default defineNuxtRouteMiddleware((to) => {
  const token = useCookie('token')
  if (!token.value && to.path !== '/login') return navigateTo('/login')
})
```

```vue
<script setup lang="ts">
definePageMeta({ middleware: ['auth'] })
</script>
```

## 项目结构

```
my-nuxt-app/
├── pages/          # 文件路由
├── components/     # 自动导入组件
├── composables/    # 自动导入 Composable
├── server/api/     # Server API
├── layouts/        # 布局
├── middleware/     # 路由中间件
└── nuxt.config.ts
```

## 总结

Nuxt 3 核心：约定优于配置。文件路由、自动导入、内置数据获取和 Server API 让开发者以极少样板启动全栈项目。适合 SEO 敏感或快速启动的项目。
