# 性能优化实战

Vue 3 底层做了编译器静态提升和 PatchFlag 靶向更新，但面对大列表和深层组件树，仍需主动优化。

## 异步组件与路由懒加载

```typescript
import { defineAsyncComponent } from 'vue'

const HeavyChart = defineAsyncComponent(() => import('./HeavyChart.vue'))

const RichEditor = defineAsyncComponent({
  loader: () => import('./RichEditor.vue'),
  loadingComponent: LoadingSpinner,
  errorComponent: ErrorDisplay,
  delay: 200,
  timeout: 3000
})

// 路由懒加载
{ path: '/dashboard', component: () => import('@/views/Dashboard.vue') }
```

## v-memo —— 跳过子树更新

```vue
<template>
  <div v-for="item in list" :key="item.id" v-memo="[item.id, selected === item.id]">
    <ExpensiveRow :item="item" />
  </div>
</template>
```

仅当依赖值变化时才重新渲染该行，适合长列表中少数行频繁变化的场景。

## shallowRef —— 浅层响应式

大对象无需深度追踪时使用，避免遍历开销：

```typescript
import { shallowRef, triggerRef } from 'vue'

const chartConfig = shallowRef({ series: [/* 数千数据点 */] })

chartConfig.value = { /* 新配置 */ }            // 整体替换触发更新
chartConfig.value.series[0].data = newData
triggerRef(chartConfig)                         // 手动触发

// markRaw 避免 Vue 递归遍历第三方对象
const chart = markRaw(new ECharts())
```

## KeepAlive —— 缓存组件

```vue
<template>
  <router-view v-slot="{ Component }">
    <KeepAlive :include="['Home', 'UserList']" :max="10">
      <component :is="Component" />
    </KeepAlive>
  </router-view>
</template>
```

## 虚拟滚动

数千行列表只渲染可见区域：

```bash
npm install @tanstack/vue-virtual
```

```vue
<script setup lang="ts">
import { useVirtualizer } from '@tanstack/vue-virtual'

const rows = ref(Array.from({ length: 10000 }, (_, i) => ({ id: i, text: `Row ${i}` })))
const parentRef = ref<HTMLElement>()

const virtualizer = useVirtualizer({
  count: rows.value.length,
  getScrollElement: () => parentRef.value,
  estimateSize: () => 40,
  overscan: 5
})
</script>

<template>
  <div ref="parentRef" style="height: 600px; overflow: auto">
    <div :style="{ height: `${virtualizer.getTotalSize()}px`, position: 'relative' }">
      <div v-for="item in virtualizer.getVirtualItems()" :key="item.key"
        :style="{
          position: 'absolute', top: 0, left: 0, width: '100%',
          height: `${item.size}px`, transform: `translateY(${item.start}px)`
        }"
      >{{ rows[item.index].text }}</div>
    </div>
  </div>
</template>
```

## 优化检查清单

| 场景 | 策略 |
|------|------|
| 首屏慢 | 路由懒加载、异步组件、SSR/SSG |
| 大列表卡顿 | 虚拟滚动、v-memo |
| 频繁渲染 | computed 缓存、debounce watch |
| 大对象开销 | shallowRef、markRaw |
| 页面切换慢 | KeepAlive |
| Bundle 大 | 按需导入、分包、tree shaking |
