# 事件处理与表单绑定

用户交互是前端应用的核心。Vue 提供了 `v-on` 指令处理 DOM 事件，`v-model` 指令简化表单数据绑定，配合修饰符可以写出简洁的交互逻辑。

## 事件监听 (v-on)

使用 `v-on` 绑定事件监听器，简写为 `@`：

```vue
<template>
  <button @click="increment">+1</button>
  <button @mouseenter="showTooltip" @mouseleave="hideTooltip">悬停提示</button>
</template>

<script setup>
import { ref } from 'vue'

const count = ref(0)

function increment() {
  count.value++
}

function showTooltip() {
  console.log('显示提示')
}
function hideTooltip() {
  console.log('隐藏提示')
}
</script>
```

## 内联处理器与事件对象

可以直接在模板中写表达式，也可以通过 `$event` 传递原生事件对象：

```vue
<template>
  <!-- 简单逻辑可直接内联 -->
  <button @click="count++">加 1</button>

  <!-- $event 获取原生事件对象 -->
  <button @click="log($event, 'button-click')">日志</button>
</template>

<script setup>
function log(event, tag) {
  console.log(tag, event.target, event.clientX, event.clientY)
}
</script>
```

## 事件修饰符

Vue 提供了常用的事件修饰符，替代在事件处理函数中手动调用原生方法：

```vue
<template>
  <!-- .prevent 阻止默认行为（等同于 event.preventDefault()） -->
  <form @submit.prevent="onSubmit">
    <button type="submit">提交</button>
  </form>

  <!-- .stop 阻止冒泡（等同于 event.stopPropagation()） -->
  <div @click="onOuterClick">
    <button @click.stop="onInnerClick">内部按钮</button>
  </div>

  <!-- .once 只触发一次 -->
  <button @click.once="doOnce">仅执行一次</button>

  <!-- .self 只在 event.target 是自身时触发 -->
  <div @click.self="onSelfClick">点击我，不包含子元素</div>

  <!-- 可串联：阻止默认行为并阻止冒泡 -->
  <a @click.stop.prevent="handleLink">链接</a>
</template>
```

### 按键修饰符

```vue
<!-- 常用按键修饰符 -->
<input @keyup.enter="submit" />       <!-- Enter -->
<input @keyup.esc="cancel" />         <!-- Esc -->
<input @keyup.delete="remove" />      <!-- Delete / Backspace -->
<input @keyup.space="play" />         <!-- 空格 -->

<!-- 系统修饰键 -->
<button @click.ctrl="ctrlClick">Ctrl + 点击</button>
<div @click.alt="altClick">Alt + 点击</div>

<!-- 精确修饰：只有 Ctrl 按下时才触发 -->
<button @click.ctrl.exact="ctrlOnlyClick">仅 Ctrl 点击</button>

<!-- 鼠标按钮修饰符 -->
<button @click.left="leftClick">左键</button>
<button @click.right.prevent="rightClick">右键（阻止默认菜单）</button>
<button @click.middle="middleClick">中键</button>
```

## v-model 深入

`v-model` 本质上是 `v-bind:value` + `v-on:input` 的语法糖，实现表单元素与数据的双向绑定。

### 基础用法

```vue
<script setup>
import { ref } from 'vue'

const text = ref('')
const checked = ref(false)
const picked = ref('')
const selected = ref('')
const multiSelected = ref([])
</script>

<template>
  <!-- 文本输入 -->
  <input v-model="text" placeholder="输入文字" />

  <!-- 多行文本 -->
  <textarea v-model="text" rows="3"></textarea>

  <!-- 复选框 -->
  <input type="checkbox" v-model="checked" />
  <span>{{ checked ? '已选' : '未选' }}</span>

  <!-- 多个复选框绑定同一数组 -->
  <label><input type="checkbox" v-model="multiSelected" value="vue" /> Vue</label>
  <label><input type="checkbox" v-model="multiSelected" value="react" /> React</label>
  <p>已选：{{ multiSelected }}</p>

  <!-- 单选框 -->
  <label><input type="radio" v-model="picked" value="a" /> A</label>
  <label><input type="radio" v-model="picked" value="b" /> B</label>

  <!-- 下拉选择 -->
  <select v-model="selected">
    <option disabled value="">请选择</option>
    <option value="cn">中国</option>
    <option value="us">美国</option>
  </select>
</template>
```

## v-model 修饰符

```vue
<template>
  <!-- .lazy — 不在 input 事件同步，改为 change 事件 -->
  <input v-model.lazy="text" />

  <!-- .number — 自动转为数字类型 -->
  <input v-model.number="age" type="number" />

  <!-- .trim — 自动去除首尾空格 -->
  <input v-model.trim="username" />
</template>

<script setup>
import { ref } from 'vue'
const text = ref('')
const age = ref(0)   // 即使输入 "25"，也会转为数字 25
const username = ref('')
</script>
```

## 表单验证入门

结合 `computed` 和条件渲染，可以实现简单的表单验证：

```vue
<script setup>
import { ref, computed } from 'vue'

const form = ref({
  username: '',
  email: '',
  password: ''
})

const errors = computed(() => {
  const err = {}
  if (!form.value.username.trim()) {
    err.username = '用户名不能为空'
  } else if (form.value.username.length < 3) {
    err.username = '用户名至少 3 个字符'
  }
  if (!form.value.email.includes('@')) {
    err.email = '邮箱格式不正确'
  }
  if (form.value.password.length < 6) {
    err.password = '密码至少 6 位'
  }
  return err
})

const isValid = computed(() => Object.keys(errors.value).length === 0)

function handleSubmit() {
  if (!isValid.value) return
  console.log('提交数据：', form.value)
  // 发送 API 请求...
}
</script>

<template>
  <form @submit.prevent="handleSubmit">
    <div>
      <input v-model.trim="form.username" placeholder="用户名" />
      <span class="error">{{ errors.username }}</span>
    </div>
    <div>
      <input v-model.trim="form.email" placeholder="邮箱" />
      <span class="error">{{ errors.email }}</span>
    </div>
    <div>
      <input v-model="form.password" type="password" placeholder="密码" />
      <span class="error">{{ errors.password }}</span>
    </div>
    <button type="submit" :disabled="!isValid">提交</button>
  </form>
</template>
```

## 常用事件修饰符速查表

| 修饰符 | 作用 |
|--------|------|
| `.prevent` | 调用 `event.preventDefault()` |
| `.stop` | 调用 `event.stopPropagation()` |
| `.once` | 事件只触发一次 |
| `.self` | 仅当 `event.target` 是自身时触发 |
| `.capture` | 使用捕获模式 |
| `.passive` | 不阻止默认行为，提升滚动性能 |
| `.enter` / `.esc` / `.space` | 按键修饰符 |
| `.ctrl` / `.alt` / `.shift` / `.meta` | 系统修饰键 |
| `.left` / `.right` / `.middle` | 鼠标按钮 |
| `.lazy` / `.number` / `.trim` | v-model 专用修饰符 |

## 小结

- `v-on` (简写 `@`) 绑定事件，支持内联表达式和方法调用
- 事件修饰符减少样板代码，`.prevent` 和 `.stop` 最常用
- `v-model` 是模板中最强大的双向绑定工具
- `.lazy`、`.number`、`.trim` 修饰符简化常见表单处理
- 结合 `computed` 可实现轻量级表单验证，无需引入额外库
