# Figma 入门：从零开始你的第一个界面设计

Figma 是当今 UI/UX 设计师使用最广泛的工具之一。它不仅完全基于浏览器运行，还支持实时多人协作，让你和团队成员能够同时编辑同一个文件，无需传文件、合并版本。更重要的是，Figma 提供了功能完整的免费套餐，足以满足个人学习和中小型项目的需求。

## Figma 的核心优势

相比传统桌面设计工具（如 Sketch），Figma 有三个不可替代的优势。第一，**跨平台**——无论你用 Windows、Mac 还是 Linux，只要有浏览器就能打开设计文件。第二，**实时协作**——多人同时在画布上编辑，就像 Google Docs 一样自然。第三，**单一文件源**——设计、原型、开发标注、评论都集中在一个文件里，开发者在浏览器中打开链接就能查看所有图层和导出资源。

## 界面导览

首次打开 Figma 桌面客户端或浏览器版（figma.com），你会看到文件列表页（File Browser）。创建一个新文件后，你进入了编辑器界面，它由几个关键区域组成：

- **顶部工具栏（Toolbar）**：包含选择工具（V）、画框工具（F）、形状工具、文字工具（T）、评论工具（C）等核心操作工具。
- **左侧图层面板（Layers Panel）**：以树状结构显示当前页面上的所有元素。每个画框（Frame）包含若干图层，你可以拖拽调整层级、重命名、锁定或隐藏图层。
- **右侧属性面板（Properties Panel）**：选中任意元素后，这里会显示该元素的所有可调整属性——位置、尺寸、旋转角度、填充色、描边、效果、字体设置等。
- **中央画布（Canvas）**：你进行设计的主要区域。画布是无限大的，你可以创建艺术板大小的 Frame 来组织你的界面。

## 创建第一个 Frame

Frame 是 Figma 中设计界面的基本容器。按 **F** 键激活 Frame 工具，在画布上拖拽绘制，或从右侧面板直接选择预设尺寸。Figma 内置了大量设备预设：点击 Frame 工具后，右侧面板会列出 iPhone、iPad、Desktop、Android 等分类，点击即可创建对应尺寸的 Frame。建议初学者从 iPhone 14（390x844pt）或 Desktop 1440（1440x1024pt）开始。

## 基本形状工具

Figma 提供了六种基础形状，每个都有对应的快捷键：

- **矩形（Rectangle）**：按 **R** 键，用于卡片、按钮、容器、图片占位等。
- **椭圆（Ellipse）**：按 **O** 键，用于用户头像、圆形图标、状态指示灯等。
- **线条（Line）**：按 **L** 键，绘制单线条，常用于分隔线或图标元素。
- **多边形（Polygon）**：三角形、六边形等，常用于图标或装饰元素。
- **星形（Star）**：评分组件、收藏图标等场景使用。
- **图片/视频占位**：直接从文件管理器拖入图片到 Figma 即可。

选中形状后，右侧属性面板可以调整宽度和高度、拖拽角半径（corner radius）圆角、设置填充色（Fill）和描边（Stroke）。

## 文字工具

按 **T** 键激活文字工具，单击画布创建单行文字，或拖拽出文本框创建固定宽度的段落文字。Figma 支持 Google Fonts 和本地字体。常用设置包括：字体家族（Font Family）、字重（Weight）、字号（Font Size，单位为 px）、行高（Line-height，可用 px 或百分比）以及字间距（Letter-spacing）。

对于中文排版，建议行高设置为 1.5 到 1.8 倍字号，最小字号不低于 12px 以保证可读性。

## 颜色设置

选中元素后，你可以在 Fill（填充）和 Stroke（描边）中设置颜色。Figma 支持多种颜色模式：

- **纯色（Solid）**：输入 HEX 值（如 #3B82F6）或使用取色器。
- **线性渐变（Linear Gradient）**：设置起点和终点的颜色与透明度。
- **径向渐变（Radial Gradient）**：从中心向外扩散的渐变效果。
- **透明度（Alpha/Opacity）**：每个颜色都可以单独调节不透明度，0% 为完全透明，100% 为完全不透明。

建议为项目创建设计系统中使用的核心颜色作为 Color Styles，后续可以一键应用到所有元素上。

## 对齐与分布

Figma 的对齐系统非常直观。选中两个或更多元素后，右侧属性面板顶部的对齐按钮提供了六种对齐方式：左对齐、水平居中、右对齐、顶部对齐、垂直居中、底部对齐。下方的分布按钮（Tidy Up 图标）可以将选中元素在水平或垂直方向上等间距排列。

如果想精确控制间距，使用 Smart Selection：选中多个元素，拖拽它们之间的粉红色手柄即可统一调整间距，输入具体数值即可精确设置。

## 导出

Figma 的导出功能极其灵活。选中任意 Frame 或图层，在右侧属性面板底部找到 Export 区域，点击 + 添加导出格式。支持的格式包括 PNG（可设置 1x/2x/3x 倍率）、JPG、SVG（矢量）、PDF（矢量，常用于印刷或打印稿）等。你可以为一个图层添加多个导出预设，比如同时导出 1x 和 2x 的 PNG。勾选 Export 后，使用 File → Export 或快捷键 **Cmd/Ctrl+Shift+E** 即可批量导出。

## 动手练习：设计一个简单的手机界面

现在来做一个入门练习。创建一个 iPhone 14 尺寸的 Frame（390x844pt），按以下步骤完成一个简洁的 App 主页：

1. **状态栏**：在 Frame 顶部用矩形工具画一个 390x47pt 的黑色矩形作为状态栏背景。
2. **Header**：用文字工具创建一个 28px 粗体标题，文字为 "欢迎回来"，居中对齐。
3. **内容卡片**：用矩形工具画一个 343x120pt 的圆角矩形（corner radius: 16），填充浅蓝色，内部叠加标题文字和描述文字。
4. **底部按钮**：画一个 343x52pt 的圆角矩形（corner radius: 26），填充主色，叠加白色文字 "开始使用"。
5. **对齐检查**：选中所有元素，使用水平居中对齐，确保所有元素在 390px 宽度上居中显示。

完成以上步骤后，你就亲手设计了第一个 Figma 界面。尝试选中 Frame，在 Export 面板导出一张 2x PNG 截图作为成果保存。
